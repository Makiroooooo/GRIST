项目名称：行星表面精确着陆导航与制导控制问题研究

起止年限：2012.1-2016.8

# 一、关键科学问题及研究内容

# 科学问题一：行星表面复杂地形多尺度时空表征与特征识别问题

行星表面地形极为复杂，布满了大量岩石、陨石坑、山脊、斜坡及不连续地貌，为了实现在行星表面安全精确地着陆，需要利用行星表面特殊地形提供导航信息，同时也需要对着陆区相关障碍地形进行识别，这就要求对行星表面地形环境表征和认知。由于行星距离地球遥远，同时已实施的探测任务有限，导致了目前大多数行星缺乏高质量的地形环境表征。行星表面地形复杂呈现多样性，且形态各异，难于用描述符表征，不同高度、视角及运动状态所获取同一地形的观测数据也表现出迥异的特性，这使得对地形的表征与认知成为行星精确定点着陆的难点问题。因此，为实现高精度的着陆探测任务，急需揭示不同类型敏感器的动态误差分布特性、测量质量和精度退化机理，掌握多尺度时空影像数据与高程数据融合地形表征方法，建立行星表面地形环境的表征与认知基础理论，提出高可靠性的行星表面特征检测和识别方法。为此设定如下研究内容：

1）行星表面复杂地形的多尺度时空关联表征  
2）行星表面不变特征鲁棒提取与跨尺度匹配  
3）多样异构行星表面障碍检测与安全着陆点评估

# 科学问题二：欠观测条件下自主导航系统信息获取与状态快速估计问题

深空目标距离地球遥远，信息传输出现大时滞，难以进行着陆过程的实时测控，需要借助自主导航与制导控制技术实现行星表面精确着陆。目前行星着陆采用惯性测量单元，观测信息有限，导航精度低，不满足精确着陆的要求。同时，行星着陆过程中动力学环境具有不确知、强非线性和时变性，导致实时高精度导航极为困难。因此，为实现精确确定着陆器空间状态，迫切需要建立行星着陆非线性导航系统可观测度分析理论，提出大气进入状态鲁棒估计导航观测方案，揭示大气模型、敏感器模型不确定性对导航性能的影响机理，提出利用行星表面自然路标与机会特征的高精度实时导航方法，建立多源扰动下着陆器状态自适应鲁棒估计理论与方法。为此设定如下研究内容：

1）大气进入状态鲁棒估计导航观测方案与误差补偿机理

2）利用行星表面自然路标与机会特征的高精度实时导航  
3）多源扰动下着陆器状态非线性估计理论与方法

# 科学问题三：多约束非线性系统精确鲁棒制导与控制问题

高精度制导与控制是着陆任务成功实施的前提和保障，但是由于大气模型与着陆器系统参数的不确定、动力学的强扰动和时变非线性等诸多制约因素的存在，导致了着陆器制导和控制性能的退化。同时着陆制导与控制还面临制导目标的非一致性、控制的强耦合性、特殊的动力学特性等多约束条件，限制了系统可达区和可控区，使得着陆器的高精度制导与控制成为亟需解决的难点问题。因此，为实现更高的着陆精度，必须深入分析动力学环境和控制能力等各种因素对着陆轨迹的影响机理，提出非一致终端约束下的行星大气进入快速鲁棒制导方法，掌握不规则弱引力场中探测器运动行为机理，建立可行解空间约束下的精确着陆规划与制导理论。为此设定以下研究内容：

1）多约束的行星大气进入快速鲁棒制导  
2）强控制耦合条件下障碍规避与精确着陆制导  
3）不规则弱引力场中探测器运动行为分析与着陆控制

# 二、预期目标

# 3.1 本项目的总体目标：

我国目前正在进行的深空探测发展规划论证，可望在“十二五”期间独立开展火星探测，火星和小行星表面着陆探测也将会列入后续计划中。针对我国在“载人航天与探月工程”重大工程实施后所面临的行星着陆探测技术需求，研究行星精确着陆探测基础问题，解决行星精确着陆的导航与制导控制理论难题，并突破相应的关键技术，建立行星精确着陆导航与制导控制的基础理论与方法，取得系列原创性成果，达到国际先进水平。本项目的研究将为深空探测、尤其是行星精确着陆探测的导航与制导控制提供理论和关键技术支撑，形成一系列具有自主知识产权的创新性成果，并应用于我国未来的火星和小行星着陆探测重大工程；同时，培养一批具有创新能力的深空探测领域的学术带头人和青年骨干，为实现我国在深空探测领域的跨越式发展奠定基础。

# 3.2 五年预期目标：

解决行星精确着陆导航与制导控制问题，包括行星表面复杂地形多尺度时空表征与特征识别问题、欠观测条件下自主导航系统信息获取与状态快速估计问题、多约束非线性系统精确鲁棒制导与控制问题，并完成原理性验证，为行星精确着陆的关键技术创新提供理论与方法。预期取得以下四个方面的突破：

(1) 建立行星表面复杂地形的多尺度时空表征，提出导航特征提取与匹配、障碍检测与识别、安全着陆点评估理论方法；  
(2) 揭示着陆器大气进入与下降着陆的观测导航机理, 建立欠观测条件下的不确定非线性系统状态实时估计理论方法;  
(3) 揭示着陆器大气内飞行与动力下降的运动规律, 提出多约束不确定非线性系统精确鲁棒制导理论方法;  
(4) 揭示不规则弱引力环境下着陆器的运动行为规律, 建立近小天体探测器稳定运动与安全着陆控制理论方法。

# 三、研究方案

# 4.1学术思路

针对行星表面精确着陆的导航与制导控制问题，围绕行星表面复杂地形多尺度时空表征与特征识别、欠观测条件下自主导航系统信息获取与状态快速估计、多约束非线性系统精确鲁棒制导与控制等多学科交叉基础科学问题，研究行星表面复杂地形的多尺度时空关联表征、不变特征鲁棒提取与跨尺度匹配问题，建立行星表面复杂地形的多尺度时空表征与特征识别方法；研究大气进入状态鲁棒估计导航观测方案、利用复杂地形信息高精度实时导航、多源扰动下着陆器状态快速高精度鲁棒估计问题，建立欠观测条件下的不确定非线性系统状态实时估计理论与方法；研究多约束条件下精确着陆的导航与制导控制、不规则弱引力场中探测器运动行为分析与着陆控制等问题，建立多约束不确定非线性动力学精确鲁棒制导理论与方法。

# 4.2 技术途径

本项目的技术途径是结合深空探测的研究前沿和行星表面精确着陆技术的发展趋势，详细分析提高行星着陆精度的途径，并从中凝练关键基础问题，采取理论研究与原理性实验验证相结合、基础研究与关键技术应用研究相结合的方式，

将高校的基础理论研究能力、研究所的工程技术优势、国家级实验平台技术验证能力等进行优势资源整合，开展行星表面精确着陆的关键基础理论和核心技术研究，为深空探测重大科学工程提供理论基础与技术支撑。

具体技术路线如下：

# （1）行星表面复杂地形多尺度时空表征与特征识别

针对各种载荷测量特点和误差特性，基于扫描激光雷达、激光测距仪、光学相机的成像机理，开展不同类型敏感器的动态误差分布、测量质量和精度退化机理等理论研究，建立并求解各种影响因子的成像参数表征方程，提出行星表面地形与基准图之间的拓扑影射关系。在此基础上，提出尺度与时空约束转化方法，基于多源信息融合理论，建立多尺度时空影像数据与高程数据融合地形表征理论方法。

考虑行星表面特征成像的形态学特点，采用系统辨识、人工智能、机器视觉等理论方法，通过理论分析、计算仿真和实验研究相结合的途径，在对着陆目标表面特征分析的基础上，开展图像处理与特征识别的相关研究，归纳各类地形的共性特征，提出不同地形特征的提取和识别方法，实现对图像中陨石坑、岩石、斜坡、裂痕以及其它图像特征点的辨识和检测。针对匹配图像对之间的旋转、尺度缩放及变形等问题，在时域和频域中研究星表特征正射投影的不变属性，提出不变特征鲁棒提取与匹配方法。针对着陆区障碍多样异构等特点，开展着陆区障

碍分布态势感知理论研究，融合视觉和高程信息，提出多尺度多类别障碍实时检测识别方法，建立多样异构障碍检测与安全着陆点评估理论。

# （2）欠观测条件下自主导航系统信息获取与状态快速估计

针对大气进入导航观测信息有限的特点，开展非线性导航系统可观测度分析研究，结合甚高频无线电测量信标和惯性测量单元的导航方式，研究不同运动方式、空间分布的导航信标对着陆器状态的可观性影响，提出大气进入状态鲁棒估计导航观测方案。分析测量误差的环境敏感性及传播特性，建立精确的敏感器误差模型，提出复杂大气进入情况下测量误差补偿与校正机理，同时针对非线性动力学系统随机不确定统计特性的传播规律开展研究工作。在此基础上，针对行星大气模型的不确定性，提出行星大气进入的快速鲁棒非线性估计方法，形成欠观测条件下的行星大气进入状态估计理论。技术路线如图6所示。

针对下降着陆自主导航，在对导航系统可观测性和着陆目标表面特征分析的基础上，利用下降图像中的自然路标特征与基准图匹配，并通过状态解耦变换，研究高精度的绝对位姿快速求解方法；通过鲁棒不变机会特征跟踪，研究基于视

觉的相对运动估计方法。分析地形导航路标分布等因素对导航精度的影响，提出以可观测度为指标的优化绝对导航方案，同时研究外部干扰与模型不确定性对导航性能的影响，提出自适应鲁棒导航滤波方法。在此基础上，融合绝对位姿确定与相对运动估计，形成利用行星表面自然路标与机会特征的高精度实时导航理论。

# （3）多约束非线性系统精确鲁棒制导与控制

针对行星大气进入动力学模型的不确定、非线性和强扰动问题，建立考虑强风干扰的大气进入动力学模型，针对着陆器大弹道系数、小升阻比的特点，分析动力学环境和控制能力等各种因素对进入轨迹的影响机理，确定有控着陆器可控区和可达区，提出行星大气进入的最佳入口条件和落点性能指标评价方法。依托着陆器高精度强鲁棒性控制理论的研究，充分考虑多源强扰动、不确定性的影响及计算效率，建立行星大气进入的精确快速鲁棒制导方法。通过入口条件优化和制导策略优化两方面理论方法的建立，解决着陆器终端目标与约束不一致的问题，形成行星大气进入快速鲁棒制导理论。

针对障碍规避与精确着陆高精度、强实时的任务需求，考虑着陆动力学不确定、非线性的特点，综合利用理论分析和数学实验方法分析多源扰动和系统不确定性对着陆器制导性能的影响，建立着陆器动力学模型、环境扰动模型和内部扰动模型等数理模型。分析强控制耦合、着陆点动态变化条件下，着陆制导的可行解空间，在此基础上，开展着陆轨迹设计与优化理论、高精度强鲁棒性着陆控制理论等方面的研究，建立可行解空间约束下的精确着陆规划与制导理论，提出强控制耦合条件下的精确定点着陆制导方法。

针对小天体引力场不规则的特点，开展不规则引力场的精确表征方法，提出基于质量和形状数据的小天体数字引力场模型建立方法，建立椭球谐波引力场模型系数确定方法，解决无轨道测量数据条件下的引力场模型建立问题。开展小天体附近绕飞、伴飞和悬停等运动规律研究工作，通过分析不同旋转速率下小天体环绕周期轨道、同步轨道以及平衡点附近周期轨道的运动特性，揭示不规则弱引力场中无控飞行的运动机理，提出探测器运动稳定的条件。开展主动盘旋轨道设计及其稳定性分析、有控着陆轨道的特性及其稳定性的研究工作，在分析外部环境扰动和着陆器内部扰动对下降轨迹和着陆点影响规律的基础上，提出对未建模扰动低灵敏的姿轨耦合鲁棒控制方法，形成不规则弱引力场中着陆器动力学与控制基础理论。

# （2）研究方法创新

(1) 针对行星表面地形时空表征、鲁棒不变特征提取与匹配及多样异构障碍检测尚缺乏统一、共性研究的问题。基于已有的探测数据, 构建行星表面地

形参考模型—参考数字高程图、参考影像图、参考障碍图、安全区域图。在此基础上，利用形态学研究行星表面地形的几何特性，进行明显特征和障碍提取与检测。通过详细的纹理分析提取鲁棒不变特征。建立行星表面复杂地形的多尺度时空表征，提出导航特征提取与匹配、障碍检测与识别、安全着陆点评估理论方法。

(2) 针对导航观测信息有限、着陆器系统和大气参数不确定、动力学环境复杂多变的着陆导航问题。根据非线性系统可观测度分析，确定对着陆器状态估计的大气进入观测方案；分析测量误差的环境敏感性及传播特性，建立精确的敏感器误差模型，揭示测量误差补偿与校正机理，建立精确的测量模型；融合自然路标与机会特征进行下降着陆的绝对导航；融合IMU、激光测距仪及下降图像匹配多种观测信息，结合自适应鲁棒导航滤波，实现相对着陆点状态的高精度鲁棒估计。揭示行星大气进入与下降着陆的观测导航机理，建立欠观测条件下的不确定非线性系统状态实时估计理论。  
③ 针对终端约束非一致、强控制耦合条件下的不确定非线性动力学制导问题。通过可控能力和可达范围分析，确定行星大气进入的最佳入口条件，综合考虑着陆器落点精度要求及开伞条件约束，建立优化的性能指标评价体系，优化的行星进入制导策略，确保满足落点精度要求与终端约束；充分考虑气动力、行星引力、引力摄动、外部环境扰动、内部扰动，建立复杂环境下着陆器的控制模型，分析多源扰动和系统不确定性对着陆器制导性能的影响，分析强控制耦合、着陆点动态变化条件下，着陆制导的可行解空间，研究最佳的制导策略。揭示行星着陆器大气内飞行与动力下降的运动规律，提出多约束非线性系统精确鲁棒制导与控制方法。  
(4) 针对不规则弱引力场中的运动特性和稳定性分析、着陆控制问题。利用多面体逼近小天体形状，通过虚拟外点引力的方式估计椭球谐波系数，建立引力场高精度数字和球谐波模型，利用鲁棒的非奇异庞加莱映射研究环绕周期轨道，利用拟周期轨道的不变流形拓扑结构研究平衡点轨道，基于低保守性鲁棒性能分析工具研究着陆轨迹。揭示非规则弱引力环境下着陆器的运动行为规律，建立近小天体探测器稳定运动与安全附着控制理论。

课题1：行星表面地形时空表征与特征识别理论方法

# 研究目标：

融合敏感器位置、姿态、图像数据等多源信息，构建高精度的基准数字高程图与影像图，建立行星表面复杂地形的多尺度时空表征模型；提出行星表面不变特征鲁棒提取和跨尺度匹配方法；建立多样异构障碍检测与安全着陆点评估理论，为精确定点着陆提供理论基础。

# 研究内容：

# 1、行星表面复杂地形的多尺度时空关联表征

在严格统一的时间和空间参考框架下，研究敏感器的成像机理及各种因素的影响，建立各种影响因子的成像参数表征方程，精确恢复像点的几何特征，建立行星表面地形与基准图之间的拓扑影射关系；基于序列下降图像特征提取与跟踪，结合激光测距仪测量的高度信息，重构着陆区三维地形图。

# 2、行星表面地形不变特征鲁棒提取与跨尺度匹配

分析陨石坑边缘的几何特征，建立精准的描述方式，研究陨石坑边缘提取方法，解决边缘退化、不完整或重叠的陨石坑检测问题；考虑行星表面特征的形态学特点，分析星表特征正射投影的不变属性，研究不变特征的鲁棒提取与匹配方法；研究特征不变的多尺度图像变换方法，解决匹配图像对间的旋转、尺度缩放及变形等问题；进一步研究未知环境下地形特征的误匹配判定方法。

# 3、多样异构行星表面障碍检测与安全着陆点评估

针对行星表面陨石坑、裂缝、陡坡、大岩石等形态各异障碍，重点开展基于光学图像和高程数据的自主障碍检测方法研究，利用地表特征与光照方向之间的几何关系，分别研究基于边缘信息和阴影信息的陨石坑和岩石识别方法；利用高程信息建立斜坡和粗糙度的表征，研究裂缝、陡坡检测方法。开展着陆区障碍分布态势感知理论研究，融合视觉和高程信息，提出多尺度多类别障碍实时检测识别方法，综合考虑工程实现和科学目标等约束提出安全着陆点评估准则。

针对行星大气进入欠观测、不确定和强扰动问题，提出大气进入状态鲁棒估计导航观测方案；分析各种扰动因素对导航性能的影响，建立大气进入状态高精度实时鲁棒估计理论；针对复杂的地形环境，提出利用行星表面自然路标与机会特征的高精度实时导航方法；揭示外部干扰与模型不确定性对导航性能的影响机理，提出自适应鲁棒导航滤波方法，实现着陆器相对着陆点状态的高精度确定，建立欠观测条件下自主导航系统信息获取与状态快速估计理论与方法。

# 研究内容：

# 1、大气进入状态鲁棒估计导航观测方案

研究非线性导航系统可观测度分析判据，提出大气进入状态鲁棒估计导航观测方案；分析测量误差的环境敏感性及传播特性，建立精确的敏感器误差模型，揭示复杂大气进入情况下测量误差补偿与校正机理，建立精确的测量模型；分析各种扰动与测量误差对导航性能的影响，研究导航系统的性能评价准则。

针对利用行星表面自然路标导航的严重非线性特性，探索新的解算结构，研究状态解耦变换与高精度的位姿快速求解方法；分析地形导航路标分布对导航精度的影响，以可观测度为指标提出自然路标的选取准则，研究优化的绝对导航方法；研究基于计算机视觉的着陆器相对运动估计方法，利用相邻序列图像中的机会特征跟踪，估计相对平移与速度；融合绝对视觉定位与相对位姿估计，研究行星下降着陆的高精度实时导航。

# 3、多源扰动下着陆器状态快速高精度鲁棒估计

针对行星引力场模型的不确定性和着陆器动力学系统的强扰动、时变、非线性，研究多源干扰导致着陆器随机运动的产生机理，建立其统计模型；探索着陆器动力学不确定性的产生机理，研究其在轨辨识的理论方法；揭示非线性动力学系统随机不确定统计特性的传播规律，提出快速鲁棒非线性估计理论与方法，实现着陆器状态的高精度鲁棒估计。

课题3：多约束条件下精确着陆的导航与制导控制方法

分析多样异构地形障碍特征，建立障碍与安全着陆区的时空映射表征模型，提出目标着陆点与安全着陆轨迹的在线实时规划方法；分析多源扰动和不确定性对着陆轨迹的影响，结合着陆器的机动能力，研究低保守鲁棒轨迹优化与机动控制理论方法。

课题4：不规则弱引力场中探测器运动行为分析与着陆控制

# 研究目标：

针对小天体引力场不规则的特点，研究不规则引力场的精确表征方法，建立精确的引力场模型；研究不规则弱引力环境下，探测器在小天体附近绕飞、伴飞和悬停等运动规律，揭示不规则弱引力场中探测器运动机理，提出运动稳定的条件；研究有控着陆轨道的特性及其稳定性，形成不规则弱引力场中探测器动力学与控制基础理论。

# 研究内容：

# 1、不规则小天体引力场的表征方法

基于雷达与光学望远镜观测获得的小天体质量和形状数据，利用多面体逼近小天体形状，建立小天体数字引力场模型；研究通过虚拟外点引力的方式估计椭球谐波引力场模型系数，解决无轨道测量数据确定引力场球谐波系数问题。

# 2、不规则弱引力场中运动行为与稳定性分析

基于均匀转动的二阶二次引力场模型，研究小天体形状不规则和旋转引起的能量和角动量变化规律、围绕小天体的探测器轨道运动作为中心体转动和质量分布的函数是如何演化的；研究不同小天体旋转速率下环绕小天体周期轨道的存在性和稳定性等问题；只考虑引力和旋转的影响，研究同步轨道的大小和稳定性；研究平衡点附近周期轨道的存在性和稳定性。

# 3、主动盘旋轨道与着陆轨迹鲁棒稳定控制

基于太阳、小天体和探测器构成的三体系统，研究主动盘旋轨道设计及其稳定性分析、有控着陆轨道的特性及其稳定性；考虑引力变化和太阳光压扰动、表面喷发物质（气体）等因素，分析外部环境扰动和着陆器内部扰动对下降轨迹和着陆点的影响规律，综合着陆器动力学与目标天体引力模型以及外界干扰等不确定因素的特点，设计不确定多扰动条件下的鲁棒着陆制导策略，研究对未建模扰

动力和力矩低灵敏的姿轨耦合鲁棒控制方法。

研究行星着陆运动模拟方法，建立行星着陆区表面模拟地形。优化基于行星表面地形特征的自主导航理论方法验证试验方案，建立高置信度的评估方法；开展自主导航与控制试验，分析和处理试验数据，实现对自主导航与制导控制方法的验证评估。
