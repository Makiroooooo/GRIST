# 目的和意义  

人工关节是目前治疗关节毁损性疾病如类风湿性关节炎、骨性关节炎等疾病晚期关节病损的重要手段，我国需要进行人工关节置换手术的病人至少在150万人以上。人工关节中生物固定型假体的远期稳定主要依靠骨一假体界面骨长入达到机械锁定。然而，由于假体表面骨长入范围及程度有限，长期随访仍有较高的无菌性松动率，因而使其应用范围受到一定限制。多种假体表面处理对骨与假体结合范围、深度和强度的增强作用也均有限。因此，如何提高假体表面骨组织长入，增加假体与周围骨床的结合力和范围，从而使其获得长期稳定，成为生物固定人工关节假体迫切需要解决的关键问题和近年来非骨水泥固定人工关节的研究焦点。  

目前普遍认为，采用微孔技术及骨诱导物质有利于骨组织长入，可扩大周围骨质与假体的结合面积。设想如将金属假体制成中空并于其壁上打出多数孔洞使之内外相通，通过诱导使骨质不仅贴假体壁生长，而且通过孔洞长入假体内部，则骨一假体的结合面积将进一步显著增大，并可使假体完全包埋于骨质中而达到真正的一体化结合。由于当今以骨髓基质干细胞（mesenchymal stem cells，MSCs或bone marrow stromal cells，BMSC）为种子细胞的骨组织工程技术的不断进步，使以中空多孔金属假体为刚性构架，与装载于其中复合有骨诱导因子（rhBMP-2）的可吸收支架材料相结合形成骨组织再生工程的构架成为可能，也为中空人工关节假体的实质性研发创造了条件。  

本研究意在根据组织工程学原理，利用中空多孔金属假体为骨组织再生工程构架，观察体内假体界面间充质细胞或体外干细胞在假体内、外的分化生长情况，并探讨应用骨诱导因子（rhBMP-2）更有效地提高成骨活性，促进金属假体与骨质相对一体化结合的可行性，为研制中空多孔人工关节假体或其他骨组织再生骨植入物提供基础实验依据。试图使植入骨腔的中空假体成为骨骼的一部分，从而保证假体在骨内受应力作用状态下的长期稳定，避免松动的发生和病人再次手术的痛苦，使人工关节更为广泛和有效地应用于需要的病人，因而具有广阔的应用前景和巨大的社会、经济效益。  

# 现状与存在的问题  

# - 非骨水泥固定人工关节假体的骨结合  

非骨水泥生物固定人工关节，尤其是人工髋关节，为达到初始稳定而设计的假体几何形状已很难有新的突破。其设计不论如何变化，假体均为实心体。这种实心体即使在界面上能与周围骨质充分整合，其结合部分相对整个假体截面而言也十分有限，因而使假体仍相对独立于骨质之外，两者的结合强度和由骨长入而达到的远期稳定性受到制约。因此，申请人提出了将假体植入骨腔或与骨质接触的部分设计为空心体，并利用复合于其中的骨诱导物质诱导其周围骨质通过壁上孔洞长入假体内的空腔中而达到两者相对一体化结合的设计构想，目前国内外均未见报告。  

# - 骨组织工程支架与人工关节  

目前骨组织工程选择支架多把注意力放在生物可降解材料上，此类材料可分为无机材料和有机聚合物两类，这些材料模拟天然骨细胞外基质成分，具有良好的生物相容性及骨诱导活性，可用于修复骨缺损。然而，其强度尤其是被骨组织替代过程中的强度却不比较薄弱。人工关节假体作为承载应力的人造器官，尤其是髋、膝关节活动时需承受超过体重的负荷，可降解支架材料显然没有足够的支撑能力，因此人工关节材料从来就没有放弃过金属。然而，始终未有报告将金属材料的刚性支撑作用与骨组织工程支架材料的骨再生引导诱导作用相结合应用于人工关节。  

# - 干细胞或间质细胞有分化成骨和向金属支架孔隙内生长的能力  

Ochi 应用已经骨转化的  $\mathrm{BMSC}_{\mathrm{S}}$  作为种子细胞，接种在已塑型的聚乳酸一聚己醇酸（poly-DL-lactic-co-glycolic acid, PLGA）胶原海绵上，作为骨缺损替代，取得良好效果。Bidic  

将40个兔颅骨缺损动物模型分成四组：A不做任何处理；B自体骨移植；C单纯骨支架（一种多聚陶瓷复合物）；D接种了自体  $\mathrm{BMSC}_{\mathrm{S}}$  的骨支架。一定时间后（6，12周），CT观察：B组的修复程度远高于其他三组，D组也取得良好的效果。两者均验证了  $\mathrm{BMSC}_{\mathrm{S}}$  的骨向分化能力。  

另有大量研究表明，在有或无生长因子存在的情况下，骨质有向金属孔隙内生长的特性。Sumner等曾在骨-假体界面上施加具有诱导成骨作用的TFG-β1，从而不仅使假体与周围松质骨之间3mm的间隙被骨质填充，而且达到了良好的假体表面骨长入和整合，提示界面间质细胞或成纤维细胞具有接受成骨因子刺激而向骨祖细胞和成骨细胞衍化的能力，故认为，界面骨质是成骨因子诱导下由骨髓和骨内膜的间质细胞或纤维组织向骨组织的分化的结果。1998年Bostrom等将rhBMP-2与胶原复合后置入中空假体中，发现成骨性细胞因子可进一步诱导周围骨质通过其壁上孔洞长入中空假体试件骨收集室（Bone harvest chamber,BHC）内，提示骨质具备通过壁上孔洞长入金属空腔体并在其中形成成熟小梁骨的能力。  

目前国外研究多集中于观察BHC内生长因子对骨质的诱导作用，，国内也有作者注意到了中空多孔柱状钛种植体复合牛骨形成蛋白后作为多载体系统的骨诱导和骨长入效应，但对如何选择诱导物及载体，提高细胞和材料的亲和力，设计中空假体壁孔洞大小及孔隙率等却了解不多。对装载有生物可降解材料的中空多孔金属假体经体外骨组织工程化处理后再植入活体骨质内更少有研究。  

综上，本研究拟在前人研究的基础上将骨组织工程的方法与多孔中空金属假体的理念  

相结合，为能够达到更为牢固生物结合的生物固定型中空人工关节或其他金属骨植入物的研制提供实验依据。本项目研究成果有可能带来一次生物固定型人工关节及有关骨植入物领域设计理念的变革，使该类关系到病人生活质量的骨科植入物具有更长的使用寿命，从而使更多的病患受益，恢复其工作能力和社会能力，并有可能开辟一个新的学科研究领域，社会、经济效益不言自喻。

# 2、项目的研究内容、研究目标，以及拟解决的关键问题。（此部分为重点阐述内容）  

# 研究内容  

(1) 中空多孔金属（钛）假体试件的制作、可降解骨组织工程支架以及rhBMP-2的复合  

1）假体尺寸、形状、孔洞大小、表面处理等  
2）可降解骨组织工程支架选择（PLGA或羟基磷灰石等）以及与假体和rhBMP-2的复合  

(2) 兔 BMSCs 的提取和体外培养  

1）提取兔  $\mathrm{BMSC}_5$  ，体外培养。  
2）传代，使第二代BMSCs黏附在中空金属假体内的可吸收性载体上。  

(3) 中空多孔金属假体多载体系统的体外骨组织工程化培养  

在体外培养基内，由 rhBMP-2 诱导下，使黏附在假体表面及假体内可降解支架上的 BMSCs  

向成骨细胞转化, 在中空金属假体表面及腔室内形成与金属假体和可吸收性载体结合牢固的组织工程化骨。  

(4）经骨组织工程化的中空多孔假体的体内植入观察  

1）植入动物体内前，用I型胶原溶液溶解rhBMP-2，以一定浓度（根据前期研究基础）滴定于空心金属体内，使空心金属体在植入前保持有利于成骨的rhBMP-2浓度。  
2）植入动物体内，对照实心金属假体。一定时期后对比表面附着组织工程化骨的金属假体在骨结合率以及假体与骨之间的最大剪切力等方面是否优于表面涂抹rhBMP-2的实心金属体和单纯实心金属体。  

(5) 中空多孔金属假体多载体系统直接体内植入的对比研究  

植入后可在中空假体腔室内加载或不加载BMSCs，均与前述处理进行对比。  

# 研究目标  

观察使用中空多孔金属假体多载体系统的各实验组的骨长入情况，并与实心假体对照组比较一定时期内金属假体在骨结合率以及假体与骨之间的最大剪切力等方面是否相同，实验组是否优于对照组。各实验组间对比，选择最佳处理方案。  

# 拟解决的关键问题  

人工关节置换术后的主要并发症为假体松动。假体松动的主要原因有：a 假体的微动；b 骨与假体之间有间隙；c 磨损颗粒引起的骨溶解反应；d 假体的表面结构；e 患者的骨质特点。其中最主要的为前三者。因为，即使患者平卧在床，也不可能完全避免假体与骨组织之间的微动。长期随访研究发现假体与骨床间的间隙不能完全被骨质填充，间隙的存在有利于磨损颗粒的扩散。  

因此，本研究拟解决的关键问题：  

1）在体外使中空金属假体表面和内部形成与之结合紧密的组织工程化骨。  
2）中空金属体内复合rhBMP-2的I型胶原溶液，使金属体在体内持续、稳定地释放rhBMP-2，从而诱导受体体内BMSCs向成骨细胞转化，提高受体自身的成骨能力。  
3）中空金属体内复合可吸收性载体，不仅起到成骨支架作用，还可促使受体自身的成骨细胞长入，并在一定时期后分解，使中空金属假体内部完全由有机骨充填。  

本研究最终拟解决如何使金属假体植入骨质后与其周围骨床达到一体化紧密结合的问题，探讨中空多孔金属假体复合骨诱导材料作为骨组织再生工程构架及中空人工关节或其他骨植入物设计和应用的可行性。

# 4、本项目的特色与创新之处。  

1）目前，人工关节假体的设计多为实心金属体。这种实心体即使在界面上能与周围骨质充分整合，其结合部分相对整个假体截面而言也十分有限，因而使假体仍相对独立于骨质之外，两者的结合强度受到制约。本设计提出了将假体植入骨腔或与骨质接触的部分设计为空心体，并利用复合于其中的骨诱导物质诱导其周围骨质通过壁上孔洞长入假体内的空腔中而达到两者相对一体化结合的设计构想，目前国内外均未见报告。  
2）作为骨组织工程学中的种子细胞，国内外对  $\mathsf{BMSC}_s$  的研究主要集中在治疗骨缺损和骨不连等方面，在人工关节假体上的研究尚无报道。本实验目的就是利用  $\mathsf{BMSC}_s$  的成骨特性，在中空人工金属假体内部形成与之结合紧密的组织工程化骨，并复合rhBMP-2，再植入动物体内，观察假体表面及假体内部与骨之间的骨结合程度及界面最大剪切力是否优于单纯的实心金属假体植入。如实验结果阳性，将为解决人工关节置换术后假体松动的治疗开创一新的方法，将从根本上改变人工关节治疗的观念和方法，具有广泛的应用前景，产生巨大的经济效益和社会效益。
