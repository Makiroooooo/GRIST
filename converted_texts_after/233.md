项目名称：我国主要人工林生态系统结构、功能与调控研究

起止年限：2012.1-2016.8

# 一、关键科学问题及研究内容

# 1. 拟解决的关键科学问题

为满足国家对人工林生态系统木材生产与生态安全功能的双重需求，通过人为调控提升生态系统生产力并充分发挥生态功能是人工林生态系统经营的核心目标；而人工林生态系统生产力形成与积累过程是人工林主体存在、进而满足双重需求最根本的基础；本项目要解决的第一个关键科学问题：人工林生态系统生产力形成的制约机制。采用高投入高产出的集约经营生产木材方式已证明无法实现人工林生态系统生产与生态功能的可持续，尤其是在轮伐、连载后发生林木生长与土壤肥力衰退现象严重；而以往研究关于人工林结构与现实、长期生产力的关系认识不足；本项目要解决的第二个关键科学问题：人工林生态系统结构对养分平衡的影响机制。生态系统各服务功能间的权衡关系已成为生态系统管理科学的国际前沿，由于以往人工林以追求木材生产为主，缺乏对人工林生态系统生产与生态功能关系的了解；本项目要解决的第三个关键科学问题：人工林生态系统主要生态功能与生产功能（木材生产）的权衡关系。

# （1）人工林生态系统生产力形成的制约机制

人工林生态系统光合产物分配、生物量积累与能量转化过程是生态系统健康评价的特征指数，已有工作更多地集中在人工林生态系统不同树种的研究上。本项研究针对我国人工林生产力低下问题，将进一步研究不同环境梯度、不同林分结构及不同经营措施对主要人工林树种光合效率与生物量积累和分配的影响机理，发现其时空分异规律，分析人工林生产力低下环境要因，为通过人为调控提高生产力提供理论基础。

# (2) 人工林生态系统结构对养分平衡的影响机制

土壤养分库不仅为树木提供有效养分物质，而且本身拥有丰富的生物多样性，进而与树木一起共同维持着人工林生态系统各种功能。土壤养分平衡过程是整个人工林生态系统功能稳定、持续的保障。过去工作主要集中在人工林自然状态下的养分消耗研究，从养分平衡及人为调控对其影响角度研究甚少。本项目将以关键养分元素 NP 为重点，阐明林木个体对土壤 NP 吸收、利用与归还机制；确定树种组成（结构）及土壤微生物功能群对生态系统养分平衡的影响，建立人为调控对系统 NP 循环过程影响及其与长期生产力维持的关系，探索人工林生态系统养分平衡机制，为建立有效的调控模式提供理论依据。

# （3）人工林生态系统主要生态功能与木材生产的权衡关系

人工林通过固定  $\mathrm{CO}_{2}$  缓解气候变暖，同时人工林必须在替代天然林提供木材方面起到关键作用。已有研究表明，人工林地上生物量及地上植被等特征可以解释土壤碳储量空间变异的  $75\%$  ，而林分结构、经营措施对系统碳储量、生物多样性及蓄积量或木材生产的影响机制，以及生物多样性、木材生产与地下生态系统碳固持关系等尚属空白。本项目将以寻求人工林地下生态系统碳固持等生态功能和木材供给权衡关系为主线，研究人为调控对生物多样性、地下生态系统碳固持与木材生产及其权衡关系的影响机制，为评价人工林生产功能和生态功能权衡关系，进而为提出最佳平衡点管理模式提供理论支撑。

# 2. 主要研究内容

围绕上述关键科学问题，以对我国木材生产具有举足轻重作用的主要树种——杉木、落叶松、杨树人工林生态系统为主要研究对象，紧紧围绕人工林生态系统生产力形成机理、结构对养分平衡的影响、主要生态功能与木材生产权衡关系及人工林生态系统结构优化原理与功能评估等四个层次开展科学研究。

# 内容一、人工林生态系统生产力形成机理与关键过程

重点研究主要人工林树种（杉木、落叶松、杨树）在不同环境梯度、不同林分结构及不同调控措施下的光合效率与生物量积累和分配的关系，确定人工林生态系统生产力分布格局及制约机制，揭示主要树种人工林生态系统的光能利用效率，分析人工林生态系统生产力低下的环境要因。研究拟从以下3方面展开：（1）环境因子对人工林主要树种光合效能的影响机制；（2）人为调控对人工林生态系统生产力形成的驱动机制；（3）人工林生态系统生产力低下要因分析及调控对策。

# 内容二、结构调控对人工林生态系统养分平衡的影响机制

重点研究主要树种（杉木、落叶松、杨树）林木根系 NP 吸收、利用及其对养分循环的作用机制；树种组成及土壤微生物功能群对生态系统养分平衡的影响，人为调控（间伐、林下植被/凋落物管理及施肥）对 NP 循环过程及长期生产力维持的影响；探索人工林生态系统的养分平衡机制。研究拟从林木个体和林分系统两个层次、4 方面展开：（1）根系分布特征与外生菌根真菌对林木 NP 吸收利用的调控机制；（2）林木根茎（干）叶、凋落物 NP 含量动态及其养分释放；（3）根际微生物、土壤生物功能群多样性和细根周转对土壤 NP 矿化及其有效性的影响；（4）残落物管理和人工林代际更替/更新对土壤养分供应和 NP 转化的影响。

# 内容三、人工林生态系统主要生态功能与木材生产的权衡关系

重点研究不同林分结构下生物多样性、碳储量、生产力时空格局及区域分异规律，人为调控对生态系统生物多样性、生产力、木材质量和系统碳库动态与构成影响，探讨生物多样性与生产力（现实一长期）、碳固持（能力与潜力）和木材生产权衡关系。研究拟从以下4方面展开：（1)地下生态系统碳储量与干材质量对林分结构调控和施肥措施的响应；（2)代际更替对地下生态系统碳储量和蓄积增长的影响机制；（3)林分结构及抚育措施对生态系统生物多样性和生产力的影响；（4）人工林生物多样性对生态系统生产功能的影响。

# 内容四、人工林生态系统功能评估与结构优化

以内容一至三研究成果为基础，建立人工林生态系统结构量化方法和生产功能与固碳等生态功能评估方法体系，评估与预测人工林生态系统生产力（现实与长期）和碳固持（能力与潜力），提出主要人工林生态系统结构优化、功能提升的调控对策。研究拟从以下4方面展开：（1）人工林结构（林分垂直结构与景观结构）量化表达与优化；（2）人工林生态系统现实生产力与碳储量的综合评估；（3）人工林生态系统长期生产力与固碳潜力变化趋势预测和情景分析；（4）人工林生态系统生态适宜性的景观/区域尺度评估与调控对策。

# 二、预期目标

# 1. 总体目标

阐明人工林生态系统生产力形成的光合效能制约机制，为提高我国人工林生态系统生产力，缩小经济高速发展与木材供给间的巨大差距提供理论支撑；揭示人工林结构对现实生产力与长期生产力影响机理，以及人工林生态系统养分平衡机制，为实现人工林生态系统功能可持续发挥提供理论基础；明确人工林生态系统生产功能与生态功能的权衡关系，找到木材生产与生态功能发挥的权衡点，改变现有人工林大面积纯林的经营模式，为满足国家对木材生产、生态功能和碳外交谈判的战略需求提供科学支撑；为发展人工林生态学做出贡献。

# 2. 五年预期目标

(1) 科学研究: 明确影响主要人工林 (杉木、落叶松、杨树) 生态系统生物量形成过程的制约机制, 探索杉木、落叶松人工林生态系统长期生产力形成与维持机制, 阐明杉木、落叶松、杨树等人工林生态系统主要生态功能、碳汇功能与木材生产的权衡关系。  
(2) 国家需求: 建立人工林生态系统生产功能与生态功能评估指标体系, 明确不同尺度下主要人工林生态系统结构优化、功能提升的调控对策与途径, 为提高人工用材林生产力 (木材供给) 提供科技支撑; 同时阐明主要类型人工林生态系统固碳能力与潜力, 为满足国家应对气候变化和碳外交谈判提供科学依据。  
（3）平台建设：带动我国主要人工林长期定位和联网观测的发展，完善我国主要人工林生态系统的长期定位观测、控制实验、植被和土壤等研究体系，形成我国主要人工林生态系统结构与功能的野外综合研究平台。  
(4) 研究论文: 国际主流刊物论文 130 篇, 争取在有重要影响的国际期刊发表论文 4-6 篇, 提供国家层面的人工林生态系统多目标、多功能的经营理论与调控对策咨询报告 1-2 份。  
(5) 数据积累: 构建我国主要人工林生态系统长期定位观测、控制实验、植被和土壤典型调查产生的多源数据, 形成我国人工林生态系统研究数据共享平台。  
(6) 人才培养: 培养一支长期关注人工林生态研究科技队伍, 成为国际人工林生态领域重要力量, 为本领域打造一支创新能力强的优秀研究群体。培养硕士、博士研究生 70-75 人, 博士后 5-10 人。

# 三、研究方案

# 1、总体研究思路

针对现有人工林生态系统存在的生产力低、功能不可持续、生产功能与生态功能不协调等三大关键问题，以人工林结构对生态系统生产与生态功能权衡关系的影响机制为主线，以国家和各部门森林生态系统野外台站网络和现有人工林长期定位研究基地为依托，采用微观与宏观、长期定位监测与野外调查、过程机理与模型预测、控制实验和模拟实验有机结合的方法，从生态系统完整性和稳定性角度出发，在林木-林分-景观-区域的空间尺度和时-日-季-代际更替的时间序列尺度上，明确人工林生态系统生产力低下的制约机制，探讨人工林生态系统功能可持续的养分平衡机理，确定人工林生态系统生产功能与生态功能之间的权衡关系；提出我国人工林生态系统结构优化、功能高效稳定的调控对策。
# 2. 关键技术途径

# （1）总体技术路线

根据上述总体学术思路，本研究将以“明确生产力低下制约机制  $\rightarrow$  剖析人工林生态系统功能可持续的养分平衡机理  $\rightarrow$  探讨生产功能与生态功能的协调关系  $\rightarrow$  提出定向调控对策  $\rightarrow$  实现科学目标并满足国家需求”的分层推进、逐级提高的技术路线开展研究。首先对旨在剖析人工林现实生产力低下的制约机制的“人工林生态系统生产力形成机理与关键过程”进行研究；其次对旨在阐明不同调控措施下人工林生态系统养分平衡机制的“结构调控对人工林生态系统养分平衡影响机制”进行研究；再次对旨在明确人工林生态系统生产功能与生态功能协调关系的“人工林生态系统主要生态功能与木材生产权衡关系”进行研究；基于上述三个层面的突破进展，最终在不同时空尺度进行“人工林生态系统功能评估与结构优化”研究，提出人工林结构优化和功能稳定高效的调控对策，解决关键科学问题，为实现人工林生态系统功能高效、稳定、可持续的最高目标提供科学支撑。

# （2）总体研究方案

# 1）研究对象和研究平台

研究对象：本项目选择杉木、落叶松、杨树人工林生态系统为主要研究对象，为定向调控人工林结构与功能，在构建高生产力人工林的同时，使其生态功能向稳定的近自然林方向发展，选择与三种主要树种人工林毗邻的天然林生态系统进行对比研究。

研究平台：为了保证研究的系统性与可集成性，本项目以国家和各部门森林生态系统野外台站网络为基础，分别在杉木、落叶松、杨树人工林集中分布且具区域代表性的地区，选择拥有大量长期试验、历史资料以及研究成果的国家或院所级野外实验站和种植基地为定点系统研究平台。对于杉木人工林，自东向西依次选择相关试验站；对于落叶松人工林，自东北向西南依次选择相关试验站；对于杨树人工林，选择相关种植基地。共计7个试验站点、4个主要种植基地，进行长期定位和控制实验条件下的机理性研究。

# 2）研究技术方案

主要内容的研究技术方案如下：

# 内容1：人工林生态系统生产力形成机理与关键过程

面对我国主要人工林普遍存在的生产力低下、无法满足经济高速发展对木材需求以及由于生产力低下不能发挥生态功能的现实问题，采用点（微观尺度，定点控制）、面（宏观尺度，典型气候区）相结合的手段，系统实施：1）主要人工林树种（杉木、落叶松、杨树等）光合效率与生物量积累和分配的关系：在不同环境梯度（利用开顶箱或人工气候室控制光照、温度、降水与UV-B辐射等多环境因子）下，确定主要人工林树种光合速率、荧光动力学等特征，利用  $^{14}\mathrm{C}$  -同位素示踪技术研究主要造林树种在不同环境下光合产物的分配规律和分配格局的变化。2）人工林生态系统生产力形成的制约机制：对于不同林分结构、立地（土壤、地貌等）、经营管理（密度调控、养分管理、林下植被管理等）驱动下的人工林生态系统，采用直接测定法（冠层分析系统等）或间接计算法（如叶面积指数法、光谱指数法等）获取冠层辐射量，进而得到系统光能利用率，揭示主要树种人工林生态系统的生态效率，分析人工林生态系统生产力低下的环境要因，为提高系统光能利用提供理论依据和技术途径。上述研究最终回答本项目的关键科学问题：人工林生态系统生产力形成的制约机制。

# 内容2：结构调控对人工林生态系统养分平衡影响机制

土壤养分供给及其平衡状况已成为杉木、落叶松人工林生态系统生产力形成、维持与提高的核心问题。本项目以人工林生态系统野外定位观测站和各级相关实验室为依托，从林木个体和林分（系统）两个层面开展人工林生态系统养分平衡研究，具体技术方案：1)在林木/个体尺度上，利用野外调查、控制试验、非损伤微测技术、实时荧光定量RT-PCR技术、稳定同位素示踪、生态化学计量等研究方法与手段，揭示林木根系分布特征与NP吸收、利用的时空格局，探讨菌根菌对林木NP吸收利用的影响机理，解析根茎（干）叶NP含量动态及其生态化学计量规律，探讨林木NP吸收、利用与归还对系统养分平衡的调节机制。2）在林分/系统尺度上，研究不同林分结构（纯林及其混交林）人工林生态系统土壤NP贮量及其分布格局；通过林下植被管理、施肥管理（土壤NP有效性）及残落物管理等原位控制试验，采用微根管技术，研究林下植物和根际微生物、细根周转对土壤NP矿化及其有效性的影响机制，杉木人工林不同代际、落叶松人工林不同更新方式（人工更新、萌芽更新与近自然更新）对土壤养分供应及其化学计量比的影响。上述研究最终解剖出土壤养分平衡以及人工林长期生产力的维

持机制。

# 内容3：人工林生态系统主要生态功能与木材生产的权衡关系

针对人工林经营方向的转变，从单方面强调木材生产到兼顾人工林生产与生态功能相协调过程中存在的现实问题，重点解析人工林生态系统主要生态功能（地下生态系统碳固持、系统生物多样性）与木材生产的关系；具体技术方案：1）以杨树无性系人工林为主要研究对象，采用定位动态观测、野外典型样地调查和田间控制试验相结合的研究方法，并运用物种多样性测度方法（Simpson 多样性指数、物种均匀度、生态优势度等）和 PCR-变性梯度凝胶电泳等方法，研究林分结构（造林密度、种植点配置、树种混交）和抚育措施（间伐、施肥、松土除草、割灌）对生态系统生物多样性和生物量分布格局的影响，揭示土壤微生物和动物多样性对人工林植物多样性的响应，探讨杨树人工林生物多样性（基因多样性、植物多样性、土壤微生物和动物多样性）与现实生产力关系。2）通过野外调查、定点控制试验、长期试验和历史数据模拟，分析集约经营强度与不用经营措施（间伐、修枝、施肥和残落物管理等）对蓄积增长和木材质量的影响规律，以及对地下生态系统碳库动态与构成的影响；采用  $^{14}\mathrm{C}$  自然丰度法跟踪采伐根桩对土壤有机碳库的影响；模拟人工林周期性采伐、更新（人工更新、近自然更新）对土壤有机碳库动态变化过程，探讨地下碳库与地上生物量碳积累之间的内在关系。上述研究最终将提出权衡人工林生态系统生产功能和生态功能之间关系的调控对策。

# 内容4：人工林生态系统功能评估与结构优化

在上述三个研究内容的支撑下，从提高人工林生产力、保持人工林可持续发展、协调人工林生态功能与生产功能关系的多目标出发，以我国主要人工林（杉木、落叶松、杨树）生态系统为对象，将上述研究尺度（林木尺度、林分/系统尺度）的结果扩展到景观/区域尺度，系统实施如下研究：1）林分和景观结构量化方法与优化：在林分尺度上，通过测定林分透光分层疏透度量化表达人工林垂直结构；在景观尺度上，探讨人工林生态系统空间分布格局（结构）量化表达方法（包括景观稳定性、景观多样性、景观破碎度等）；以人工林生态系统生产功能（木材生产力为主）和生态功能（以固碳能力为主）的权衡关系为基础，建立多目标的林分结构、景观结构优化（包括代际更替）模式。2）结合地面定位实验和野外调查结果、多源遥感影像和模型模拟法（CASA模型），在GIS支持下综

合评估杉木和落叶松人工林生态系统在景观/区域尺度上的现实生产力与碳储量。

3）人工林生态系统长期生产力与固碳潜力的变化趋势与预测：根据人工林生态系统的特点，通过优化InVEST（Integrated Valuation of Ecosystem Services and Tradeoffs，生态系统服务和权衡综合评估）模型参数，以生态系统养分消耗和有机质积累过程为依据，在现有的人工林生态系统管理措施（模拟氮沉降、林下植被管理、间伐等）、林分结构、轮伐或代际更替方式（人工纯林、混交林更新、天然更新、人工诱导天然更新）等情景下进行模拟与预测。上述研究最终将为制定适合人工林生态系统功能提升的调控对策提供科技支撑。

# 3. 与国内同类研究相比的创新性与特色

# （1）创新性

# 1）关联地上与地下生态系统，揭示人工林生产功能与生态功能的权衡关系

生态学领域发展的趋势是把地上和地下作为一个有机整体进行关联研究。长期以来，在人工林生态系统的研究中，地上部分的生态过程、机制一直是生态学者关注的重点。同样，由于人工林生态系统结构、功能对经营管理措施和全球变化的响应依赖于地上和地下过程的紧密联系，因此，地上和地下的整合目前被认为是人工林生态系统研究的最有效途径。随着我国人工林由以木材开发为主向兼顾木材生产与生态服务功能为重点的转变，一系列重大科学问题亟待解决，如人工林生物多样性与生产力维持，地上和地下固碳功能、潜力等等，均需要把人工林和土壤（环境）作为一个完整的系统加以深入研究。本项目的实施将有利于全面揭示结构影响下人工林生态系统生产功能与生态功能的权衡关系，促进人工林生态学科发展，并为指导国家人工林发展决策提供理论依据。

# 2）建立多尺度人工林结构量化方法体系，具有普适性和创新性

林分尺度结构通过影响林内环境与生物因子，景观尺度结构通过改变区域人工林的稳定性，进而决定人工林生态系统的生产功能和生态功能的发挥。本项目提出量化不同尺度人工林结构的方法，可推动定量生态学的发展，同时将量化的人工林结构作为重要参数集嵌入到生态系统服务和权衡综合评估模型中，可精确预测人工林生态系统在结构要素、管理措施等影响下的长期生产力与固碳潜力。

# 3）多尺度和对比研究方法，更具说服力

本项目针对我国主要人工林生态系统，分别在林木/个体尺度上揭示生产力

低的光合制约机制和吸收、利用养分的机理；在林分/系统尺度上阐述生物量积累与转化过程，结构与系统养分平衡的关系以及生物多样性、地下生态系统碳固持与生产力的量化关系；在景观/区域尺度上评估人工林生态适宜性和预测人工林长期生产力与固碳潜力等变化趋势；尺度转换贯穿于每个层次。研究团队将按照统一方法对相同或不同树种的生态系统开展试验，对同一类型生态系统的研究将涵盖主要立地类型和经营管理方式，并与相应的天然林进行联网对比研究。这些将克服以往林学单一林分尺度短期调查为主研究存在的缺陷，成果将为国家决策、区域生态系统管理和人工林可持续经营提供科学支撑。

# (2）项目特色

# 1）前瞻性基础科学研究

生态系统结构与功能的关系及作用机制，是系统生态学研究的核心内容。目前，国内外过多关注的是草原、农业和天然林生态系统，忽略了人工林生态系统及其特有的关键科学问题，诸如在人为调控下的人工林生态系统可持续发展的自养机理、结构对生产功能与生态功能权衡关系的影响机制等。而我国人工林面积居世界第一，为实现人工林生态系统功能高效、稳定与可持续，开展人工林生态系统结构、功能与调控的基础研究具有前瞻性，同时为世界人工林生态学发展做出重大贡献。

# 2）构建中国人工林生态系统经营管理数据库

中国是世界上人工林面积最大的国家，面对我国人工林目前存在的生产力低、不可持续、生产功能与生态功能不协调等三大现实问题，亟需采取适当的经营管理措施解决。但是，目前的相关研究较分散，未形成一个完整的体系。本项目将以我国主要类型人工林生态系统（杉木、落叶松、杨树，约占我国人工用材林总面积的  $52\%$  ，总蓄积的  $65\%$  ），以及相应的主要立地类型、经营管理措施（间伐、林下植被管理、残落物管理、密度调控、树种配置、养分管理、代际更替方式等）为对象开展系统研究，建立人工林生态系统碳储量、林分蓄积量、系统生物多样性等参数的数据库，为提升人工林生态系统服务功能提供基础数据。

# 3）利用现有联网研究平台

项目研究内容是已有研究计划（中科院知识创新工程项目及国家自然科学基金）的衔接和补充。依托的研究平台将有机结合现有的CERN森林监测站和

CFERN 森林生态观测站，全国森林资源清查数据以及 NSTEC 调查数据等资源。

# 4）集人工林生产功能和生态功能于一体的多目标科学研究

人工林生态系统结构不仅决定木材生产功能，其物种配置模式与环境要素、生态过程、管理措施、代际更替等共同作用于生态功能；从不同的时空尺度，研究人工林生态系统结构、功能与调控将集生产与生态功能于一体，构建具有多目标综合性特色的试验平台与数据库。

项目主持和参加单位，具有国家、CERN相关森林生态站和CFERN森林站的长期实验基地、大型科学装置和数据支持，具有系列的环境因素控制装置，可进行加氮、降水、凋落物、植被、间伐等人工林控制实验，以及开展过程机理研究的大型分析仪器、专题制图和标准化数据，为项目取得突破性成果提供必要的平台基础。本项目依托和承担单位集合了相关国家重点实验室、野外森林网络实验站、国家级国际联合研究中心、省院部级重点实验室，具备较完善的技术平台。

我国主要人工林生态系统生产功能与生态功能权衡关系及自然驱动与人为调控机制，是本项目科学研究的重点和难点。提交系统和完整的人工林生态系统结构及功能动态数据，结合收集的历史数据，将为项目取得重大突破提供坚实的数据支撑。

# (2) 各课题的预期目标和研究内容

# 课题1、人工林生态系统生产力形成机理与关键过程

研究目标：

揭示以杉木、落叶松、杨树等为主要造林树种的人工林生态系统生产力形成及生物量分配的生理生态学机制，定量描述环境梯度、林分结构及管理措施对生态系统生产力的影响与调控过程，阐明人工林生态系统生产力形成的光合效能制约机制，解析导致我国主要人工林生态系统生产力低下的环境要因，进而提出解决生产力低下问题的调控对策与综合管理措施。发表国际主流期刊论文30篇，培养年轻学术骨干3-5名，博士研究生8-12名、博士后2-3名。

研究内容：

选择点（微观尺度，定点控制）面（宏观尺度，典型气候区）相结合的研究方法，在不同环境梯度、不同林分结构及不同经营措施下研究主要人工林树种（杉木、落叶松、杨树等)光合效率与生物量积累和分配的关系，确定人工林生态系统生产力分布格局及制约机制，分析人工林生态系统生产力低下的环境要因。

# 1）环境因子对人工林主要树种光合效能的影响机制

利用开顶箱或人工气候室在不同立地气候区对人工林主要造林树种进行光照、温度、降水、氮沉降与UV-B辐射多环境因子的交互实验，通过多因子的控制实验，利用Li-6400便携式光合作用测定系统及叶绿素荧光仪测定叶片光强度响应曲线、 $\mathrm{CO}_{2}$ 浓度响应曲线、光合速率（Rubisco酶的数量和活性）、荧光动力学（PSII的电子传递、PSII原初光能转换效率、PSII量子效率）、荧光淬灭分析；利用C14-同位素示踪技术研究主要造林树种在不同环境下光合产物的分配规律和分配格局的变化，以及光合产物在植被和土壤系统中的迁移过程及其与环境因子的关系；分析植物源-库-流的协调与调节机制及其在发挥光合潜势和整合碳流中的作用；探讨树种/品系对光合作用特征、光合作用产物分配以及光合作用效率的影响，揭示人工林主要树种对环境的适应性。

# 2）人为调控对人工林生态系统生产力形成的驱动机制

研究人工林生态系统在立地（土壤、地貌等）、经营管理（密度调控、养分管理、林下植被管理等）驱动下的生物量空间分配特征，比较生物量累积差异，探讨生物量累积规律的定量评估方法；采用生物量调查法获取研究时段的林木干物质量及最终产量，采用直接测定法（冠层分析系统等）或间接计算法（叶面积指数法、光谱指数法等）获取冠层辐射量，从而得到光能利用率；利用涡度相关法研究短时间尺度上光能利用率的变化特征；在此基础上，结合生物量调查法研究不同因素影响下生态系统光能利用效率的变化规律；研究不同空间尺度（叶片、冠层、种群、群落、生态系统）和时间尺度的人工林生态系统生物量在水分和温度梯度上的分布格局与响应规律，分析不同尺度过程之间的机理联系，定量描述立地变化与经营管理对人工林生态系统生产潜力的影响；确定主要树种人工林生态系统的生态效率，为提高光能利用提供理论依据和技术途径。

# 3）人工林生态系统生产力低下要因及调控对策

通过树种、立地和管理措施对人工林生态系统的光合产物分配、生物量积累与光合能量转化过程影响研究，建立人工林生态系统结构和生产力耦合关系，揭

以林分尺度上功能形成过程的驱动机理和本课题景观结构量化研究结果为基础，基于RS、GIS及过程模型，对典型人工林生态系统生态适宜性、衰退风险性及造林实施成效（环境效应)在景观尺度上进行评估，制定适合人工林生态系统功能提升的调控对策，为我国人工林生态系统可持续经营提供理论支撑。
