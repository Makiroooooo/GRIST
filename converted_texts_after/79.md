摘要  
(限400字):纳米材料在医药领域已经越来越受到重视,一旦应用于临床,将对于传统医药产生很大的冲击。虽然纳米生物技术的发展为我们带来了诸多诱人的优点,但在充分安全、有效地将纳米载药体系用于临床应用前,解决纳米载体的释药性、分散性、稳定性、生物相容性、降解性、纳米仿生材料和生物组织的相容性、毒理性等方面的问题,都需要进行系统的研究。而纳米药物的生物相容性、毒副作用等研究将是其应用于临床的基础实验,也是目前纳米药物研究的最前沿。纳米材料应用于生物领域必须具有良好的生物活性和相容性特征。融合纳米与生物两项技术研究方法,研究无机和有机纳米材料的生物活性修饰原理,及趋磁细菌生物合成的纳米磁性颗粒的生物膜特征及重建基础,使纳米材料的生物活性和生物利用度得到大幅度提高;并从化学组成和结构方面考察纳米材料的生物活性,并对纳米物质的生物相容性进行研究,促进纳米材料在生物医药领域得到实际应用。

# 报告正文  

# 一立论依据与研究内容  

# 1. 项目立论依据  

纳米生物技术目前在医疗的诊断与治疗方面国内外进行了广泛的研究，其中包括纳米载体载基因或药物对各种肿瘤进行治疗；纳米生物传感器；作为生物可降解材料应用于骨损伤修复、心血管支架、口腔科材料等。目前应用于临床的纳米药物非常少，其中就有纳米材料的生物相容性、细胞毒性等无法完全确定的原因。通过本研究项目从而获得纳米粒相关方面的参数，对于促进我国纳米生物技术的发展具有重大的意义，故立题具有必要性及可行性。  

在医学领域中，最引人注目的是作为药物载体，或制作人体生物医学材料，如人工肾脏、人工关节等。在纳米铁微粒表面涂一层聚合物后，可以固定蛋白质或酶，以控制生物反应。国外用纳米陶瓷微粒作载体的病毒诱导物也取得成功。  

用表面活性剂对纳米粒进行表面修饰可能减少纳米粒在体内对网状内皮系统的靶向性。实验结果可见用Myrj53和Myrj59对载环孢素A聚乳酸纳米粒用物理吸附法进行表面修饰后在体内没有减少而是增加了其对肝的靶向性，原因可能是本实验所制备的NP的粒径较小(59nm)，粒径较小(<70nm)的胶体亚微粒对肝的渗透性较强，渗透性的增强使NP在肝发生了主动的聚集，而表面修饰的结果是增加了这种渗透性。王杰的研究表明将CyA包埋于普通纳米粒中，可以提高CyA对巨噬细胞的靶向性而提高免疫抑制作用，但其缺点是这将使CyA在体内的清除加快。如果将CyA包埋于长循环表面修饰的纳米粒中，将使CyA在体内的循环时间延长，即提高了生物利用度，而游离的CyA本身对免疫细胞具有靶向性，因而将CyA包埋于长循环纳米粒中相对于普通纳米粒而言表面上是降低了对巨噬细胞的靶向性，实际上可能是疗效的明显改善。  

生物相容物质的开发，是纳米材料在医学领域中的另一个重要应用。是因植入物的不相容性所引起炎症反应或排斥反应，而纳米物质所展现的非凡的物理特性正好能解决这一难题。用纳米技术制造的纳米物质与其在自然界中的常规状态相比，其物理性质有着巨大的区别。1994年，英国Bonfield成功地合成了模拟骨骼亚结构的纳米物质，该物质可取代目前骨科常用的合金材料。它的主要成分是经与聚乙烯混合压缩后的羟基磷灰石网（骨骼的主要成分），其物理特性正好符合理想的骨骼替代物的模数匹配（Modulus matching，例如具有与骨骼相似的强度和密度指数），不易骨折，且与正常骨组织连接紧密，显示了明显的正畸应用优势。其他可望应用于临床的纳米物质有人工关节面和关节腔、美容植入物、口腔正畸物等。  

药物合成化学的中心点是分子，目前已经发展了组装分子的十分复杂的步骤，即根据共价键顺  

序形成的一般方法，同时形成一个或几个化学键。其中一个是与纳米结构最为相关的方法——分子的自组装。即分子自发组装成结构上稳定的非共价键连接的聚集体。它不仅坚固，而且自身具有对抗外界物质的结合能力。目前，尽管化学合成纳米结构的药物分子的制作方法不像一般微加工那样发展较快，但它提供了控制单个原子的选择和移动的诱人方法，因而最终将得到更大的发展。  

生物传感器是当前国际国内研究的热点课题，研究酶固定并保持其活性是所要解决的关键问题，剪裁金属纳米颗粒的新性质和新效应，将超细纳米颗粒引入酶膜，研究以高效率工作的灵敏、价廉、微型化分子识别反应的酶生物传感器器件，增强酶电极的响应电流并提高其灵敏度和使用寿命。  

纳米生物材料是当前国际上生物工程技术领域的前沿和热点研究课题，其在生物工程医药领域应用方面的研究尚处于初级阶段，但却有着广阔的应用前景。纳米生物材料可应用于制造各类软组织的支架（如血管、气管、输尿管、韧带与肌腱），组织工程用支架材料，内固定件，药物控释载体材料，骨组织缺损修复材料，基因治疗药物携带材料或靶向材料。已有的纳米材料尚不能应用在生物领域，其主要原因之一是这些纳米材料缺乏生物亲和性和生物相容性等。  

具有生物活性和靶向作用的纳米材料，可作为治疗恶性肿瘤化疗药物的载体，通过其靶向作用，提高病灶局部的药物浓度，有效杀死癌细胞，降低全身的毒性作用。  

具有生物活性和可控降解的纳米材料，同时具有几种特殊功效：可作为细胞生长的支架或药物基因载体，通过自身的生物降解控制在体内保存的时间和载体内药物释放的作用。特别是随着基因工程和生化合成技术的发展，越来越多的多肽、蛋白质药物被合成并在临床应用中表现出许多独特的疗效。然而，这类药物存在体内易降解、易失活，在肠胃内被吸收难于到达病灶区域等缺点，阻碍了在临床上的有效使用。以生物降解性高分子等生物材料作为药物载体，控制多肽、蛋白质药物在体内持续释放，改变给药方式，可提高药物疗效。  

具有良好生物活性和相容性的纳米体内植入物，如人工心瓣膜，全人工髋关节，人造大血管置换，将大大减少以这些生物医用材料为中心的感染发生机会。  

纳米材料在医学研究和临床有广泛的应用前景，但纳米颗粒是纳米材料的基本构成单元，颗粒的性能不仅取决于颗粒的大小尺度，而且取决于它的形貌、结构。纳米粒子由于纳米尺寸而具有表面效应，体积效应，量子尺寸效应，宏观量子隧道效应等特性，但同时纳米尺寸又使颗粒表面具高活性，极不稳定，形态及结构难以控制，从而严重妨碍了纳米材料优异性能的发挥。要解决这一矛盾，对纳米结构材料进行表面生物化学修饰，引入各种活性基团，通过这些基团实现与基体材料的复合，可以有效地形成无机、有机或复合纳米材料的生物活性，赋予材料以特殊的光、电、磁等功能特性，防止粒子团聚，增加其稳定性和分散性，使其在医学研究和临床应用成为可能。  

我们在前期工作过程中启动了磁纳米粒治疗肝癌研究，制成了一批磁性纳米颗粒，研究了磁性阿霉素白蛋白纳米粒在正常大鼠肝脏的磁靶向性、磁性阿霉素白蛋白纳米粒在正常大鼠体内的分布及其对大鼠移植性肝癌的治疗效果。结果表明，磁性阿霉素白蛋白纳米粒具有高效磁靶向性，在大鼠移植性肝癌中的聚集明显增加，而且对移植性肝癌有很好的疗效。但发现良好的磁性纳米药物需  

要载体能稳定的携带药物，其关键在于载体的生物活性及其和药物的结合特性。  

针对上述问题，本课题融合纳米与生物两项尖端技术研究可行的方法，对纳米材料的表面进行修饰，实现纳米材料的生物亲和性；同时从化学组成和材料结构两个方面考察纳米生物材料的生物活性和相容性，促进纳米颗粒在生物医药领域得到实际应用。

# 2. 研究内容、研究目标和拟解决的关键问题  

# 研究目标：  

(1) 研究无机纳米颗粒  $\left(\mathrm{Fe}_{3} \mathrm{O}_{4}\right.$  颗粒、软磁锰锌、铁氧体、  $\mathrm{SiO}_{2}$  、碳纳米物质等)、有机纳米材料（高分子丙交酯、乙交酯、丙交酯乙交酯粉体、等）和天然生物纳米颗粒（细菌磁颗粒）的生物活性改性原理和技术；  
(2) 研究上述纳米生物材料的生物相容性基础；  
(3) 建立纳米材料的生物活性和生物相容性评估体系。  

# 研究内容  

(1) 无机纳米颗粒  $\left(\mathrm{Fe}_{3} \mathrm{O}_{4}\right.$  颗粒、软磁锰锌、铁氧体、  $\mathrm{SiO}_{2}$  、碳纳米物质等) 的表面生物活性分子固定技术原理和技术、生物相容性基础; 有机纳米材料（高分子丙交酯、乙交酯、丙交酯乙交酯粉体、聚乳酸等）的自组装单分子层生物表面改性原理和技术、生物相容性基础。  

1）合成出一系列性能不同的无机生物材料、有机高分子生物材料、有机/无机材料。  
2) 将所合成的生物材料进行纳米化，采用不同方法研究纳米载药体系的构建与组装过程，对纳米载药体系进行全面表征的探讨。  
3) 探讨不同的纳米载药体系在不同动物、不同的给药途径对不同生物粘膜的影响, 阐明纳米药物与生物膜的相互作用机理。并考察纳米载体经不同途径输入体内后, 在各器官的分布及其随时间的变化规律的研究, 明确纳米载体在体内的时空分布规律。  
4) 筛选某特定细胞株的最佳抗体, 将抗体与纳米载体结合, 探讨抗原与抗体分子的作用规律及对纳米药物体内作用过程的影响。  
5）获取实时监测注射后动物的呼吸和循环指标，完成纳米材料对血流动力学和血管内皮的影响研究。  
6) 探讨药物与载体在不同阶段的各种相互作用，阐明纳米药物理化性能的调控机制。  
7) 研究评价药物分子与载体分子的相互作用, 弄清药物与载体分子的相互作用对纳米药物体内外各种行为的影响。  
8) 在不同层次上考察生物膜的转运方向、生物膜载体的作用等, 阐明纳米药物的转运机制与输送策略。  
9) 测定重要脏器的供血情况的测定, 测定血流阻力、血液粘度、血小板聚集率及检测血管内皮细胞完整性和其分泌功能的改变, 明确纳米载体对重要组织器官、免疫系统、血液系统的影响。  
10) 完成对主要脏器的功能检查，明确纳米生物材料的毒理学效应。  
11) 观察纳米材料进入生物体后的生物可降解度及代谢方式、排泄途径等，建立纳米载体降解的代谢动力学方程。  

(2) 天然生物纳米颗粒（细菌磁颗粒）的生物活性原理、生物相容性基础。

# 4. 拟采取的研究方法、技术路线、实验方案及可行性分析  

通常用于制备纳米颗粒的核材料的物质大都不能直接与生物分子相互连接，或者这种相互连接可能影响功能材料的特性与生物分子的活性。为了使纳米颗粒表面功能化实现预期的应用，必须使纳米外壳具可修饰性。我们的技术路线是：在纳米颗粒装上生物活性分子和自组装单分子层生物表面改性；并通过研究天然细菌磁性颗粒表面生物膜的特征，在人造无机纳米颗粒和有机纳米颗粒构建生物膜。  

# (1)纳米颗粒表面生物修饰  

根据目标物的分子结构，设计出选择性强、作用专一、无副作用的表面活性分子，从而通过固定不同生物分子实现核壳纳米颗粒的生物亲和性，扩大纳米颗粒的应用领域。为了实现生物细胞分离，磁性纳米颗粒表面生物活性分子修饰技术是关键技术之一。根据生物体内“目标”分子结构特征，选择仅与“目标”（如癌细胞）有亲和力而对其它非目标物不敏感的生物活性剂后，对核壳磁性颗粒表面固定化。  

# （2）纳米颗粒表面化学修饰  

借鉴药剂分子设计原理，设计、选择合成表面修饰剂。表面修饰剂分子应与磁核颗粒和生物活性分子作用力都很强。  

亲固能计算：应用普遍化的微扰理论建立表面修饰剂与纳米颗粒表面相互作用的能量方程，依据能量最低原理的能量盘踞确定它们相互作用力的大小，运用量子化学方法（Guussian软件）计算拼合不同官能团的表面修饰剂与纳米颗粒相互作用的能量大小。  

表面修饰剂与生物活性分子相互作用能：依据能量最低原理的能量判据确定表面修饰剂与生物活性分子作用能力的大小，运用分子力学方法（OFF软件）定量计算生物修饰剂与表面活性分子相互作用能。

# 纳米物质的血液相容性  

研究纳米物质进入血液循环系统时,对血小板、白细胞系、红细胞、补体系统以及凝血系统的影响。研究纳米物质通过哪种途径激活机体的凝血系统，研究纳米物质的血细胞和血浆蛋白质吸附、材料表面电荷、湿润性、表面自由能、微观结构和对功能基因的影响等方面。  

在纳米颗粒显示高亲合力的组织器官取标本，用表达谱基因芯片检测纳米药物对靶细胞基因表达的影响，并找出相关基因或基因产物。  

# 生物可降解性：  

进行不同组织细胞的体外培养，观察纳米颗粒的降解速率，确定不同材料在体内降解的主要组织器官和组织细胞。  

纳米颗粒注入机体后：观察其在各组织中含量随时间的变化；观察血液中浓度的变化；测定尿液、粪便、胆汁等途径的排泄速率；建立降解的动力学方程。
