# 报告正文

# 纸载体绳播种技术关键装备的应用基础研究

# 1 立项依据与研究内容

# 1.1 项目的立项依据

# 1.1.1 本项目的研究意义

播种是作物种植过程中的重要环节，其作业质量的好坏直接影响产量。常见的播种方法有裸种播种和包衣播种两种形式。传统的播种作业是在耕整后的土地上进行开沟、播种、施肥、施除草剂、覆土、镇压等工序的田间作业过程，其本质特征是直接将裸种播入田间。它的优点是操作简单，对生产工具要求不高，可以利用人工刀耕火种，也可以利用现代化的农业机械进行大规模作业。包衣播种是在种子的外表包上一层具有肥效作用的可溶性物质，使种子的形状更加规则，有利于机械化播种也有利于作物出苗。对于裸种播种和包衣播种来讲，为了实现机组的行进动作与排种和排肥等动作的同步，排种器和排肥器等的动力必须来自于地轮（镇压轮或专用的行走轮）。由于土壤坚实度、含水量、物质构成和地表形状等对地轮的滑移率有较大影响，因此播种机的作业质量，如条播机的播种量和排种均匀性，穴播机的穴距和每穴粒数等受土壤条件的影响较大。

本项目提出了纸载体绳播种理论，其核心是将作物的播种作业分两步进行。第一步，利用研制的关键装备——种绳捻制机，在工厂内将作物种子、肥料和除草剂等按农业技术要求卷在纸带内，制成种绳盘。第二步，到田间，利用专门研制的种绳直播机进行开沟、放种绳和覆土镇压。在该理论指导下的纸载体绳播种植技术，能够实现种绳捻制作业在空间上不受土壤及其它条件限制，时间上不受农时季节限制，在技术上保证种子、肥料和除草剂的相对位置和每穴用量等性能参数的标准化，为订单农业奠定技术基础，最终实现田间作业的简单化和轻型化。纸载体绳播种植理论，不仅适用于水稻、大豆、玉米等粮食作物，也适用于蔬菜、花卉等种子价格高、穴距要求严格、肥料和农药施用量精确的经济作物；不仅可以应用于裸地栽培，也可应用于保护地栽培；不仅适合大规模农业生产，也适合小规模农业生产。

纸载体绳播技术的关键装备是种绳捻制机械，种绳捻制质量直接影响该技术优势的充分发挥。按照农业技术要求，被包容物（种子、肥料和除草剂）在种绳内应满足一定的数量要求，不能混杂并应各自保持一定的相对位置精度；被包容物料在种绳捻制过程中的脱落率和在运输和播种过程中的泄漏率低；种绳应具有较高的抗拉强度；粘结剂和包容材料（纸带）等不能影响种子发芽和出苗，也不能污染土壤环境。

本项目拟在研究主要材料物理机械特性的基础上，对其进行筛选；揭示主要因素对

种绳性能参数的影响规律；建立多种物料在动态条件下的精确定位与包容控制方法，为种绳捻制装备的研制和纸载体绳播技术的应用奠定基础。

# 1.1.2 本领域研究的国内外现状分析

以将种子直接播入田间为本质特征的传统种植技术经历了几千年的历史，近年来出现的玉米侧深施肥技术，大垄双行技术，米麦套种技术，水稻直播技术等仍属传统种植技术。但是，在这种传统的种植技术条件下，若实现精量播种、施肥和施除草剂，则对耕整地质量和机具性能的要求较严格，成本较高；若不实现精量播种，则浪费种子、农药和化肥等。以美国为代表的发达国家，于20世纪70一80年代在精量穴播领域做了大量研究，但由于技术要求和作业成本太高而逐渐被放弃[4]。

20世纪80年代美国等发达国家转而研究精确农业，从本质上看，精确农业是在传统种植技术的框架内进行的，其实质仍是将种子直接播入田间。所谓精确农业(PrecisionFarming或PrecisionAgriculture)是利用3S技术，根据田间每个操作单元的具体条件进行变量播种、变量施肥、变量施药，以增加效益、减小浪费、保护环境的农业生产管理技术。美国首先从“玉米带”开始，主要研究变量施肥和变量喷药，目前该技术已经在多种作物和土壤上得到了应用，但这些技术的最小变量地块单元是1公顷[1][3][7][14]，中国、澳大利亚、加拿大、德国等国的研究结果表明，该技术的应用条件是大规模农业经营、高度发达的农业机械化以及农业信息化[6][9][10][11][12][24]。理论上讲，智能化的播种机能够根据实际情况实时地改变播种量、施肥量和喷药量等，即构成所谓的线性控制系统；但从控制技术角度来考虑，播种机具不可能实时地调整上述参数，否则相关部件将过度磨损，同时也将过度消耗能源。因此，这些装备必须设计成带有死区的非线性系统，即只有在工作条件差异达到一定程度时才启动变量调整系统[10]。然而，现阶段我国的农情是：(1)以家庭为单位的农业生产规模小，适合精确农业的大规模农业机械系统无法施展；(2)农民文化水平较低，经济承受能力较弱；(3)农业信息化程度较低。可见，精确农业在我国实际应用还需要相当长的时间。

纸载体播种技术有利于控制杂草、节水和促进机械化，比较适合我国国情。日本的上野秀人等研究了再生纸覆盖水稻直播技术，其核心是在再生纸上按照行距  $30\mathrm{cm}$ ，株距  $15\mathrm{cm}$ ，用打孔器打出直径  $2.5\mathrm{cm}$  的小孔，在孔的周围涂上粘结剂，将5粒稻种夹在2块直径  $5\mathrm{cm}$  的网状纸性无纺布之间，粘贴在打好的孔上，制成再生纸直播膜，卷成筒状，然后用铺膜机铺在耙后的水田表面，研究结果表明该技术生产成本太高，只适合生产价值较高的有机稻米[5][15][16]。我国的李显生等人，对水稻纸载体绳播技术的节水问题进行了试验研究，证明了该技术具有节水增效的作用，但未涉及该技术的基础研究[19]。

本课题组从1998年开始，研究了纸膜覆盖水稻种植技术的节水、灭草、防病、环保和增效机理和工艺规范。2001年开始，研究了水稻种绳直播种植技术的最佳种植时间、开沟深度、覆土厚度、中耕除草、水管理等方面的技术特点。

农业物料的物理机械特性，在农业工程学科中有着广泛的应用。种子、肥料、除草

剂等材料的摩擦特性和流动特性对种植机械的工作性能有较大影响，但相关研究很少[25]。李雪珍等采取实验方法研究了杂交水稻种子的自然休止角、外摩擦角和内摩擦角[23]。有关肥料和除草剂的摩擦特性和流动特性的研究鲜见相关报道。

在多物料动态精确定位与包容控制研究领域，种绳捻制工艺与用于包装的纸绳和用于制造鞭炮的药捻制作工艺比较相近。所不同的是用于包装的纸绳，其内不卷任何物料并且有些纸绳是多股的；用于制造鞭炮的药捻，火药在药捻内的分布要求是连续的。而对用于播种的种绳，要求其内种子、肥料、除草剂等物料的分布是间歇的，并且有严格的穴距和物料在穴内相对位置要求。用于包装的纸绳和用于制造鞭炮的药捻捻制技术已有相当长的历史，其捻劲机构、卷盘机构均可以借鉴，但有关材料筛选、系统机理与参数优化等基础性问题的研究几乎是空白。

检索结果表明，有关纸载体绳播技术所需肥料、除草剂、粘结剂和包容材料的物理机械特性，多物料动态精确定位与包容控制方法，各主要因素对种绳性能参数的影响等问题的研究，国内外尚未见相关报道。

# 1.2 研究内容、目标及拟解决的关键问题

# 1.2.1 研究内容

# (1) 多种物料在动态条件下的精确定位与包容控制

纸载体绳播种植技术的关键是捻制高质量的种绳。高质量种绳的重要指标之一是：种子、肥料和除草剂按照农业技术要求的数量和相对位置被卷在可降解的包容材料内。为此，首先要在系统动力学理论指导下，构造一个种绳捻制系统，然后再利用计算机模拟分析与实物动态模拟试验相结合的方法，研究系统各环节之间的运动学和动力学关系，对各环节的结构和参数进行优化，探索多种物料在动态条件下的精确定位与包容控制方法。

# (2）种绳性能参数的主要影响因素

种绳的性能参数主要有：种绳的抗拉强度，密封性（种绳捻好以后种子、肥料和除草剂不泄漏），种子发芽率，出苗速度等。拟在试验统计学理论指导下，通过正交试验的方法，探索种绳单位长度上的捻劲数；种子、肥料和除草剂的包容量；粘结剂涂刷量等主要因素对种绳性能参数的影响规律，确定最佳的参数组合。

# (3）原材料的物理机械特性

肥料和除草剂的选择与作物的品种等因素有关，为了后续研究工作的顺利进行，需要以一种作物为研究对象，对其进行筛选。拟在农业物料学相关理论指导下，以水稻种

植为基本研究条件，试验研究被包容物料（肥料、除草剂、种子）等材料的流动特性（内、外摩擦角和自然休止角）以及包容材料（简称纸带）的抗拉强度和腐烂规律，试验研究粘结剂原料品种、浓度和涂刷厚度等因素对被包容物料粘结效果的影响，最终筛选出纸载体绳播技术通用的纸带和粘结剂等材料。

# 试验过程中主要检测与观察指标如下：

1）种子、肥料、除草剂与塑料板、镀锌板和木板之间的外摩擦角；2）种子、肥料、除草剂的自然休止角和抗压强度；3）土样的氮、磷、钾、有机质、腐殖酸、酸碱度和含盐量；4）纸带的初始抗拉强度；5）种绳在播种前和播种后一定时期的抗拉强度；6)种绳单位长度上的捻劲数；7）种绳单位长度内的种子、肥料和除草剂包容量；8）粘结剂涂刷厚度；9）种子、肥料和除草剂在种绳上的相对位置及其分布规律；10）在种绳捻制过程中，种子、肥料和除草剂的脱落量；11）将种绳从卷制好的盘上退下，检测种子、肥料和除草剂从种绳上的泄漏量；12）绳播后种子的发芽时间和出苗时间及其相应的数量；13）种绳捻制过程中，纸带和种绳所受的张力；14）纸带放送环节的转动阻尼系数；15）种绳卷盘环节摩擦驱动机构的摩擦系数。

# 1.2.2 研究目标

筛选出纸载体绳播技术所需的粘结剂和纸带等主要材料；揭示单位长度上的捻劲数，种子、肥料和除草剂包容量，粘结剂涂刷量等主要因素对种绳抗拉强度、种子发芽率和出苗率等性能参数的影响规律；建立多种物料动态精确定位与包容控制方法，为种绳捻制装备的研制以及纸载体绳播种植技术的应用奠定基础。

# 1.2.3 拟解决的关键问题

# (1) 种子、肥料和除草剂在动态条件下的精确定位与包容控制

种绳制作过程中，首先要将种子、肥料和除草剂按照一定的数量和位置精度放在纸带上，然后扭转纸带将其捻成绳状，形成所谓的种绳，最后再将种绳卷成盘。与普通的播种不同，在普通的播种过程中种子、肥料和除草剂等物料直接落在相对静止且比较松软的土壤上，这些物料在落到地面后，其位置一般不会有太大变化；而在种绳捻制过程中，种子、肥料和除草剂是落在宽度较窄并且以较高速度运动的的纸带上，若系统各环节之间不协调、系统结构或参数设计不合适，这些物料很容易产生二次弹起或滚动，影响其位置精度，甚至从纸带上脱落。这正是本项目研究的难点所在。

为解决上述问题，拟构造一个由纸带放送环节，粘结剂涂刷环节，物料（种子、肥料和除草剂）排出环节，种绳捻劲环节和种绳卷盘环节等构成的种绳捻制系统。在系统动力学理论指导下，采取计算机模拟试验和实物动态模拟试验相结合的方法对系统进行集成，探索种子、肥料和除草剂在纸带上的精确定位与包容控制方法。

①纸带放送环节 要求纸带运行平稳、张紧力一致。在该环节上，纸带的受力主要来自于纸带放送轮的阻尼拉力和粘结剂涂刷环节的种绳拉力。由于系统的振动和粘结剂

涂刷环节的拉力波动等因素，可能造成纸带受力和运行速度的不稳定。研究该环节的动力学问题，对提高物料的定位精度有重要意义。

②物料排出环节 要求被排出物料的数量精确；物料被同步地输送到各自的排出口；物料在到达纸带时刻的运动速度与纸带的运动速度大小和方向相近。贯穿于系统各个环节的纸带，除了受到沿其运动方向的拉力作用以外，在种子、肥料和除草剂排出环节，还受到这些物料的冲击作用，以及来自种绳捻制过程的拉力波动和机架振动等因素的干扰。纸带在诸多因素的耦合作用下所表现出来的振动特性（幅值和频率）对种子、肥料和除草剂的精确定位有重要影响。研究该环节的动力学问题，可为系统设计和纸带材料的参数选择提供依据。  
③粘结剂涂刷环节 要求涂刷厚度满足要求且一致，对物料的固定能力强。在该环节，粘结剂涂刷部件、粘结剂和纸带构成了一个运动系统，三者之间既有相互作用力又有相对运动，粘结剂既受到压力作用，又受到剪力作用，研究该系统的动力学问题和粘结剂的流变特性，对提高粘结剂涂刷质量、稳定纸带的运行速度有重要意义。  
④种绳捻制环节 要求转速稳定、振动小，在物料排出段纸带不能扭转。在该环节，纸带除了受到沿其运动方向的拉力以外，还收到扭转力矩的作用，同时受到机架振动等因素的干扰。研究该环节的动力学问题，有利于提高种绳捻制质量。  
⑤种绳卷盘环节 要求种绳所受拉力不超过其强度极限，种绳在盘上均匀有序地排列。在该环节，种绳受力主要来自于种绳盘的拉力和种绳捻制环节的阻力，若两者大小相等则种绳所受拉力平衡。但由于种绳盘驱动机构的运动不平稳和种绳捻制机构的阻力波动等原因会造成种绳所受拉力不稳，严重时种绳被拉断。因此，有必要研究该环节的动力学问题。

# (2) 提高种绳的抗拉强度

在用于田间放种绳的播种机上，为了实现机器的行进动作与种绳的播放动作相协调，需要利用种绳的拉力驱动种绳盘转动，实现绳播。若种绳的抗拉强度不够，作业过程中容易被拉断。为此，在保证种绳捻制质量的同时，增加种绳的抗拉强度，是提高纸载体绳播技术作业质量的关键。

探索性试验结果表明，在肥料、除草剂、粘结剂、纸带和种子等材料确定后，种绳单位长度上的捻劲数，种子、肥料和除草剂的包容量，粘结剂的涂刷量是影响种绳抗拉强度的主要因素。拟通过正交试验的方法，探索各主要因素对种绳抗拉强度的影响规律并确定最佳的参数组合。

# 1.3拟采取的研究方案及可行性分析

# 1.3.1 拟采取的研究方案

# 1.3.1.1 研究方法、实验手段与关键技术

# (1) 主要原材料的物理机械特性

1）肥料和除草剂：以水稻种植为研究对象，以作用时间长、无环境污染为基本要求，取典型的固体化肥和生物肥料各5种、除草剂10种，利用物料外摩擦角检测装置，检测各物料与塑料板、木板和镀锌板等盛装和输送材料的外摩擦角；利用三轴仪检测肥料、除草剂和种子的内摩擦角；利用自制的自然休止角检测装置，检测上述物料的自然休止角。每个试验均重复10次。最后再以流动性好为主要指标，确定合适的肥料和除草剂。  
2）包容材料：以无环境污染且能在30天之内腐烂为基本要求，取无纺布纸带和牛皮纸带各3种，首先利用CMT3000微型电子拉力试验机检测材料的抗拉强度。再利用人工捻制种绳，不加肥料和除草剂，与普通直播相对照，在人工气候室内进行3次重复的盆栽对比试验。从播种5天后开始取土样，利用RL-3C型土壤分析测试仪，检测土壤中的氮、磷、钾、有机质、腐殖酸、酸碱度和含盐量变化，分析种绳对土壤的污染作用程度；通过观察与检测种绳抗拉强度相结合的方法，分析种绳的腐烂程度。以抗拉强度大、环境污染小和腐烂速度快为指标，综合考虑筛选包容材料。  
3）粘结剂：粘结剂的作用是把种子、肥料和除草剂固定在纸带上，确保它们在纸带内的位置不窜动并防止这些物质从卷好的种绳内泄漏出去。粘结剂应具有增加种绳抗拉强度的作用，但不应影响种子发芽和污染土壤环境。拟从以淀粉为主要原料的有机粘合剂中筛选3种，进行种绳捻制试验。以粘结剂原料品种、浓度和涂刷厚度为试验因素，进行3因素、3水平、3次重复的正交试验。利用高速摄影方法观察和检测种子、肥料和除草剂在纸带上的运动规律和定位精度，人工收集并检测物料在种绳捻制过程中的脱落量和捻制后的泄漏量。在人工气候室内进行绳播试验，每天取样一次，通过观察方法来检测绳播后种子的发芽时间和出苗时间及其相应的数量。最后，综合考虑以物料定位精度高、脱落量和泄漏量小、发芽率和出苗率高为主要指标，筛选粘结剂。

# (2）种绳性能参数的主要影响因素分析

以种绳单位长度上的捻劲数；种子、肥料和除草剂包容量；粘结剂涂刷量为试验因素，各取3个水平，构成3因素、3水平、3次重复的种绳捻制正交试验。采用CMT3000微型电子拉力试验机检测种绳的抗拉强度，研究各因素对种绳抗拉强度的影响规律；用与前述人工气候室绳播试验相同的方法，检测绳播后种子的发芽时间和出苗时间及其相应的数量，研究各因素对种子的发芽率和出苗率的影响规律，确定最佳的参数组合。

# (3）多种物料在动态条件下的精确定位与包容

拟在系统动力学理论的指导下，首先根据前期探索性研究成果结合理论分析，初步构建一个由纸带放送环节，粘结剂涂刷环节，物料排出环节，种绳捻劲环节和种绳卷盘环节等构成的种绳捻制系统。以种子、肥料、除草剂排出时的相对位置和物料对纸带的冲击作用等参数为输入，以这些物料在种绳上的相对位置和纸带运行速度等参数为输出，建立各环节和系统的数学模型，利用MATLAB软件进行系统分析和计算机模拟，探索系统中各环节的最佳结构形式和参数组合。在此基础上，采取实物动态模拟试验的

方法进一步验证上述结果、修改相关结构和参数，对该系统进行集成与参数优化。各主要参数控制方案如下：

1）种子、肥料和除草剂排量一致性与穴距准确性控制：拟采用三个相互独立的型孔轮式物料（种子、肥料、除草剂）排出装置且让三者同轴集中驱动，同时按照农业技术要求的各种物料每穴用量和穴距，优化设计型孔轮的直径和节距以及各自型孔的容量，以提高物料排量一致性和穴距准确性。  
2）种子、肥料和除草剂在穴内相对位置准确定位的控制：①按照农业技术要求的种子、肥料和除草剂在穴内的相对位置关系，优化各物料排出装置型孔的工作顺序。②建立该环节动力学模型，通过计算机模拟与动态试验相结合的方法，研究各参数对纸带运动特性的影响，再结合物料的摩擦和流动特性，优化各物料从排出装置的出口至纸带的运行过程中各自输送管路的材料、形状和行程，使物料按照要求的时间差，被同步地输送到各自的排出口且与纸带的运动速度大小和方向相近，减小物料与纸带的相对运动速度。③通过动力学分析与试验研究，优化粘结剂材料、浓度和涂刷厚度参数组合，以减小物料在到达纸带后的位置变动。  
3）纸带和种绳运行平稳性与张紧力控制：拟通过动力学分析，采取计算机模拟分析与动态模拟试验相结合的方法，探索：①纸带放送环节转动部分阻尼系数的最佳值，以达到纸带运行平稳、张紧力一致的目的。②在物料排出环节和捻绳环节之间的适当位置，设计一个纸带稳定环，以减小捻劲动作对纸带运行平稳性的影响。③采用摩擦传动方法驱动种绳卷盘机构，优化摩擦材料及其表面参数，通过合理丢转，解决在种绳卷盘过程中，因直径越卷越大，导致种绳所受张力越来越大，甚至被拉断的问题。

# 1.3.1.2 拟采取的技术路线

# (1) 肥料、除草剂、粘结剂和包容材料的筛选

1）筛选肥料和除草剂：以水稻直播为基本试验条件，按照农业技术要求初步选择肥料和除草剂的品种和每穴施用量  $\rightarrow$  试验研究所选物料的外摩擦角，内摩擦角和自然休止角  $\rightarrow$  综合考虑摩擦和流动特性、作用时间与效果和环境影响等因素，选择合适的肥料和除草剂。  
2）筛选包容材料：初选试验材料  $\rightarrow$  检测其抗拉强度  $\rightarrow$  人工捻制无肥料和除草剂的水稻种绳  $\rightarrow$  在人工气候室进行绳播盆栽试验  $\rightarrow$  检测土壤中各种物质含量变化  $\rightarrow$  观察和检测种绳的腐烂程度  $\rightarrow$  分析种绳对土壤污染作用程度以及其腐烂速度  $\rightarrow$  按抗拉强度大、对土壤环境影响小和腐烂快为基本要求，综合考虑确定包容材料。  
3）筛选粘结剂：按正交试验方法捻制种绳  $\rightarrow$  检测种绳抗拉强度、物料脱落量和泄漏量  $\rightarrow$  在人工气候室内进行绳播盆栽试验  $\rightarrow$  检测种子发芽和出苗的时间与数量  $\rightarrow$  以抗拉强度大、种子发芽率和出苗率高为基本要求，综合考虑筛选合适的粘结剂。

# (2）种绳性能参数的主要影响因素分析

以种绳单位长度上的捻劲数，种子、肥料和除草剂包容量，粘结剂涂刷量为试验因素，进行种绳捻制正交试验  $\rightarrow$  检测种绳抗拉强度  $\rightarrow$  人工气候室绳播栽培试验  $\rightarrow$  检测发芽和出苗时间及其相应的数量  $\rightarrow$  研究各因素对种绳抗拉强度、发芽率和出苗率的影响规律  $\rightarrow$  做必要的追加试验  $\rightarrow$  探索这些参数的优化组合。

# (3）多种物料在动态条件下的精确定位与包容控制

根据前期研究成果，初步确定系统各环节的功能和物理结构  $\rightarrow$  理论分析建立系统数学模型  $\rightarrow$  计算机模拟分析  $\rightarrow$  初步探索系统中各环节的结构形式和参数组合  $\rightarrow$  实物动态模拟试验  $\rightarrow$  检测种子、肥料和除草剂在纸带和种绳上的位置精度和包容数量  $\rightarrow$  修改相关结构和参数  $\rightarrow$  必要时重复前述过程  $\rightarrow$  分析确定系统的最佳结构形式和参数组合  $\rightarrow$  形成多种物料动态精确定位与包容控制方法。

# 1.3.2 可行性分析

本研究可能遇到的主要技术困难和解决办法是：

# (1) 纸载体栽培试验条件的控制

在包容和粘结剂的材料选择以及种绳性能参数影响因素分析过程中，需要进行纸载体绳播栽培试验，在田间试验条件下，土壤成分、环境温度、湿度、光照、 $\mathrm{CO}_{2}$ 含量等条件难以控制。通过人工气候室内盆栽试验的方法能够很好地控制试验条件。

# (2) 种子、肥料和除草剂在纸带上运动及其位置精度检测

在种绳捻制系统作业过程中，被包容的物料（种子、肥料和除草剂）及其包容物（纸带）均是运动的，观测并及时精确地记录这些被包容物料在纸带上的运动特性和位置关系比较困难。采用高速摄影方法观测和记录各物料在纸带上的运动及其位置，能够很好地解决上述问题。

# (3) 纸带和种绳在运行过程中所受张力的检测

在系统工作过程中，纸带和种绳均是运动的，其所受张力在振动等因素影响下是动态变化的，普通的拉力传感器无法安装，张力检测困难较大。采取电测技术，分别在纸带放送机构的转动轴和摩擦式种绳卷盘机构的被动轴上贴电阻应变片，形成由电阻应变片、动态应变仪、A/D转换器和计算机数据采集和分析等环节构成的电测系统，实时检

测和记录上述两组参数，并采用系统标定的方法消除误差。这是一个很成熟的检测技术，本课题组已在插秧机动态模拟试验等课题研究中多次成功地应用。

综上所述，前期已在种子、肥料、除草剂和包容材料的初选，各主要参数对种绳抗拉强度的影响，水稻种绳捻制机研制和水稻种绳直播技术等方面进行了探索性研究，初步证明了纸载体绳播技术在水稻种植和种绳捻制等方面的技术和经济可行性，相信本项目研究是可行的。
