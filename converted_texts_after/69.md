# 报告正文

# （一）立项依据与研究内容：

# 1、项目的立项依据（附主要的参考文献目录）：

肺癌的发病率在我国居常见肿瘤的前列，而且绝大多数肺癌患者就诊时肿瘤已达晚期，大部分病人失去了手术的机会。目前，化疗是治疗肿瘤的重要方法，约  $70 - 80\%$  的肺癌患者接受化疗[1]。化疗减轻病情的同时也对正常组织细胞、脏器的生理功能有一定程度的破坏作用，并抑制机体的免疫功能和骨髓造血功能等，从而影响化疗的继续进行。因此,防治化疗毒性反应是癌症治疗的重要课题。

天灸是将具有刺激性的药物贴敷在穴位上，使局部皮肤充血、发泡，甚至化脓，有如灸疮，故曰天灸。天灸通过刺激相应的腧穴或患处，调理脏腑，行气活血，调和阴阳，达到治疗疾病目的的一种外治法。因为其可以引起皮肤局部起泡所以也叫做“发泡灸”；相对于火灸、艾灸来讲，因为不需要热源，所以也叫做“冷灸”；利用药物本身对皮肤的刺激性就可以达到灸的效果，所以也叫做“自灸”。天灸是传统医学中较为独特的外治法，有数千年的历史，一直沿用至今，现也称为穴位药物贴敷疗法。它将中医透皮给药与经络穴位相结合，通过药物刺激经络、腧穴，对机体产生内在的调节作用，从而达到治疗疾病的目的。现代研究认为，天灸对皮肤产生刺激，扩张毛细血管，改善局部血液循环；刺激皮肤的神经感受器，通过神经反射达到提高机体免疫力的作用。另一方面由于天灸集艾灸、中药于一体，可以使腧穴局部自动加热有利于药物的吸收，药物的有效成份经皮肤表面的角质层、毛囊皮脂腺及汗管等渗透吸收进血液、淋巴液、内分泌液中，发挥一定的药理作用。

天灸疗法越来越受到学者和专家们的重视，尤其是近十几年来，在理论探讨和临床应用等方面，均有突破性的创新，并开始用于疑难杂症的治疗。例如赵氏[2]观察天灸治疗非感染性尿道综合征，与西药组比较，天灸组痊愈13(40.6%)好转14(43.7%)无效5(15.7%)总有效率84.3%；对照组痊愈2（6%）好转9(33%)无效19(61%)总有效率39%。两组比较差异有显著性（P<0.01）。说明天灸在治疗尿道综合症方面有明显的优势。发泡疗法用于治疗慢性气管炎[3]和支气管哮喘[9]有大量报导，还有用于三叉神经痛[4]、过敏性鼻炎[5][6]、乳腺纤维囊肿[7]、腮腺炎、风湿性关节炎、硬皮病[8]、肩周炎[10]、肱骨外上髁炎[11]、慢性乙型肝炎[12]、获得性寒冷性荨麻疹、额窦炎、梅核气、慢性溃疡性结肠炎、强直性脊椎炎[13]、胃脘痛[14]均获得满意的疗效，临床例证不胜枚举。在基础理论研究方面，据文献报道，天灸可调节化疗大鼠下丘脑、垂体中β-内啡肽（β-Ep）水平[15]，改善环磷酰胺大鼠的造血功能和免疫功能，提高荷瘤小鼠的T淋巴细胞的应答能力和NK细胞[16][17]、肿瘤坏死因子（TNF）的杀伤活性下者升高，趋向协调平衡，正是《内经》“阴平阳秘，精神乃治”理论的体现。

本研究选择肺癌作为研究对象，一是因为肺癌是我国最常见的肿瘤之一，积极开展中西医结合防治肺癌研究具有较大的现实意义；二是约  $70 - 80\%$  的肺癌患者接受化疗。化疗减轻病情的同时也对正常组织细胞、脏器的生理功能有一定程度的破坏作用，并抑制机体的免疫功能和骨髓造血功能等，从而影响化疗的继续进行。因此,防治化疗毒性反应是癌症治疗的重要课题，而天灸正是防治化疗毒性反应的临床和基础行之有效的方法。本实验通过研究移植性Lewis肺癌小鼠经天灸治疗和顺铂化疗后机体的免疫功能和肿瘤转移抑制基因的表达了解天灸在肺癌治疗过程中的增效减毒作用.本实验采用MTT法检测Lewis肺癌化疗小鼠脾脏NK细胞活性，采用放射免疫分析法检测Lewis肺癌化疗小鼠外周血白细胞介素-2（IL-2）水平，研究天灸对小鼠免疫功能的影响。此研究的意义在于为天灸在临床治疗肺癌过程中的增效减毒作用提供理论依据，为中西医结合治疗肺癌方面进行一些有益的尝试。

# 2. 项目的研究内容、研究目标以及拟解决的关键问题。

# 2.1研究内容：

（1）天灸对 Lewis 肺癌化疗小鼠的减毒作用。  
（2）天灸对Lewis肺癌化疗小鼠脾NK细胞活性及血清IL-2水平的研究。  
（3）天灸对Lewis肺癌化疗小鼠的癌细胞生长、转移及nm23-M1基因表达的研究。

# 2.2 研究目标：

通过天灸肾俞穴，观察其对Lewis肺癌小鼠体重变化、饮食情况、脾脏NK细胞活性、血清IL-2水平、肿瘤组织形态学、肿瘤转移抑制基因（nm23-M1）的蛋白表达、nm23-M1基因的mRNA表达的影响，为开拓天灸防治化疗反应提供基础研究，为临床应用提供科学的实验数据。

# 2.3 拟解决的关键问题：

（1）肾俞穴在小鼠身上的定位。  
(2) 小鼠脾脏 NK 细胞活性、小鼠血清 IL-2 水平的测定。  
（3）肿瘤组织形态学的观察。  
（4）肿瘤转移抑制基因的蛋白表达。  
（5）nm23-M1基因的mRNA表达

# 3.拟采取的研究方案及可行性分析：

# 3.1研究方案：
