项目名称：BNI融合的微纳传感器及其系统基础研究

起止年限：2006.1至2010.12

# 一、研究内容

# 1、拟解决的关键科学问题

本项目的核心是超微量快速检测微纳传感器。对传感器来说，目前存在的主要问题是灵敏度和选择性等特性不能满足超微量快速检测的要求，采用传统的方法已很难达到超微量快速检测的水平。当传感器进入纳米尺度后，利用相应的纳米尺度效应可以大幅度提高传感器的灵敏度，从而为实现传感器的超微量快速检测开辟一条新的思路。

为了提高微纳传感器的灵敏度和选择性，要解决的关键科学问题主要包括：纳米尺度效应及模型、生化特异性反应能量通过界面效应转换为微纳表征能量的机制、微纳制造原理与方法和纳米尺度下的特性表征。

纳米尺度效应及模型：当材料的尺度到了纳米尺度后，会产生明显的尺度效应。如硅材料在纳米尺度下其热传导、杨氏模量和压阻系数等特性不仅与材料的尺度有关，而且热传导系数变小有利于检测生化特异性反应产生的热量，杨氏模量变小则有利于在同样的应力条件下获得大的位移，压阻系数变大将可以大大提高压阻式传感器的灵敏度。因此，通过搞清材料的纳米尺度效应及模型，就可以将这些效应及模型用于微纳传感器的研究，从而大幅度提高传感器的灵敏度，研制出超微量快速检测微纳传感器。而且，这些效应及模型是材料的基本特性，其他微纳器件的研究也需要搞清这些问题，如纳电子器件的散热就与热传导纳米尺度效应密切相关。因此，这一科学问题的解决，对其他微纳器件的研究也有十分重要的意义。

生化特异性反应能量通过界面效应转换为微纳表征能量的机制：由于要检测的外界信息很多不是物理量，因此如何将需要检测的外界信息转换为微纳敏感结构可以检测的物理量是很关键的科学问题。利用生化特异性反应可以将要检测

的生物量或化学量转换成微纳敏感结构可以检测的物理量。由于生化特异性反应具有高度选择性的特点，因此利用这一特点可以大幅度提高微纳传感器的选择性。但是，生化特异性反应是如何与微纳敏感结构作用的以及反应能量是如何通过相关的界面效应实现转换的，目前还不是很清楚，搞清这些转换机制对超微量快速检测微纳传感器的研究是十分重要的。

微纳制造原理与方法：对于微纳传感器来说制造是基础，没有制造手段做保障就无法研制出微纳传感器，也无法制备出样品来研究材料的纳米尺度效应和相关界面效应，因此本项目要解决的另一关键科学问题就是微纳制造原理与方法，主要研究自上而下和自下而上微纳制造及其融合技术、跨微纳尺度的传感器理论模型和模拟设计问题，不仅要为研制出突发应急传感监测网络所需的超微量快速检测微纳传感器提供制造和模拟设计手段，还要为纳米尺度效应和相关界面效应研究的样品制备服务，并为进一步研究其他微纳传感器提供制造和模拟设计基础。

纳米尺度下的特性表征：对于纳米尺度来说，不仅制造难度非常大，而且特性表征也是一个要解决的关键科学问题。无论是纳米尺度效应和相关界面效应的研究，还是微纳制造和微纳传感器的研制，都存在如何在如此小的尺度范围内实现特性表征的问题。这一问题的解决不仅可为超微量快速检测微纳传感器的研究提供相应的表征手段，而且发展的表征原理与方法对其他微纳器件的研究同样有重要意义。

# 2、主要研究内容

当生化特异性反应信息在微纳敏感结构上转换成物理量后，采用什么样的微纳敏感结构将这些物理量转换成电信号，是一个非常重要的问题。本项目从结构的功能、通用性和代表性出发，拟考虑将三类微纳敏感结构：微纳机电敏感结构、仿生微纳敏感结构和基于碳纳米管的敏感结构来实现物理量到电信号的转换。

机电敏感结构（例如微纳悬臂梁等）特别适合将质量、应力应变、热、电荷、磁等物理量转换成电信号，因此，机电敏感结构一直受到人们的重视。机电敏感结构如果到了纳米尺度，其灵敏度将有质的飞跃，如它的质量检测分辨率在超高真空下可达  $10^{-18}$  克量级，特别适合超微量检测。而且，悬臂梁到了纳米尺度后，其力学、热学和电学等特性有许多尺度效应，以它为载体特别适合进行纳米尺度效应的研究。

世界上最早的传感器存在于大自然，实际上许多动植物由于生存的需要有非常好的敏感结构，灵敏度和选择性非常高，向大自然学习有可能研制出高性能仿生传感器。研究表明许多动植物的敏感结构就是微纳敏感结构，因此利用仿生微纳敏感结构可以研制出超微量快速检测微纳传感器。

碳纳米管作为一种性能优异的纳米材料，在快速、高灵敏检测方面有独特

的优势。而且, 碳纳米管是人们研究得非常多的纳米材料, 无论是制备方法还是材料特性, 有许多结果。因此本项目将充分利用碳纳米管的特点, 对基于碳纳米管的敏感结构进行探索, 从而研制出超微量快速检测微纳传感器。

基于上述考虑，本项目主要进行三方面的研究内容：生化特异性反应在微纳敏感结构上的信息转换机制、微纳敏感结构的纳米尺度模型与可控构筑方法、微纳传感器的超高灵敏检测与识别。其中生化特异性反应在微纳敏感结构上的信息转换机制主要研究如何利用敏感材料将生物量或化学量转换成物理量的问题，这方面将设一个课题；微纳敏感结构的纳米尺度模型与可控构筑方法主要研究微纳传感器的模拟设计、自下而上的制造原理与方法、自上而下的制造原理与方法，为微纳传感器提供模拟设计与制造基础，这方面将分别设三个课题；微纳传感器的超高灵敏检测与识别主要从微纳机电敏感结构、仿生微纳敏感结构和基于碳纳米管的敏感结构出发，研究相关的纳米效应及利用纳米效应研制出超微量快速检测微纳传感器，这方面将分别设三个课题。。

上述三方面的具体研究内容如下：

# 生化特异性反应在微纳敏感结构上的信息转换机制：

针对目标传感器，采用量子化学理论优化敏感反应效果，在微纳结构上进行表面修饰，将敏感材料固定于微纳结构表面，研究生化特异性反应机理以及生物量和化学量通过界面效应转换成物理量的机制，寻求提高灵敏度的方法，探索提高敏感材料共价交联的固定效率和稳定性方法。研究微纳尺度下界面效应对检测对象的扩散、富集、混合、表面反应等方面的影响，探索有效提高微反应速率的方法。合成筛选出用于炭疽杆菌和沙林等超微量检测的敏感材料。

# 微纳敏感结构的纳米尺度模型与可控构筑方法：

进行自上而下微纳制造技术研究。研究基于MEMS的纳米制造技术，探索纳米梁、桥、腔体、探针等典型纳米结构的制造技术；研究加工纳米尺度结构的新方法，主要包括基于刻蚀与腐蚀相结合的纳米结构加工方法，新型侧墙技术，基于聚焦离子束（FIB）等先进手段的纳米尺度局域化处理方法等；研究与纳米材料工艺兼容的微纳制造技术和阵列化微纳结构制造技术以及纳米结构的表征方法；为本项目研制超微量快速检测微纳传感器提供自上而下的微纳加工手段，为进一步研制其他微纳传感器打下工艺基础。

进行自下而上微纳制造技术研究。着重发展纳米结构及其器件的“自下而上”构筑新原理和新方法，实现按需构筑纳米结构体系；探索纳米结构生长机理，研究纳米尺度的材料物理化学特性，实现纳米结构可控生长(形态、相结构)；研究纳米结构与敏感性能间的关系，进行纳米结构表面化学、生物的修饰(包括官能团的接枝)，揭示纳米结构新的物理效应及纳米结构单元的物理/化学行为等，研究纳米结构与效应的表征方法，为研制微纳传感器件提供设计思路和制造手段。

进行自上而下和自下而上融合的微纳制造技术研究。研究在自上而下方式加工出的微/纳结构（如Si材料微/纳结构）上，进一步利用种子层或过渡层采用自下而上方式操纵、生长或组装纳米尺度结构（如碳纳米管或纳米带等）的方法，再进而研究制作成微纳传感器的工艺集成方法，包括驱动、检测元件的制作及集成工艺、敏感材料与敏感结构的工艺集成方法等。

研究应力（应变）和环境温度对微纳尺度材料或结构的电流影响的物理机制及数学模型构建；环境温度与电场渗透对力学性能的影响；微纳结构动态性能的能量耗散机制与数学模型；声子的能量输运特性，结构应变、晶格失配对声子输运的影响；原子级作用模型耦合到连续体模型中的方法；研究微纳结构材料的力电耦合、力热耦合、热电耦合和力热电耦合的模型及跨微纳尺度设计方法。纳米元件与生物大分子的相互作用模型；微纳米结构自组装模拟。

# 微纳传感器的超高灵敏检测与识别：

研究生物量与化学量转换为力、质量、热量和电荷等物理量传感过程中的能量转换原理，建立将生化敏感反应转换为电学量表征的传感模型；研究噪声机

制对超高灵敏检测的影响因素；探求突破现有分辨限制的方法。

研究机电、仿生微纳敏感结构和基于碳纳米管的敏感结构原理和相关的纳米尺度效应及表征方法，以及机械、热学、力学和电学等特性的微纳尺度效应对敏感特性的影响；探索限制敏感分辨极限的机制和表达模型，研究提高传感分辨极限和超微量检测的原理性方法，寻求符合超微量快速检测要求的传感器优化设计原理和结构模型。

研究微纳阵列传感中的关键问题，主要包括传感阵列邻近单元间的生化反应、电学信号的相互干扰问题，传感阵列多种敏感材料固化的方法及一致性、重复性问题，阵列与多功能传感器信息融合及解耦等问题。

研制出基于机电、仿生结构以及基于碳纳米管的炭疽杆菌、沙林和TNT等系列超微量快速检测微纳传感器，为进一步研究其他微纳传感器提供有较强覆盖能力的具有一定通用性的微纳敏感结构。

# 二、预期目标

# 1. 总体目标

本项目面向我国社会发展中的突发公共事件应急、环境监测、生化反恐等重大需求，以重要致病病原体（如炭疽杆菌等）、有毒有害化学物质（如沙林等）和烈性爆炸物（如TNT等）等的超微量快速检测为突破口，系统地开展微纳传感器基础研究工作。通过上述超微量快速检测微纳传感器的研究在两方面取得突破，一方面，通过“生物一纳米一信息”（BNI，Bio-Nano-Informatics）的学科交叉融合，在纳米尺度效应、界面效应、纳米尺度下的特性表征和微纳制造等微纳传感器基础研究方面取得突破，为其他微纳器件的研究提供理论、设计、表征和制造基础，推动我国纳米科技从材料研究向器件研究发展；另一方面，针对突发公共事件应急传感预警监测网所急需的超微量快速检测传感器，研制出原型器件，促进我国公共安全从被动应付型向主动保障型转变。同时，还将带动一批高水平研究基地的建设，培养一批微纳器件研究的高水平中青年专业人才。

# 2. 五年预期目标

- 针对微纳敏感结构与检测对象的特点, 探明相应的生化特异性反应在微纳敏感结构上的信息转换机制和界面效应, 提出在微纳敏感结构上形成敏感膜的方法, 建立生化敏感膜超微量响应性能的评价方法, 合成筛选出 5 种以上特异性强的敏感材料, 满足阵列微纳传感器超微量快速检测重要致病病原体 (如炭疽杆菌等) 和有毒有害气体 (如沙林等) 的要求, 达到空气中  $10^{-4} \mathrm{mg} / \mathrm{L}$  量级沙林的检测水平。  
- 根据本项目所研究的目标传感器对纳米结构的设计要求, 实现按需构筑;开拓拥有自主知识产权的纳米结构及其器件的构筑技术, 发展纳米敏感结构与器件“自下而上”构筑的新原理和新方法, 提出相应的表征方法,构筑的纳米结构单元尺寸一致性优于  $90\%$ , 为深入的功效研究和原理性器件系统的集成提供所需的基础。为研制超微量检测微纳传感器提供设计思路和制造手段。  
- 针对几类典型并具有相对普适性的微纳敏感结构,解决自上而下及其它组合方式下微纳敏感结构的实现问题,发展具有自主知识产权的基于自上而下方式的微纳制造及自上而下与自下而上融合的微纳制造新原理新方法,提出相应的表征方法,其宽度或厚度特征尺寸为  $10 \sim 100$  纳米,为研制微纳传感器提供制造基础。  
- 提出微纳结构或材料的力学、热学及其耦合模型、算法及其跨微纳尺度的设计方法。在理论上能够较精确的分析微纳结构力学、热学及其耦合

特性。初步考察结构的尺度、界面、表面粗糙度对性能的影响；建立描述界面、表面粗糙度、结构尺寸的电学、力学、热学及其耦合数学模型。初步建立纳米结构与生物大分子的相互作用模型，为设计和制造相应的微纳米器件与系统奠定理论基础。模型或设计方法与实验结果对比或与MD、MC和CA的原始算法比较，偏差小于  $10\%$  、计算效率提高10倍以上。

- 针对微纳机电传感器（如微纳悬臂梁等）超微量快速检测的需要，探明微纳机电敏感结构的纳米尺度效应与机制，提出相应的表征方法，建立将生化反应的机械量转变为电学量的表征方法和模型。揭示微纳机电传感器能量损耗（谐振 Q 值、塑性变形）和噪声特性进入微纳尺度的作用机制和随尺度变化规律，形成微纳机电传感器超微量快速检测的优化方法和结构。利用敏感元件产生的量子效应显著提高灵敏度。研制出 4—8 单元微纳机电阵列传感器原理样机，在检测环境中质量变化分辨达到  $10^{-12}$  克量级，对炭疽菌等重要致病微生物，实现数百个以内病原的检测。  
- 探明近气体分子自由程间隙内的气体电离传导规律及与气体种类的关系，利用碳纳米管高曲率表面的电离特性，实现碳纳米管气体传感器敏感方法与结构。解决表征和分辨技术中的关键问题，建立该类气体传感和检测的器件模型和理论基础。实现检测沙林等气体的智能化碳纳米管气体传感器原理样机，对沙林的检测分辨率达到  $10^{-4} \mathrm{mg} / \mathrm{L}$  量级，检测速度达到秒级。探明碳纳米管与微米波长红外光的相互作用机理，实现两个波段的红外探测器。  
- 建立高灵敏度、高选择性化学与物理敏感材料、结构与器件的仿生学方法与实现途径。合成和筛选出对TNT等炸药分子具有高度敏感性和特异性的纳米结构与分子印记材料，研制出TNT仿生嗅觉传感器，灵敏度达到纳克量级，并探明仿沙蚁导航传感器和仿壁虎触觉粘附阵列的微纳结构敏感机理与构筑方法，探索出采用非生物材料与结构再现生物敏感效应的科学途径，为发展长寿命超微量快速检测传感器奠定基础。

# 三、研究方案

# 1．总体思路

面向重大需求，牵引微纳传感器基础研究：本项目将从突发公共事件应急、环境监测、生化反恐等重大需求出发，针对所需的超微量快速检测传感器存在的灵敏度和选择性的问题，来开展纳米尺度效应、界面效应、纳米尺度下的特性表征和微纳制造等微纳传感器基础研究工作，寻找提高传感器灵敏度和选择性的途径，为相关的微纳传感器提供理论、设计及制造等基础。在此基础上，主要针对重要致病病原体（如炭疽杆菌等）、化学毒剂（如沙林等）和烈性爆炸物（如TNT等）等的超微量快速检测需要，重点研究三类原理先进、检测覆盖能力较强的阵列微纳敏感结构，并研制出超微量快速检测微纳传感器，满足突发公共事件应急等的需求。

通过学科交叉，突破超微量快速检测瓶颈：传统生化传感器在灵敏度和选择性方面还难以满足超微量快速检测的要求，其主要原因是还没有系统地解决生化反应与敏感结构的作用机制、敏感方法、传感结构和信息融合等问题，这些问题在单一学科的框架下很难解决，必须组织跨学科的队伍，通过学科交叉来解决问题。为此，本项目从学科交叉的角度出发，联合了国内的优势单位组成了跨学科的研究团队（包括微系统、传感器、纳米、生物、化学、物理、智能等学科），围绕同一个目标来攻克超微量快速检测微纳传感器中的关键科学问题，提高微纳传感器的灵敏度和选择性，从而突破超微量快速检测的瓶颈。

# 2. 技术途径

根据上述总体思路，本项目首先将从纳米效应和生化特异性反应与微纳敏感结构的作用机制出发，提出新的纳米尺度下特性表征原理与方法，发现新的效应，探讨其用于超微量快速检测的途径，发明新的超微量快速检测微纳传感器的结构，建立微纳传感器模拟设计和制造原理与方法，研制出所需的超微量快速检测微纳传感器。具体技术途径如下：

- 在我们已观察到硅的杨氏模量纳米尺度效应的基础上，从分子动力学理论出发进一步深入研究其机理，建立相应的模型。在纳米线和纳米梁等典型微纳结构上，利用热传导和杨氏模量小的纳米尺度效应，研究提高生化特异性反应产生的热和位移检测灵敏度的途径。在我们已观察到金属富勒烯 Y@C82 对于芳香族分子具有特殊的纳米敏感效应的基础上，利用特殊的六元环与芳香类分子具有很强的亲和力的特性，研究实现分子量级芳香类化合物（包括 TNT 分子）的检测途径。利用纳米材料反应活性强和比表面大的特点，研究提高传感器灵敏度的方法。以我们已采用的光学方法检测位移、电学方法检测电阻和电容、谐振模式提取特性

参数等表征手段为基础，发展新的纳米尺度下的特性表征原理与方法，研究纳米结构的其他力学、热学、电学等纳米尺度效应，并探讨其用于提高传感器灵敏度的途径。

- 针对生化特异性反应在微纳敏感结构上的信息转换机制与界面效应研究，拟合成筛选出特异性强的生化敏感材料，并在加工与修饰的微纳敏感结构表面上，固定（炭疽杆菌等）病原体特异性单克隆抗体，定点定向固定化学敏感材料；再利用微纳敏感结构热传导小的优点，通过高灵敏检测反应产生的热量，来表征被测对象与敏感膜的相互作用强弱和动力学过程，通过微纳尺度下生化敏感反应的微热力学和微动力学分析，探明生化材料的膜敏感反应机理及界面效应。  
- 采用生长等方式，结合沉积方法及模板一组装的集成技术来构筑所需的纳米结构，实现简单纳米结构的自下而上制造；对于复杂纳米结构，将借助纳米压印、聚焦离子束等手段获得特定纳米结构及花样的生长模板，再利用生长等手段在模板中实现纳米结构的可控生长与组装，最后再通过SPM纳米操控实现对纳米结构和器件的修整和改造。基于MEMS工艺研究自上而下微纳制造技术，实现微纳制造技术融合；在自上而下加工的特定微/纳结构上，通过操纵、生长或组装来制作具有敏感功能的纳米结构，实现自上而下和自下而上微纳制造技术的融合；针对典型的微纳阵列传感器，采用自上而下方法加工出微纳传感器总体结构，再利用自下而上方式实现敏感部件或敏感膜区域的自组装。基于分子或原子的基本作用势，应用MD、CA模型计算，并利用力学的大量重复结构的连续力学等效原理，将原子级作用模型耦合到连续体模型中来模拟微纳结构材料的力学特性；基于MD模型和玻尔兹曼方程建模，采用解析法和Monte Carlo方法，来求解微纳结构材料的热学特性。  
- 从硅的热传导和杨氏模量等纳米尺度效应出发，利用硅纳米线和纳米梁等结构来实现对生化特异性反应产生的热和位移的片上检测；利用纳机械谐振器质量分辨率高的特点，来实现片上对生化特异性反应产生的质量变化的检测；利用碳纳米管高曲率易产生强电场和纳米间隙内气体电离的特点，来实现对沙林等气体的检测；利用金属富勒烯Y@C82和分子印记材料对芳香类化合物（包括TNT分子）的特殊选择效应，来实现对TNT等分子的检测。利用悬臂梁阵列和参考悬臂梁来抑制噪声和降低环境影响；利用高阶谐振模态低能量耗散原理，来提高频率检测分辨率；针对单个传感器很难分辨出混杂超微量成分的问题，拟采用阵列式微纳传感器来提供足够多的正交组合信息，通过后续信息融合处理来实现高分辨检测。

# 3．创新点、特色及其可行性分析

本项目从纳米尺度效应、生化特异性反应与微纳敏感结构的作用机制及界面效应、纳米尺度下的特性表征和微纳制造等微纳传感器基础工作入手，来进行微纳传感器的研究工作，可望在以下三方面产生创新成果：

- 观察到新的纳米尺度效应：在前期的工作中，我们已观察到硅杨氏模量的纳米尺度效应。事实上，在纳米尺度下硅的许多特性有可能与宏观特性不同，等待人们去了解。目前我们已有了制备悬空的硅纳米线和纳米梁的手段，可以进一步去研究硅的其他热学和力学等特性（如断裂强度等）的纳米尺度效应，并有可能观察到新的纳米尺度效应。从自下而上的方法出发，我们可以制备出不同的纳米结构，这为我们观察到其他材料的纳米尺度效应提供了手段。  
- 产生新的纳米结构构筑方法和微纳制造及微纳器件模拟设计手段：利用MEMS工艺具有高度选择性的特点，可以产生硅纳米线、纳米梁和纳米结构阵列制造的新方法；将自上而下和自下而上微纳制造技术融合，可望发展纳米结构及器件的新构筑原理与方法，实现按需和可控构筑各种纳米结构；将原子级的作用模型看作一个单元，类比结构力学中桁架结构的受力分析的连续体等效，将原子级作用模型通过大量重复性单元耦合到连续体模型中，可望产生微纳器件模拟设计的新方法。  
- 发明新的超微量快速检测微纳敏感结构与传感器：利用硅纳米线和纳米梁热传导小的特点，可以发明热检测灵敏度高的微纳敏感结构；利用硅纳米线和纳米梁杨氏模量小的特点，可以发明位移检测灵敏度高的微纳敏感结构；利用纳机械谐振器质量分辨率高的特点，可以发明质量检测灵敏度高的微纳敏感结构；将生化特异性反应与微纳敏感结构融合，可以发明灵敏度高选择性好的生化微纳传感器；利用悬臂梁阵列和参考悬臂梁来抑制噪声和环境影响，利用高阶谐振模态低能量耗散原理，利用悬臂梁表面的纳米网格化形成微区边界限制等手段，可以发明信噪比和分辨率高的微纳传感器。

# 可行性分析：

本项目根据设定的研究目标和内容，凝聚了一支来自国内主要优势单位的多学科紧密配合的创新研究群体，共有9个不同学科的国家重点实验室或部门重点实验室参与研究工作，研究条件互补性强，已经基本具备了开展研究的软硬件条件。而且，各参加单位已经在本项目相关的前期研究工作中取得了一些创新性成果。

纳米尺度效应方面：上海微系统所在去年结题的MEMS973项目实施过程中，首次观察到硅杨氏模量的显著纳米尺度效应，发表在APL上的文章被SCI论文多次引用，并被引用者用来解释其实验结果。东南大学在硅纳米线的热传导纳米尺度效应模型方面做出了漂亮的工作，提出了热传导纳米尺度效应模型。中科院化学所研究发现了金属富勒烯Y@C82对于芳香族分子具有特殊的纳米敏感效应，利用金属富勒烯Y@C82的六元环与芳香类分子具有很强的亲和力特点，将可能实现分子量级芳香类化合物（包括TNT分子）的检测。中科院固体物理所利用在陶瓷基底上构筑的多层  $\mathrm{SnO}_2$  纳米线阵列，发现当纳米线特征尺寸从1微米降到200纳米时，其对气体的检测灵敏度提高了6倍，而反应时间则缩短了5倍。这些工作为我们进一步研究其他纳米尺度效应和发展纳米尺度下的特性表征方法积累了经验，为我们利用这些纳米尺度效应来研究超微量快速检测微纳传感器提供了基础。

微纳模拟设计与制造方面：上海微系统所在去年结题的MEMS973项目实施过程中，将MEMS制造方法进一步深化，发展了一套基于MEMS的纳米制造技术，已经可以制造出纳米探针（零维纳米结构）、纳米线（一维纳米结构）和纳米梁（二维纳米结构）等多种纳米结构，制造出了薄至  $12\mathrm{nm}$  的悬臂梁阵列，研究成果在IEEE Sensors2004国际会议上做了特邀报告，反响热烈；而敏感一驱动侧壁集成的悬臂梁微制造技术成果发表在国际电子器件大会（IEDM2004）上，他们已在这方面申请了多项发明专利。中国科学院化学所利用模版法结合电化学沉积技术，发展了制备包括富勒烯、金属及非金属复合纳米结构阵列的制造技术，研究成果近两年在国际著名杂志Adv.Mater.和Adv.Func.Mater.上发表后，被选为该系列杂志2003年年度封面，并被美国化学会网站作为Heart Cut文章专门介绍；他们还通过在STM针尖和单晶硅表面间施加电脉冲的方法，将单晶硅表面氧化成设计的图案，其分辨率优于  $100\mathrm{nm}$  。中科院固体所利用基于模板的催化诱导CVD和电沉积方法，发展了一系列金属、半导体、化合物纳米线阵列的制造技术，其中关于制备ZnO和GaN纳米线阵列的工作1999年和2000年在

APL.上发表后，单篇引用数分别达到148次和109次。东南大学已有CA、MC计算的前期工作基础，在所研究的材料或结构的电学、力学和热学性能方面有较多的数据积累；MD、MC、CA方法在计算材料或结构的电学、力学和热学性能已经取得初步成功。这些成果为我们进一步发展新的微纳制造技术和实现自上而下和自下而上的微纳制造技术融合提供了基础，使我们进行微纳传感器模拟设计方法的研究有了切入点。

微纳传感器研究方面：上海微系统所通过国际合作，利用纳机械谐振器研制出的氢气传感器实现了对氢气的痕量检测，在超高真空下对氢气的分辨达到 $5\times 10^{-18}$  克，2003年研究成果发表在Rev.Sci.Instrum.上，是当时国际上质量分辨率最高的传感器。上海交通大学利用碳纳米管尖端的场发射特性制作出了气体压力传感器，利用碳纳米管表面气体吸附引起电导变化特性制作出了氨气传感器，并对基于单壁碳纳米管的几种微纳电子器件进行了探索。中科院合肥智能所采用半导体气敏阵列和人工神经网络研制成功了四单元智能气体传感器，用于氢气、一氧化碳、乙炔、乙烯监测和组份分析；利用TNT和黑索金的相变吸热现象，研制成功了微梁型爆炸物传感器。这些工作为我们进一步研究超微量快速检测微纳传感器提供了基础。

总之，本项目给出的技术路线和研究思路充分考虑了原理与技术的可行性，既考虑了创新性和预期成果的水平，又充分考虑了已有工作的基础以保障研究目标的实现。因此，研究技术路线先进可行，已经具备了取得重大突破的条件。

检测微纳传感器。

# 课题1.生化特异性反应在微纳敏感结构上的信息转换机制

主要研究目标：

针对微纳敏感结构与检测对象的特点，探明相应的生化特异性反应与微纳敏感结构的作用机制，提出在微纳敏感结构上形成敏感膜的方法，建立生化敏感膜超微量响应性能的评价方法，合成筛选出5种以上特异性强的敏感材料，满足阵列微纳传感器超微量快速检测重要致病病原体（如炭疽杆菌等）和有毒有害气体（如沙林等）的要求，达到空气中  $10^{-4}\mathrm{mg / L}$  量级沙林的检测水平。

# 主要研究内容：

（1）化学敏感分子识别机理的研究

依据沙林等有毒有害化学物质的物化特征，以及LSER理论、氢键、酸一碱、络合配位与化学吸附等作用原理，建立化学敏感膜超微量响应性能的评价方法，研究敏感反应的响应速度和特异性，采用量子化学理论优化敏感作用效果，完善微纳尺度下化学敏感膜的分子识别理论。

(2) 生物敏感分子识别机理的研究

通过基因操纵和生化连接技术等构建敏感生物分子元件，研究其在微纳敏感结构上均相自组装行为，探索活性生物分子组装后与底物分子相互作用的机制、相关动力学及生物学特性等。

（3）生化敏感反应速度和灵敏度的研究

研究微纳敏感结构界面效应对生化检测对象的扩散、富集、混合、表面反应等方面的影响，探索有效提高微反应速率的方法；在定向固定于氨基（羧基）修饰或金沉积的微纳结构表面，研究亲和素和抗体上修饰的金胶粒在微纳结构表面上的放大结合质量，寻求提高灵敏度的方法。

（4）生化敏感材料的合成与固化

针对目标传感器，合成筛选出用于炭疽杆菌和沙林等超微量检测的敏感材料，并研究其与微纳敏感结构的兼容性。对微纳结构进行微纳加工和表面修饰，将敏感材料在微纳敏感结构表面进行均质定向固定，研究优化各种条件和提高过程的稳定性等，建立在微纳敏感结构上特异识别的敏感分子元件组装、固定与灵敏信号捕获的模式技术和方法。

# 课题2.自下而上的纳米结构构筑原理与方法

主要研究目标：

本课题的研究目标为根据微纳传感器对纳米结构的设计要求，实现纳米结构按需构筑；拥有自主知识产权的纳米结构及其器件的构筑技术，发展纳米敏感结构与器件“自下而上”构筑的新原理和新方法，为深入的功效研究和原理性器件系统的集成提供所需的基础。为研制超微量检测微纳传感器提供设计思路和制造手段。

- 在与TNT与沙林检测相关的功能纳米材料研究方面取得重要进展；得到特异性强、灵敏度高的富勒烯及其他纳米材料。  
- 对该领域内的一些重要科学问题，如纳米结构的可控生长、纳米材料特异性和纳米效应的机理、“自上而下”和“自下而上”制造技术的融合等问题有更深入的认识，发现一些新的物理现象，得到一些规律性结论。  
- 发展具有自主知识产权的纳米结构可控构筑的技术，对于所构筑的纳米结构单元尺寸一致性优于  $90\%$  。发展纳米敏感结构与器件“自下而上”构筑的新原理和新方法，提出相应的表征方法，为深入的功效研究和原理性器件系统的集成提供所需的基础。

# 主要研究内容：

（1）发展纳米结构生长的前沿技术

着重发展纳米结构及其器件的“自下而上”构筑新原理和新方法，实现按需构筑纳米结构体系，为其他课题组提供纳米传感器件的研究载体和设计思想。

（2）特定微结构上按需构筑纳米结构的方法

根据目标传感器的需要，研究在特定的微结构上按需构筑纳米结构的方法，解决与“自上而下”工艺的兼容性、定位定向生长问题，探索在不同基底上纳米结构的构筑机理，实现纳米结构可控生长(形态、相结构)。

（3）纳米操控技术

根据目标传感器的需要，对于难以直接生长构筑的纳米结构，研究利用SPM纳米操控技术，实现对特定纳米结构的切割、修整、合并或布线等功能。

（4）特定纳米结构的纳米效应

研究特定纳米结构的纳米效应，揭示纳米结构新的物理效应及纳米结构单元的物理/化学行为等，探索将其用于传感器的技术途径与实现方法；

（5）可重复构筑问题

研究纳米结构的一致性、可重复性、结构稳定性等问题，探索提高纳米结构单元尺寸一致性的构筑方法。

# 课题3.自上而下的微纳制造原理与方法

主要研究目标：

针对几类典型并具有相对普适性的微纳敏感结构，解决自上而下及其它组合方式下微纳敏感结构的实现问题，发展具有自主知识产权的基于自上而下方式的微纳制造及自上而下与自下而上融合的微纳制造新原理新方法，其宽度或厚度特征尺寸为  $10\sim 100$  纳米，为研制微纳传感器提供制造基础。

# 主要研究内容：

（1）基于MEMS的纳米制造技术

系统研究基于MEMS技术的纳米梁、桥、腔体、探针等典型结构的可控、可集成、可批量制造技术；研究结构的稳定成型、无损释放方法；研究残余应力的测量、消除方法；研究相应的纳米结构表征方法等。

(2) 自上而下方式加工纳米结构的新方法

基于刻蚀与腐蚀相结合的纳米尺度结构并行加工方法，新型侧墙技术等；基于聚焦离子束（FIB）等先进手段的纳米尺度局域化处理方法等，包括局域化刻蚀、淀积/外延（如氧化）、掺杂、光刻技术等；发展自上而下方式加工纳米结构的新方法。

(3) 与自下而上和纳米材料工艺兼容的微纳制造技术

研究在自上而下方式加工出的微/纳结构（如 Si 材料微/纳结构）上，利用种子层或过渡层采用自下而上方式操纵、生长或组装作为关键的传感器性能表征的纳米结构，再进而制作成微纳传感器的工艺集成方法；并研究其驱动、检测结构的制作、集成工艺；敏感材料与敏感结构的工艺集成方法等。

（4）阵列化微纳结构制造技术

研究阵列化微纳结构的制造技术；研究阵列化微纳结构的一致性控制技术；研究多功能传感器阵列的加工方法等。

（5）微纳传感器封装技术

研究微纳传感器微弱信号的热隔离、电磁屏蔽等封装技术；研究适用于生化检测的封装技术；研究传感器芯片与测试电路的低噪声集成技术等。

# 课题4. 跨微纳尺度的传感器理论模型与仿真

主要研究目标：

提出微纳结构或材料的力学、热学及其耦合模型、算法及其跨微纳尺度的设计方法。在理论上能够较精确的分析微纳结构力学、热学及其耦合特性。初步考察结构的尺度、界面、表面粗糙度对性能的影响；建立描述界面、表面粗糙度、结构尺寸的电学、力学、热学及其耦合数学模型。初步建立纳米结构与生物大分子的相互作用模型，为设计和制造相应的微纳器件与系统奠定理论基础。模型或设计方法与实验结果对比或与MD、MC和CA的原始算法比较，偏差小于  $10\%$  计算效率提高10倍以上。

# 主要研究内容：

（1）微纳尺度力学特性模拟与仿真

研究将原子级作用模型耦合到连续体模型中的方法，提高设计与计算效率；研究环境温度与电场渗透对力学性能的影响；研究微纳结构的尺度、界面、表面粗糙度对力学和热学性能的影响模型；研究微纳结构动态性能的能量耗散机制与数学模型；研究微纳尺度下杨氏模量、泊松比的计算方法。

（2）微纳尺度热学特性模拟与仿真

在微纳尺度下，研究声子的能量输运特性，仿真系统的尺度对仿真结果的影响；获得工作载荷下的结构应变、晶格失配对声子输运的影响；微纳尺度下热传导系数、赛贝克系数的计算方法。

（3）微纳尺度耦合模型与仿真

建立微纳结构材料的力电耦合、力热耦合、热电耦合、力热电耦合以及纳米结构与生物大分子相互作用的模型，并研究跨微纳尺度的设计方法，为目标传感器设计提供模拟手段。

（4）微纳制造模拟与仿真

针对自下而上构筑纳米结构的工艺要求，研究纳米结构生长与自组装工艺模型，为工艺研究提供模拟手段。

（5）微纳尺度模拟设计方法验证

针对仿生传感器特定纳米结构，为研制仿生传感器建立数学模型，验证仿生传感器的敏感效应与工艺可行性，为仿生传感器提供优化设计手段。

# 课题5. 微纳机电敏感结构与传感器

主要研究目标：

- 探明微纳机电敏感特性和敏感结构的纳米尺度效应与机制，提出相应的表征方法，建立将生化反应转换的机械量转变为电学量的表征方法和模型。  
- 揭示微纳机电传感器能量损耗和噪声特性进入微纳尺度的作用机制和随尺度变化规律，形成微纳机电传感器超微量快速检测的优化方法和结构。  
- 利用敏感元件产生的纳米尺度效应和量子尺度效应显著提高传感器灵敏度。  
- 研制出 4-8 单元微纳机电阵列传感器原理样机, 在检测环境中质量变化分辨达到  $10^{-12}$  克量级。  
- 用传感器原型样机对炭疽菌等重要致病微生物进行检测，实现数百个以内病原的检测。

# 主要研究内容：

(1) 生化反应过程的物理量转换与表征

研究超微量生化反应能量向机械能量转化实现物理表征过程中的效应和机理；研究将生化反应转换为物理表征（包括质量变化、热量变化、表面作用力等）的方法；研究反应区域内静电作用和纳米网格化形成机制，提高特异性结合分子的附着强度、实现一定的重复反应能力和对表面能表征实现优化。

（2）微纳机电敏感结构相关基础研究

研究微纳机电传感效应转换为可读出电信息（如将反应质量转变为悬臂梁谐振频率变化等）的原理方法；研究机电传感结构的微纳多尺度耦合效应、表面效应、有限粒子系统的效应和纳米尺度效应；研究微纳机电传感相关的噪声体系和机电耦合效应，研究微纳机电的灵敏度放大原理，建立微纳机电传感模型。

（3）微纳机电传感结构优化设计和实现

研究微纳机电传感器驱动和信号拾取方法及其耦合关系、综合利用静态和动态参数的方法；研究频率检测中模态优化、提高品质因子的原理与方法、质量改变与弹性性能变化间的交叉影响机制、振动的非线性规律；研究表面应力检测中的界面效应和液体粘滞、流动和温度的干扰机制；研制出传感结构。

（4）阵列式微纳机电传感器与智能算法的融合

研究阵列式微纳机电传感结构基础问题，尤其是阵列中各单元之间的相互作用和组合原理，实现阵列多敏感信息融合和环境噪声抑制，为智能算法提供足够的传感器信息，提高检测环境下微纳机电的超微量快速检测水平。

（5）量子电子机械系统的原理方法探索

研究纳米尺度下能量损耗（品质因子）和表面效应随尺度进一步缩小的Scaling-down规律，研究进入量子效应尺度的噪声机制随尺度、频率、温度、压力等参数的变化规律，从而建立起一套进入量子效应尺度的纳机电传感器的体系分析方法。探索量子电子机械系统的敏感效应和高性能敏感表征方法，力争实现对现有检测分辨率限制的突破，为进一步研究量子机械敏感器件等提供扎实的科学基础。

(6) 在基础研究成果的基础上, 进行微纳机电传感器原理样机的研制。研究传感器对炭疽杆菌等生化物质在环境下进行超微量检测的相关技术。最终实现传感器原型样机和超微量检测

# 课题6.基于碳纳米管的敏感结构与传感器

主要研究目标：

探明近气体分子自由程间隙内的气体电离传导规律及与气体种类的关系, 利用碳纳米管高曲率表面的电离特性, 实现碳纳米管气体传感器敏感方法与结构。解决表征和分辨技术中的关键问题, 建立该类气体传感和检测的器件模型和理论基础。实现检测沙林等气体的智能化碳纳米管气体传感器原理样机, 对沙林的检测分辨率达到  $10^{-4} \mathrm{mg} / \mathrm{L}$  量级, 检测速度达到秒级。完成碳纳米管红外探测机理、设计、制造、测试技术的完整研究, 为相关传感器的进一步研究奠定基础。

具体指标：

（1）自主创新性成果：纳米尺度电极间隙内气体电离与电荷传输效应。  
(2) 自主知识产权：碳纳米管纳米尺度电极间隙结构生化气体传感器。  
（3）提供碳纳米管与微米波长红外光相互作用机理研究报告。  
（4）碳纳米管电离气体传感器原型器件工艺和性能评价。  
(5) 制作出阵列碳纳米管纳米尺度电极间隙结构生化气体传感器原理样机, 对沙林的检测分辨率达到  $10^{-4} \mathrm{mg} / \mathrm{L}$  量级, 检测速度达到秒级。  
（6）完成两个波段红外探测器的原理样机。

# 主要研究内容：

（1）纳米间隙内的气体电离传导规律

探索纳米间隙内在近气体分子自由程情况下的电子与气体分子结合、激发作用和气体分子、离子的运动规律，以及库仑电流的起伏与环境气体参数之间的关系；研究较低电压下高曲率碳纳米管表面形成的强电场在纳米间隙内的分布及对气体电离导电性能的影响，建立纳米间隙内的气体电离理论模型和碳纳米管气体传感器的敏感模型。

(2) 阵列纳米间隙气体传感器对混合气体的交叉敏感特性研究

研究基于多传感器或者传感器阵列的气体识别和定量分析，使用支持向量机(SVM)技术来对多气体传感器进行数据融合，研究高精度识别和检测混合气体的方法，实现智能化、多功能气体检测的方法。

（3）碳纳米管气体传感器结构形成的相关基础研究

利用碳纳米管气体传感器的敏感模型，研究传感器的结构与优化设计方法，建立传感器数值模拟与设计理论，研究阵列碳纳米管气体传感器制备和封装相关问题，研制出碳纳米管气体传感结构。

# （4）碳纳米管红外吸收机理研究：

研究碳纳米管的红外吸收机理，为碳纳米管红外传感器奠定理论基础。包括碳纳米管结构、修饰对其能带的影响，不同结构、不同修饰下的碳纳米管与不同波长红外光的作用机理，碳纳米管缺陷对红外吸收的影响，红外吸收中热对碳纳米管电子传输特性的影响等。

# （5）碳纳米管的定制生长技术：

研究碳纳米管的结构定制技术，为碳纳米管红外传感器提供技术手段。包括碳纳米管的一致性生长技术，碳纳米管的定位生长技术，碳纳米管的红外修饰技术以及不同结构碳纳米管的筛选技术等。

# （6）碳纳米管红外传感器研究：

研究碳纳米管红外传感器的结构、建模、仿真设计研究，以及制作和测试技术等。包括红外传感器的结构设计和优化，不同波长红外传感器的设计模型，红外传感器的集成工艺方法以及相应的封装技术、处理电路研究等。

# 课题7.仿生微纳敏感器件与智能系统

主要研究目标：

建立高灵敏性和高选择性敏感材料、纳米结构、传感器件和信号获取的仿生学方法与实现途径，为发展仿嗅觉、视觉、触觉传感器提供材料、结构、器件设计和智能识别的知识和原理基础。

合成和筛选出对TNT等炸药分子具有高度敏感性和特异性的纳米结构，研制出TNT仿生嗅觉传感器，灵敏度达到纳克量级。为发展长寿命超微量快速检测传感器奠定基础。

阐明仿沙蚁导航传感器视觉感知与导航机理，构建利用偏振光模式导航定位的理论模型。为微纳米结构的仿生导航传感器的设计与制备提供可行的技术路线。

发展出仿壁虎触觉粘附的微纳结构材料和敏感阵列，及其器件构筑方法；提出先进合理的粘附与脱附控制方法以及技术实现方案；探索出采用非生物材料与结构再现生物敏感效应的科学途径。

研制出TNT仿生嗅觉传感器、仿沙蚁导航传感器和仿壁虎触觉粘附阵列实验样机，并建立相应的测试标定方法与系统。

# 主要研究内容：

# （1）微纳尺度下的生物敏感机理与仿生方法研究

针对目标传感器要求，以飞蛾、沙蚁、壁虎的生物学研究结果为基础，进一步在微纳米尺度上了解生物嗅觉、触觉、视觉器官的敏感机理与结构特征，探索利用纳米材料和微纳结构进行机理仿生的方法。

# （2）仿生微纳敏感器件的数学模拟与仿真

针对仿生敏感器件微纳结构特点，利用课题4建立的计算机模拟与仿真方法，分析和优化仿生纳米结构和分子印记材料的敏感效应与性能，并构建利用偏振光模式导航定位的理论模型和基于分子吸引力的纳米阵列粘附和脱离控制模型。

# （3）仿生微纳敏感器件的设计与制备

针对仿生微纳敏感器件功能需求，结合现有的微纳结构制造能力，设计出仿生微纳传感器与制备工艺，并利用课题4建立的计算机工艺模拟方法验证和优化工艺条件，利用课题2、3建立的微纳制造方法与平台进行工艺实施。

# （4）仿生敏感器件与系统的智能化技术

根据仿生传感器功能要求，研究多传感器信息融合方法，建立具有协同和竞争双重特性的视觉选择性注意机制的认知模型，发展仿生智能信息处理方法，研制出TNT仿生嗅觉传感器、仿沙蚁导航传感器和仿壁虎触觉粘附阵列实验样机，并建立相应的测试标定方法与系统。
