报告正文  

# （一）立项依据与研究内容（4000-8000字）：  

# 1. 项目的立项依据（研究意义、国内外研究现状及发展动态分析，需结合科学研究发展趋势来论述科学意义；或结合国民经济和社会发展中迫切需要解决的关键科技问题来论述其应用前景。附主要参考文献目录）  

战略性新兴产业是新兴技术和新兴产业的深度融合，既代表着科学技术创新的方向，也代表着产业发展的方向，而制约新兴产业发展的主要因素是战略性新兴技术的创新和应用。战略性新兴技术已经成为推动产业结构升级和引领社会进步的根本动力。战略性新兴产业的发展加速了能源需求扩张与环境保护之间的矛盾。产业化进程对能源的快速需求应该转化为对战略性新能源技术创新和发展的推动力。对战略性新性技术的辨识、确认和遴选已经成为制定战略性新能源产业发展宏观政策与管理的重要决策依据。本课题以此为背景，提出战略性新兴技术的概念和特征，建立系统的辨识战略性新兴技术的可视化方法，分析战略性新兴技术体系演化的路径和机制。本项目的研究对我国当前大力发展的战略性新兴产业的培育和发展具有重要的理论意义和现实意义。  

# 1.1 研究意义  

# 1.1.1 研究的理论意义  

(1) 提出战略性新兴技术的概念，丰富和发展新兴技术管理理论  

本项目在大力发展战略性新兴产业的背景下，基于新兴技术管理理论，提出战略性新兴技术的概念。综合应用包括知识计量、科学技术管理、科技政策等多学科交叉理论和方法，从专利计量和信息可视化的视角，对战略性新兴技术进行辨识和确认，建立系统的战略性新兴技术辨识方法，开辟新兴技术管理理论研究的新领域，丰富和发展新兴技术管理和技术创新理论。同时，进一步扩展信息可视化技术的理论和应用。  

# (2) 建立系统的战略性新兴技术辨识方法，开辟技术管理理论新视角  

新兴技术管理是当前国内外技术管理学研究的前沿和热点问题，相关学者在新兴技术管理的方法、组织和战略等宏观层面开展了大量的研究工作，但对于战略性新兴技术微观层面的技术辨识，技术体系的演化机制等方面研究较少。本项目在建立战略性新兴技术辨识方法的基础上，分析战略性新兴技术体系的演化机制，结合技术演化的内部和外部环境因素的共同作用，探索战略性新兴技术体系与战略性新兴产业发展的共生演化机制。将进一步拓展技术共生演化理论和方法的研究，开辟技术管理理论研究的新视角。  

# (3) 丰富和拓展知识计量学理论和研究范式  

目前的知识计量理论和方法的研究，仍然主要是对科学论文进行计量研究，专利计量研究的成果比较少。将科学计量学领域的共现分析、引文分析等理论与方法和信息可视化技术，通过移植、改造和创新，用于专利计量研究，并基于此，建立战略性新兴技术的辨识方法，探索战略性新兴技术体系的演化路径和机制，是一个比较前沿的尝试和创新。研究成果将丰富和完善知识计量学理论和研究范式。  

# 1.1.2 研究的实践意义  

# （1）推动战略性新兴产业发展的创新基础平台的建设  

战略性新兴技术是战略性新兴产业发展的核心和关键环节，决定新兴产业发展的主导方向和进程，直接作用于战略性新兴产业的创新基础平台建设。科学知识的创新和战略性新兴技术体系是构建战略性新兴产业创新系统的两个重要基础。本研究在界定战略性新兴技术的范畴和特征的基础上，提出系统的战略性新兴技术创生、辨识和共生演化机制的分析方法和应用，对推动当前我国大力发展战略性新兴产业，构建科学技术创新基础平台具有重要的战略意义和现实意义。  

# （2）服务我国新能源产业的战略性新兴技术的选择与管理  

当前世界，各国正在进行抢占科技制高点的竞赛，全球将进入空前的创新密集和产业振兴时代。新兴技术不同于传统技术和一般的新技术，它具有创造一个新产业或改变一个现存产业并对经济结构产生重要影响的能力。新能源产业领域具有投资大、周期长、关联复杂等特征，也是我国科学技术发展战略和经济结构调整的重要组成部分，对新能源产业中的战略性新兴技术辨识和管理对新能源产业的发展具有重要的现实意义，对我国新能源产业的结构优化调整和产业升级、确定核心竞争力等方面都具有重要的影响和指导作用。  

# (3) 为政府部门开发和管理战略性新兴技术提供决策支持  

辨识和选择战略性新兴技术是政府部门发展战略性新兴产业首先要解决的重点问题之一。由于战略性新兴技术的核心和关键地位，其发展和演化的路径和机制不仅能够带动相关技术的发展和创新，推动共性技术的迅速提升，而且对战略性新兴产业的发展和结构调整具有重要战略意义。深入从微观层面对战略性新兴技术的创生和技术体系的演化路径进行可视化分析，可以指导宏观层面的新能源产业的发展和政策制定，为政府部门开发和管理战略性新兴技术提供重要的决策支持和参考。  

# 1.2 国内外研究现状及发展动态分析  

新兴技术理论相关的研究始于20世纪80年代的西方国家。1994年，美国宾夕法尼亚大学沃顿商学院首创开始了对新兴技术管理的研究。我国从2000年以后，在建设创新型国家和大力发展新兴产业的背景下，相关领域的学者迅速开展了关于新兴技术及其管理问题的研究。  

# 1.2.1 新兴技术管理的相关研究  

国际上关于新兴技术管理的开创性研究成果集中在美国宾夕法尼亚大学沃顿商学院的新兴技术管理研究小组于2000年出版的《Wharton on Managing Emerging Technologies》一书中。它将新兴技术（Emerging Technology，ET）定义为：“基于科学的，有可能创立一个新行业或者改造一个现有行业的创新”[1]。同时给出了新兴技术的三个特点：①知识基础在扩展；②其在现有市场中的应用在经历着革新；③新市场正在形成和发展。  

国内相关的研究机构主要集中在电子科技大学、浙江大学、清华大学和北京工业大学等。  

从检索到的文献来看，国内外学者对新兴技术创生与演化的研究，主要是集中在结合新兴技术的特征和发展的不同阶段，分析技术演化形式和影响因素。国内以电子科技大学王敏、银路和宋艳等为代表的学者则主要从类生物物种进化的“技术物种”共生演化的视角，对新兴技术的形成和发展路径等问题进行了研究。从微观层面基于可视化技术和技术体系演化视角的研究较少，甚至尚没有出现，本项目就是基于专利计量对新兴技术辨识和演化分析的权威性和可行性，利用信息可视化技术辨识战略性新兴技术，探测其技术体系的演化路径，对新能源产业核心领域的战略性新兴技术体系演化进行多元、动态的可视化分析。  

# 1.2.3 专利计量与信息可视化方法的相关研究  

# （1）专利计量的相关研究  

国际上，专利计量(patentometrics或者patent bibliometrics)研究已经有50多年的发展历史。美国、日本等许多发达国家将专利计量分析用于比较、评估不同国家或不同企业之间的技术创新活动与技术发展状况，进而追踪和预测技术发展趋势与技术发展前沿，并以此作为产业结构调整和提高技术竞争力战略制定的依据。纳林(Narin F.)是国际上有关专利计量研究最早和最权威的学者之一。1994年，纳林发表了专利计量研究的经典之作《专利文献计量学》，文章认为科学论文文献计量学与专利文献计量学有着惊人的相似性，可以适用于相同的范畴[43]。他认为，有三种不同的文献计量学，即科学论文文献计量学(literature bibliometrics)、专利文献计量学(patent bibliometrics)和科学技术关系文献计量学(linkage bibliometrics)，它们都可以作为政府与企业的绩效评价手段和方法。学者们利用专利指标在技术生命周期[44]、技术竞争[45]、R&D投入政策与专利产出的关系[46]、技术合作战略的作用与低-中-高技术产业专利组合的关系[47]、R&D管理和技术评估、技术趋势分析、以及新兴技术预测等方面进行了广泛的研究。但这些研究成果主要是运用传统的统计分析方法对专利进行研究。  

# （2）基于信息可视化技术的专利计量相关研究  

信息可视化(information visualization)是一门以计算机科学、地图学、认知科学、信息传播学与信息系统为基础，为直观、形象地表现、解释、传递空间信息并揭示其规律，关于信息表达、传播的理论、方法与技术的学科[69]。近年来，信息可视化技术得到了迅速的发展和传播，被广泛应用于数据挖掘、知识管理、科学计量和专利计量等领域。信息可视化已经成为数据可视化(data visualization)技术中新的研究热点。其中，陈超美博士开发的基于JAVA平台的CiteSpace应用软件系统[70]，开创了适于多元、分时、动态的复杂网络分析的新一代信息可视化技术，目前已被近60个国家和地区万余次下载使用，正成为科学计量学普遍应用的新手段。但该可视化软件在专利计量研究领域的应用，还有待于进一步开发和推广。本研究将基于CiteSpace应用软件的信息可视化技术应用于专利计量，对战略性新兴技术进行辨识分析，探索战略性新兴技术体系的演化路径。  

在科学计量学和科学知识图谱研究领域，信息可视化技术已被国内外许多学者广泛用来探测科学发展前沿[50-52]。申请人在科学和技术前沿探测和知识领域演进的可视化分析方面也做了一些创新性工作[53-56]。此外，国内外学者已经开始运用国际新兴的信息可视化技术进行专利计量研究，以大连理工大学WISE实验室为代表最早开展了基于信息可视化技术的专利计量研究，武汉大学信息管理学院、浙江大学信息资源管理研究所等也从具体的前沿技术领域开展了专利计量研究[57-66]。  

# （3）研究述评  

在美、日等发达国家，专利计量已经被比较普遍地用于国家的技术前沿、产业结构的探测和分析。随着信息技术的发展，信息可视化技术手段被广泛用于科学前沿的探测研究。在数量不多的基于信息可视化的专利计量研究中，我们尚未发现将专利计量和信息可视化技术结合用于分析战略新兴技术的辨识及其体系演化机制的研究。当前国内外的专利分析研究，还主要集中在对专利数量和趋势方面的分析，鲜有针对专利文献，深入到具体技术知识内容的研究。本项目研究将基于专利计量的信息可视化技术应用到对战略性新兴技术的辨识方法的研究中，探索战略性新兴技术从技术单元-技术链-技术群-技术网络-复杂技术系统到战略性新兴产业发展的演化机制。  

# 1.2.4 新能源产业培育与发展战略的相关研究  

新兴产业是指随着新的科研成果和新兴技术的发明，应用而出现的新的部门和行业。芮明杰将产业形成的方式分为四种：一是新技术的产业化；二是原有产业的分化；三是相关联的产业中一个产业的发展带动另一个新产业产生；四是产业融合[67]。随着产业化进程的加速，能源供应和环境问题日趋严峻，新兴能源产业成为各国政府与研究人员竞相追逐的热点。陈莞、谢富纪从新兴技术管理、城市化与新能源技术发展的关系两个角度对已有研究进行回顾，围绕区域协同发展、城市空间演化以及新能源技术共享三个相关主题提出了可继续深入探讨的命题[68]。王发明、全震等分析了基于新兴技术的我国能源产业现状及面临的挑战。提出我国能源产业政策建议[69]。申请人也曾利用信息可视化方法对新能源技术的科学文献做过计量分析，对新能源技术的研究前沿和研究热点[70]，新能源技术的国家合作网络[71]，以及研究机构合作网络[72]等进行了研究。  

在检索的文献中，基于专利计量方法和信息可视化技术，对新能源产业核心领域的专利信息进行计量分析，从微观层面探测新兴技术创生，技术体系演化的视角对新能源产业的发展与管理进行的相关研究仍未出现。本项目立足于专利计量理论和信息可视化技术，对新能源产业的战略性新兴技术进行可视化辨识，分析其技术体系的演化路径，进而有针对性的提出我国新能源产业发展与管理对策。

# 2. 项目的研究内容、研究目标,以及拟解决的关键科学问题。（此部分为重点阐述内容）  

# 2.1 项目的研究内容  

本项目主要的研究内容包括：战略性新兴技术的界定与特征分析；建立系统的辨识战略性新兴技术的可视化方法；战略性新兴技术从创生、成熟到创造性毁灭的演化机制；以新能源产业为例，梳理新能源产业核心领域技术体系优化演进路径。本项目的研究内容框架如图所示。  

# 2.1.1 战略性新兴技术的概念与特征界定  

结合文献资料与专家访谈，分析战略性新兴技术与传统高新技术、新技术、新兴技术等概念的区别与联系，给出战略性新兴技术的界定，分析其特征，为项目后续研究建立分析的基础概念。  

1. 新兴技术、新技术、高新技术与传统技术的区别与界定  
2.战略性新兴技术的提出与释义  
3.战略性新兴技术的特征分析  

# 2.1.2 战略性新兴技术创生的可视化分析  

以同一研究领域的数据为基础，绘制基于专利引文网络的可视化图谱、基于科学文献引文网络的科学知识图谱和文献-专利混合共被引网络可视化图谱。基于知识流动理论框架，探测战略性新兴技术创生的科学知识基础到新兴技术前沿的知识流动过程。  

1. 基于专利数据共被引网络的技术科学可视化图谱  
2. 科学文献共被引网络的知识基础可视化图谱  
3. 专利-文献混合共被引网络可视化图谱  
4. 基于知识流动的战略性新兴技术创生的可视化分析  

# 2.1.3 战略性新兴技术辨识的系统方法  

以战略性新兴技术创生的可视化图谱为基础，构建分时段、动态演化的专利数据共被引网络，并对专利共被引网络可视化图谱进行深入解读，确立筛选和辨识战略性新兴技术的初级框架。结合专家综合评价指标，对战略性新兴技术进行确认和检验。  

1. 基于可视化网络图谱的战略性新兴技术的筛选  
2. 基于综合评价指标的战略性新兴技术辨识  
3.战略性新兴技术的确认与检验  

# 2.1.4 战略性新兴技术体系的演化机制  

从微观、中观和宏观三个层面，梳理战略性新兴技术在新能源产业发展中的演化脉络，全面探测新能源产业中战略性新兴技术体系的演化机制和优化路径。  

1. 技术体系的内部演化（微观层面）  
2. 技术体系与外部环境的共生演化（宏观层面）  
3. 技术体系与战略性新兴产业的发展（中介层面）  

# 2.1.5新能源产业中战略性新兴技术辨识与技术体系的演化机制  

产业的孕育和形成主要由两个方面的决定因素，即科学技术的进步和市场消费的需求。战略性新兴技术是新兴产业形成和发展的内部核心动力因素，而市场需求则是新兴产业形成的基本动力。战略性新兴产业的培育和发展离不开战略性新兴技术的发展与创新。战略性新兴产业发展的同时，也不断推动战略性新兴技术的向前发展和创新。  

# 2.2 项目的研究目标  

（1）提出战略性新兴技术的概念及特征；  
（2）建立战略性新兴技术辨识的可视化方法；  
（3）探索战略性新兴技术体系在新能源产业发展中的演化机制；  

# 2.3 拟解决的关键科学问题  

(1) 界定战略性新兴技术的概念，建立系统的战略性新兴技术辨识方法；  
（2）探测基于专利计量的战略性新兴技术体系到新能源产业发展优化演进路径；  
（3）提出新能源产业中战略性新兴技术辨识、演进与管理对策。  

# 3. 拟采取的研究方案及可行性分析。（包括有关方法、技术路线、实验手段、关键技术等说明）  

# 3.1 研究方案  

# 3.1.1 研究思路  

本项目研究主要拟基于专利计量学和技术体系演化等相关理论基础，应用当前国际研究前沿的信息可视化技术，通过CiteSpace等可视化软件系统探测战略性新兴技术的创生和演化机制。结合专  

利引文网络中关键节点和突现节点及其相关计量指标的统计分析，依据结构洞原理，辨识和确认战略性新兴技术；基于知识流动理论，依循技术单元(基因)-技术链-技术群-技术网络-复杂技术系统，构建从战略性新兴技术体系到战略性新兴产业发展的优化演进路径；以新能源产业为例进行实证研究，辨识新能源产业核心领域的战略性新兴技术，探索其技术体系的演化机制。结合专家调研和访谈，对研究的方法和结论进行检验和修正，为新能源产业发展与管理提出科学的对策和决策依据。  

# 3.1.2 主要研究方法：  

# （1）定量研究和定性分析相结合  

本项目采取定量研究与定性分析相结合的方法。将定量研究结果与定性分析的结论相比较，以验证结论的有效性。对可视化图谱进行定性分析，深入诠释定量研究结果。运用的具体方法是专利计量方法，包括共现分析、引文分析、共被引分析等。技术手段主要是信息可视化技术，应用工具包括CiteSpace可视化应用软件系统，NetDraw社会网络图谱绘制软件，Spss统计分析软件等。  

# 专利计量方法  

将科学计量方法移植到对专利信息数据的分析。应用数理统计和计算技术等数学方法对技术活动进行定量分析，从中找出技术活动和发展规律的一种研究方法。通常结合信息可视化技术对专利计量结果进行直观的可视化展示。  

# 信息可视化技术  

利用计算机可视化信息处理软件，通过直观的动态图像信息处理的方式，显示出技术领域中出现的新兴技术的复杂现象，从而获得详尽的新兴技术信息分析结果，了解和预测新兴技术演化动态，在复杂的技术领域中开辟新的未知领域等。  

# (2) 专家意见咨询与调查  

本课题拟邀请能源能产业核心领域的技术专家和政府战略性新兴产业政策顾问组成专家顾问组。以专家顾问咨询、调查问卷、Email等形式征询专家意见，与计量研究方法相结合，确保分析结果的权威性、科学性。  

# (3) 实证分析方法  

在方法和理论研究的基础上，结合新能源产业核心领域的战略性新兴技术为例，分析其创生和演化机制，验证可视化辨识方法的合理性和科学性。  

# 3.1.3 数据来源与处理  

# （1）来源数据库：  

本研究所需的数据主要是用于专利计量分析的专利数据，辅助用于科学文献计量分析的科学引文数据库。本项目研究主要的专利数据来源于世界权威专利数据库《德温特创新索引》。科学文献数据则来源于美国科技情报所(ISI)开发的大型规范化的数据库SCI和SSCI等。  

# （2）建立研究数据库：  

数据下载后，需要进行标准化处理并建立相应数据库以备分析：  

——新能源核心技术领域的科学文献数据库；  
——太阳能电池技术的专利数据库；  
——氢能与燃料电池技术专利数据库；  
——风能发电技术专利数据库；  
——生物质能技术专利数据库；  
——专家调查访谈数据库。  

# 3.2 可行性分析  

# 3.2.1 可行性论证  

# (1) 理论上可行  

本课题申请人和课题组成员具有比较深厚的科学学与科学技术管理、专利计量学、科学计量学的理论基础。研究方向包括科学技术前沿分析、信息可视化技术、专利计量、产业发展战略、科研管理、科技政策等多个方面。这为本课题的完成提供了理论上的可行性。  

# (2) 知识技能上可行  

申请人曾作为联合培养博士在享有“科学计量学之都”的美国费城Drexel大学的信息科技学院留学，在当前国际“信息可视化技术领军人物”Chaomei Chen博士的指导下，系统学习基于CiteSpace软件系统的信息可视化方法和技术。申请人也是将CiteSpace信息可视化软件第一个引入国内，并开展理论及应用研究的学者。2007年，申请人在《科学学研究》发表第一篇使用CiteSpace软件系统测探科学研究前沿的论文《战略管理学前沿演进可视化研究》。CiteSpace可视化软件系统于2007年被首次引入到国内以来，得到了迅速的发展和广泛的传播，引起了国内信息科学和科学计量学、文献计量学界研究者的广泛关注和使用。申请人及项目组成员所在的大连理工大学WISE实验室开创了国内科学计量学研究的新范式，被著名科学计量学家加菲尔德博士（Dr. Garfield）誉为“世界五大科学计量学研究中心之一”。因此，申请人及项目组成员具备完成本项目研究的扎实的知识基础和科研能力。本项目在申请人近年来相关研究和博士期间研究的基础上，将是对先前研究成果的深化和提高。目前本项目的理论构建、研究方案的设计、知识基础条件、技术手段等都比较成熟和切实可行。  

# (3) 技术上可行  

申请人具有熟练使用本项目研究的相关软件的能力。比如Citespace、NetDraw、Derwent Analystist和Spss等，这些软件是当前国际专利计量领域研究的最先进的应用工具，他们不仅能够对专利数据进行计量分析，并且可以绘制专利可视化图谱，直观的展示专利数据计量的结果。作为大连理工大学WISE实验室成员，可以方便掌握这些专利计量和信息可视化最先进的相关软件工具和技术。同时，项目组主要成员也都能够熟练地使用这些软件进行科研。课题组成员王贤文、刘盛博等还能够自己开发研究过程中所需的部分科研软件，这为本项目的研究提供了技术上的保障。  

# (4) 数据库、设备材料可行  

大连大学图书馆和大连理工大学图书馆购买了中国全文期刊数据库和美国科学情报所(ISI)开发的、大型规范化的商业数据库，世界权威专利数据库《德温特创新索引》等，为战略性新兴技术的辨识提供了极其理想的、规范的专业数据库。  

此外，大连大学和大连理工大学的两个实验室都具有数十台高端配置的计算机、宽带高速运转的服务器等硬件资源，保证了本项目研究的数据库、设备材料的可行性。  

# 3.2.2 试验方案  

本项目研究过程中，将综合运用国际上多种专利计量与信息可视化技术手段，采用多种不同的国际先进的计量软件和信息可视化软件，进行反复的测试、对比和分析。并在相关专家的咨询建议下，进行数据选取、图谱绘制、图谱解读、方法选择与调整和应用分析的校准。  

# 3.2.3 技术路线  

本项目的研究工作，将遵循以下的技术路线：  

# (1) 项目总体设计  

通过国内外文献检索和调研、专家咨询，确定项目的总体研究目标、研究内容和整体研究方案。  

# (2) 建立数据库，确定研究方法和技术  

本项目研究以新能源产业领域为例进行实证研究，将建立新能源产业的核心领域（太阳能电池、氢能、风能等）的专利数据库，科学文献数据库和专家调查数据库。初步确定以专利计量、知识图谱和信息可视化技术为主要的分析方法。  

# (3) 建立战略性新兴技术辨识的系统方法  

提出并界定战略性新兴技术及其特征，基于专利计量方法和信息可视化技术，对战略性新兴技术的创生和发展进行辨识。结合专家访谈与调查，检验并修正辨识方法的可行性，建立系统的战略  

性新兴技术辨识方法。  

# (4) 探索战略性新兴技术体系的演化机制  

在对战略性新兴技术进行辨识的基础上，依循科学知识基础—新兴技术--关键技术—核心技术—共性技术的技术创生和发展路径，以及技术单元（基因）--技术单元一技术链一技术群--技术网络一复杂技术系统，探索战略性新兴技术体系演化机制。  

# (5) 战略性新兴技术辨识方法的应用分析  

以新能源产业为例，通过对新能源产业的核心领域中的战略性新兴技术的辨识和技术体系的演化机制的分析，提出我国新能源产业发展和管理的对策。  

# 4. 本项目的特色与创新之处。  

# 4.1 本项目的特色之处：  

结合我国大力发展战略性新兴产业的宏观背景，提出战略性新兴技术的概念，分析战略性新兴技术的特征。从专利计量和信息可视化的视角，重点解决如何辨识和确认产业发展中的战略性新兴技术的可视化方法。从微观、中介和宏观以及技术体系内部演化、产业中技术创新、外部环境共生等方面，探索战略性新兴技术体系的演化机制。对我国新能源产业的战略性新兴技术的创新、辨识和演化的路径进行实证分析，提出基于战略性新兴技术演进的新能源产业发展与管理对策。  

# 4.2 本项目的创新之处：  

# 4.2.1 建立和完善战略性新兴技术辨识的可视化方法  

提出战略性新兴技术的概念及其特征。基于专利计量、引文分析理论和信息可视化技术，建立和完善战略性新兴技术的创生和辨识的系统的可视化方法。为准确把握战略性新兴技术创生和演化，确认优先发展领域提供直观的可视化探测手段。  

# 4.2.2 构建从战略性新兴技术体系到战略性新兴产业的优化演进路径  

基于战略性新兴技术的可视化辨识和知识流动理论视角，以技术单元(基因)-技术链-技术群-技术网络-复杂技术系统为基础，构建从战略性新兴技术体系到战略性新兴产业的优化演进路径。根据技术体系演化与技术创新的密切关联，为创建新能源产业发展的技术创新系统，搭建知识基础平台。  

# 4.2.3 提出基于产业发展优化演进路径的新能源产业发展对策  

以新能源产业核心领域的战略性新兴技术的可视化辨识为例进行实证研究，在构建从战略性新兴技术体系到新能源产业发展的优化演进路径的基础上，有针对性地提出我国新能源产业发展与管理对策。

本项目的研究是在申请人多年从事的信息可视化技术和科学计量、专利计量领域研究的基础上，结合当前我国大力发展战略性新兴产业的背景下，提出的具有重要理论和现实战略意义的研究课题。尤其是针对新能源产业，申请人在曾参加的国家自然科学基金课题中作为新能源技术子课题的负责人，曾对新能源产业核心技术领域的知识基础结构、太阳能电池专利共被引网络等进行过专门的可视化研究，这些研究为本项目的新能源产业的战略性新兴技术的辨识的可视化分析提供了研究的基础。同时，申请人近年来的关于信息可视化技术和专利计量方法也为本项目的研究提供了充分的方法保证和技术支撑。
