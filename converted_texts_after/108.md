# 报告正文

# 一、立项依据与研究内容：

# （一）项目的立项依据

《京都议定书》于2005年2月16日生效，我国虽然以发展中国家的身份不承担定量减排  $\mathrm{CO}_{2}$  的义务，但中国目前是世界上仅次于美国的全球第二大  $\mathrm{CO}_{2}$  排放国，承担的压力越来越大，加强我国碳源汇强度的研究，对国家未来的环境外交有重要的政治意义。

全球草地面积约占陆地总面积的  $25\%$  ，超过  $3000\mathrm{Mkm}^2$  ，是全球陆地生态系统的主要类型之一；草地生态系统也是当前人类活动影响最为严重的区域，其碳通量对全球碳循环有着重要影响。我国拥有草地4亿公顷，占国土面积的  $41.7\%$  ，草地碳循环的研究是我国陆地碳收支评估中重要的组成部分。

草地生态系统区别于森林等其他陆地生态系统的特点之一，就是它不具有固定而明显的地上碳库，其碳素贮量绝大部分集中在地下土壤中，主要是通过冠层光合作用而积累的，因此草地冠层光合作用和土壤呼吸是理解草地生态系统碳循环的重要环节。草地生态系统与大气之间的碳交换受温度、光照、水分等环境条件的综合制约，其关系很复杂。要揭示其复杂的机理，一方面要进行长期的实地观测，但对幅员辽阔的草地生态系统，要进行详尽的实地观测是难以实现的，在解决大尺度估算和预测等实际问题中，模型模拟是最重要的技术途径。按建模思想可以将模型划分为经验或统计模型、半经验模型和机理模型三大类，前两种模型一般适用于某个区域的特定生态系统类型，通用性差，而理论模型因机理性强，适用性好，成为该研究领域的热点和前沿。

草地生态系统碳通量是生态系统各组分光合生产和呼吸释放的代数和，机理模型一般将其简化为冠层的净光合生产和土壤层（含根系）呼吸两个部分，后者主要利用呼吸观测与环境温度和湿度建立经验关系来推算，模型相对简单。机理较复杂的是冠层净光合生产过程，用冠层光合生产的生理生态学模型进行模拟，它是整个模型的核心，常用的有大叶模型、多层模型和二叶模型。大叶模型将冠层简化为一个拓展的叶片，并将单叶的生理生态学过程拓展到冠层，将能量平衡、光合作用、呼吸作用等过程结合起来，寻求模型简化与机理完整性之间的平衡，典型的单叶模型是Amthor（1994）提出的，由于模型对冠层的简化、参数化方案经验性较强等原因，模拟结果与实际有较大的偏差。多层模型考虑了冠层叶片和环境要素的垂直结构，更接近于实际情况，所以应用较多。它将冠层(包括叶片和空气)划分为垂直的若干层次，分层考虑阳生叶和阴生叶，依据冠层生理和环境要素的垂直结构逐层计算各层的通量，并累加成冠层尺度的量。Leuning等（1995）提出的冠层光合作用时空积分模型具有较好的代表性，它假设植物的叶片随机分布，模型区分了阴生叶与阳生叶所接受辐射的差异，使用耦合的光合—气孔模型，引入光合能力与冠层氮含量的关系，使用简便有效的冠层五点Gaussian积分方法，计算冠层通量。二叶模型是在多层模型的基础上由Wang和Leuning(1998)发展起来的，它将冠层简化为受光照叶片和被遮荫叶片两个部分，对土壤-植被-大气间的辐射与能量交换进行了改进，并对气孔导度公式进行了土壤水分订正，使得计算速度比多层模型快了10倍。尽管不同学者提出的光合生产的生理生态学模型有一定差异，其模型的核心原理是相似的，一般都包括光合作用-气孔导度耦合子模型、辐射吸收子模型、表面能量平衡子模型、叶片生理参数分布子模型及湍流输送子模型等。

近30年来，国外学者对草地碳通量机理模型进行了研究，并以多层模型和对气候变化的响应居多。国内相关的研究主要针对草地碳交换的直接观测以及遥感模型。这些研究对象几乎都是针对羊草草原或者整个生态系统，没有考虑草地不同植物种固有的生理差异和它们所处的环境生态位差异，例如不同植物种的光合特性不同，展叶、落叶等物候期不同，由于高度不同，导致叶片所处的光照也不同，因此草地内不同植物种光合生产过程有明显的差异，均一化处理或简单求和必然使模拟结果产生偏差。只有将植物种光合生理参数和环境生态位引进机理模型中，才能准确地揭示草地生态系统碳交换的机制。天然草地绝大部分含有多个植物种，所以建立具有明确生理生态学机理的草地碳通量模型具有重要的科学意义。

全球各类草原中，单位面积温带草原的碳贮量约为热带草原的2—4倍。科尔沁是我国典型的温带草原区，在东西向延伸的多条沙带之间分布着大面积的草甸草地，主要是以莎草(Cyperaceae)、芦苇(Phragmites communis)、羊草(Aneurolepidum chinense)、紧穗三棱草(Bolboschoenus)、牛鞭草(Hemarthria compressa)、委陵菜(Potentilla chinensis)等为主的群落（地上生物量约占生态系统的  $90\%$  )，长势受降水量影响很大，根据降水不同，草地平均植株高度变化于  $30 - 100\mathrm{cm}$ ，是该地区主要的草场资源，在其他温带国家也有广泛的分布，因此以科尔沁草甸草地为对象进行草地碳通量的机理模型研究，不仅能给我国的草地碳通量研究提供科学依据，对全球草地碳收支的评估也具有重要价值。

本项申请拟以科尔沁草甸草地为研究对象，依托相关生态实验站开展研究，项目组在碳通量观测研究方面具备较好的研究基础和技术储备，2007年8月在内蒙古自治区某地的草甸草地上安装了国际先进的涡动相关通量观测系统，并安装了SPAC研究方法所需的小气候梯度观测系统，这些系统全部自动观测和存贮数据，可以精确地测定30分钟时间步长的生态系统净碳交换和其他环境因子。同时也配备了叶片光合生理、土壤呼吸、群落结构观测等方面的先进仪器。

本项申请拟充分利用上述的仪器设备，开展温带草地碳通量机理模型研究，拟采用机理性明确的多层模型，并与大叶模型比较，以草甸草地不同植物种的光合作用生理参数和叶面积垂直分布的差异为模型创新的核心，力争构建具有自主知识产权的温带典型草甸草地碳通量机理模型。

# （二）项目的研究内容、研究目标，以及拟解决的关键问题

# 1. 研究目标

以科尔沁草甸草地为研究对象，利用土壤-植物-大气连续体（SPAC）分析方法，构建草甸草地生态系统碳通量机理模型。根据群落结构特点和优势植物种叶片生理差异，建立冠层的辐射吸收子模型、表面能量平衡子模型、光合作用-气孔导度耦合子模型、湍流输送子模型和叶片生理参数子模型。以冠层、大气和根系土壤环境要素的自动观测为模型的环境输入条件，进行草甸草地生态系统碳通量的动态模拟，利用同步观测的生态系统尺度的涡动相关通量数据及土壤呼吸计算结果，对模型模拟结果进行检验和比较，揭示草甸草地生态系统碳交换的过程机理，为我国草原碳循环的评价和预测提供理论依据。

# 2. 研究内容

（1）科尔沁草甸草地群落结构和叶面积特征研究

根据生态系统群落调查结果，分析主要植物种密度、叶面积指数特征，进而确定群落内主要物种的叶面积垂直分布规律和总量比例关系，及其季节变化规律。  
(2) 主要植物种关键光合生理参数及其季节动态分析

不同物种叶片光合作用的量子效率  $(\alpha)$  和 RuP2(二磷酸核酮糖) 光合羧化速率 Vcmax, 分析 Vcmax 与 N 含量的关系, 进行不同季节的观测, 分析其季节动态。

(3) 科尔沁草甸草地冠层净光合模拟研究

前两项研究结果和小气候（土壤）梯度观测数据为输入条件，构建温带草甸草地冠层净光合生产的多层模型（辐射吸收、表面能量平衡、光合作用-气孔导度耦合、湍流输送和叶片生理参数五个子模型耦合在一起构成），分析环境条件（光、温、水）对冠层光合生产的综合影响规律，并与大叶模型进行比较。

(4) 科尔沁草甸草地生态系统碳通量模拟研究

冠层净光合模型与土壤呼吸模型结合，构成草地碳通量模型，利用生态系统尺度的涡动相关观测数据进行验证，并对模型进行主要植被参数和环境参数的敏感性分析。

# (5) 科尔沁草甸草地碳通量预测分析

利用该模型预测不同植被结构和环境条件情景下科尔沁草甸草地生态系统碳通量的变化规律。

# 3. 拟解决的关键问题

(1) 将不同优势植物种叶片的光合生理差异（光合作用的量子效率  $\alpha$  、RuP2 的光合羧化速率 Vcmax 等）和冠层结构差异（叶面积垂直分布）反映在各子模型中。  
(2) 用模型准确描述草甸草地的  $\mathrm{CO}_{2}$  通量与环境要素的关系是本项目的核心。

# （三）拟采取的研究方案及可行性分析

# 1. 技术路线

以科尔沁草甸草地为研究对象，以Farquhar的光合作用生理生化模型、Ball-Berry的气孔导度子模型和Leauning的多层辐射子模型为基础，通过群落物种、叶面积分布、叶片光合生理特征等综合观测，结合草地小气候梯度和土壤温度湿度分层观测，建立草地的冠层光合生产模型（含五个子模型)，结合土壤层呼吸模型，构建草甸草地碳通量机理模型，并利用生态系统尺度碳通量的涡动相关观测数据对模型进行验证和比较。主要技术过程包括：

(1) 群落主要植物种叶面积的垂直分布和冠层辐射吸收子模型的建立：根据生态系统群落调查结果，确定主要物种的密度、高度，估算不同植物种叶面积的垂直分布、分配比例及其季节变化，按照植物种分层计算冠层辐射吸收方程，进而建立群落辐射吸收子模型。  
(2）草甸草地冠层能量平衡子模型建立：根据能量平衡原理，将叶面的蒸腾速率、显热交换与辐射吸收等过程耦合起来，求解动态的叶面温度，作为光合作用-气孔导度耦合子模型的输入条件。  
(3) 草甸草地叶片生理参数与冠层光合作用-气孔导度耦合子模型的建立：根据不同植物种叶片光合-光响应曲线（An-PAR 曲线）测定结果确定光合作用的量子效率  $\alpha$ ；根据不同  $\mathrm{CO}_{2}$  浓度下光合-胞间  $\mathrm{CO}_{2}$  浓度曲线，确定 RuP2(二磷酸核酮糖) 的光合羧化速率 Vcmax，分析 Vcmax 与 N 含量的关系。将这些动态参数代入 Farquhar 光合生理生化模型和 Ball-Berry 气孔导度子模型，建立耦合的光合作用-气孔导度子模型，并对气孔导度模型进行土壤水分订正。  
(4) 湍流输送子模型的建立：根据叶片特征尺度、冠层内和冠上风速垂直分布规律，分别建立叶片边界层、冠层和冠上大气边界层的传输导度，构成湍流输送子模型。  
(5) 土壤层呼吸模型的建立：根据土壤层呼吸、土壤湿度和土壤温度的同步观测结果，建立土壤层呼吸模型。  
(6）草甸草地生态系统碳通量模型的构建、检验与敏感性分析：将上述五个相互耦合的子模型与土壤层呼吸模型结合，构成草甸草地碳通量模型，利用生态系统尺度的涡动相关观测数据进行验证，并对模型进行主要植被参数和环境参数的敏感性分析。并将构建的多层模型与大叶模型进行比较，分析二者的在草甸草地的适用性。

# 2. 实验方法

观测地点在内蒙古自治区某地的草甸草地上。温带大陆性季风气候，年均温度  $6.2^{\circ}\mathrm{C}$ ，多年均降水量  $439\mathrm{mm}$ ，集中于  $6\sim 8$  月份，雨热同季，年均总辐射  $5035\mathrm{MJ}\cdot \mathrm{m}^{-2}$ ，日照时数  $2837\sim 2892\mathrm{h}$ ，地势平缓。主要土壤类型为草甸区风沙土。根据年降水不同，草地平均植株高度变化于  $30 - 100\mathrm{cm}$ 。

为了使模型模拟结果与涡动相关观测结果有较好的比较性，群落调查、光合作用和土壤层呼吸等观测和实验在涡动相关探头的信息源区内进行，根据2007年8-9的观测数据分析，主要信息源区在观测塔周围  $100\mathrm{m}$  以内的范围。主要观测和实验内容如下：

# (1) 群落结构调查

在观测塔周围设计15—20个  $1\mathrm{m}\times 1\mathrm{m}$  小样方，调查主要植物种的密度、叶面积指数的季节变化和垂直分布，包括莎草(Cyperaceae)、芦苇(Phragmites communis)、羊草(Aneurolepidum chinense)、紧穗三棱草(Bolboschoenus)、牛鞭草(Hemarthria compressa)、委陵菜(Potentilla chinensis)等植物种。

# (2）优势物种叶片光合生理观测

光合-光响应曲线的观测：选择上述6个优势植物种（地上生物量占生态系统的  $90\%$ ），利用光合作用分析仪，分别进行叶片光合-光响应曲线的观测，采用原地观测方法，1小时内完成3—5个观测重复。生长季每个月进行1—2次观测，5—9月共进行5—8次观测。测定的曲线用于叶片光合参数确定（光合作用的量子效率  $\alpha$  、最大光合速率Pmax和暗呼吸强度R）。

光合-  $\mathrm{CO}_{2}$  浓度响应曲线的观测：利用光合作用分析仪和外置  $\mathrm{CO}_{2}$  小钢瓶，进行不同  $\mathrm{CO}_{2}$  浓度下的叶片光合作用曲线的测定，采样方法和观测频次同上。测定的曲线用于胞间  $\mathrm{CO}_{2}$  浓度Ci和RuP2光合羧化速率Vcmax的确定。

叶片N含量测定：在光合-光响应曲线的观测的同时，采集冠层上、下部位的叶片各数十片，测定其鲜重和叶面积，烘干研碎，用元素分析仪分析N含量，测定的结果用于确定参数Vcmax与N含量的关系。

# (3）生态系统尺度的  $\mathbf{CO}_2$  通量观测

$\mathrm{CO}_{2}$  通量自动观测系统探头安装在草地微气象观测塔上  $3 \mathrm{~m}$  高度的伸展臂上, 由三维超声风温仪和开路  $\mathrm{CO}_{2} / \mathrm{H}_{2} \mathrm{O}$  分析仪组成, 原始采样频率为  $10 \mathrm{~Hz}$ , 数据传输给数据采集器进行存储, 同时进行 30 分钟的通量计算并存储, 从 2007 年 8 月 20 日开始一直连续观测。观测结果用于生态系统碳通量机理模型的验证。

# (4) 气象要素梯度观测

包括3层大气温度和湿度和风速，高度分别为0.3, 1.5, 3m，光合有效辐射设于1.5m, 地温设置深度为5, 10, 20cm；土壤水分TDR探头设于5, 20cm这些要素的原始采样频率为0.5Hz，通过数据采集器采集并按30分钟计算平均值进行存储。从2007年8月20日开始一直连续观测。观测结果用于生态系统碳通量机理模型的输入条件。

# (5）土壤层呼吸观测

利用便携式光合作用分析仪的土壤呼吸观测系统，进行野外实地和采集土样的呼吸观测。野外实验选取5—8个样点进行观测（观测前割除地上植物)，生长季每个月进行1—2次观测，5—9月共进行5—8次观测。同时在研究区内采原状土(0.8m左右深度)，分4层放入容器内，每层测3个重复，每周观测一次，连续观测3—4个月。

(6) 群落叶面积和物候观测：生长季每5—10天进行1次叶面积观测，生长初期和末期采用样方收割法，生长旺季采用光学仪器观测，物候观测根据草地物候观测规范，观测结果用于生态系统碳通量机理模型的输入条件。

# （四）本项目的特色与创新之处

(1) 构建的草甸草地生态系统碳通量的机理模型，在建模机制上不同于国际上常见的单一物种的草地或简单的物种比例求和的方法，实际测定不同植物种叶片光合生理参数差异，分析不同植物种因冠层高度不同而导致的吸收辐射差异，从而揭示草地不同植物种对生态系统碳通量的贡献。  
(2) 模型可以预测该草地生态系统在物种组成比例变化和环境变化条件下碳通量的变化规律。

# （五）完成自然科学基金项目情况

国家自然科学基金“红松针阔叶混交林生态系统碳收支动态及其驱动机制研究”，总负责（主持)。

完成情况：按任务书顺利完成。

与本申请项目的关系：已完成项目实施过程中掌握的涡动相关观测技术和数据分析方法是本申请项目的基础，本申请项目将把这些方法和技术应用到草地生态系统中，获取草地生态系统碳通量和环境要素的高时间分辨率监测数据，为建立草地生态系统碳通量机理模型提供检验和验证数据。

总结摘要：森林的碳收支是《联合国气候变化框架公约》及《京都定书议》关注的焦点，不仅直接影响气候变化预测及其影响评估的准确性，也直接关系到国家环境外交与履约。本项目以我国温带典型阔叶红松林为研究对象，基于国际公认的碳通量观测技术与植物生理生态观测，探讨了森林生态系统过程对碳收支的影响。在碳通量研究的方法论方面，重点分析了涡动相关技术的观测方法和数据质量评价和分析技术体系（包括通量信息源空间分布的 footprint 分析、通量的综合修订和能量闭合分析）；在碳通量的特征分析方面，揭示了阔叶红松林生态系统日、季、年动态及其主导控制因子，阐明阔了叶红松林生态系统的固碳机制，建立阔叶红松林生态系统碳收支的动力学模型。研究结果表明，该涡动相关观测系统的信息源分布于迎风方向 0—1500m 之间，横向范围在 -1000—1000m 之间，最大值分布在 100—400 米距离内，能量闭合度在 80—90%。该生态系统碳通量的日变化主要受光照和空气湿度的影响，季节变化主要与叶面积、温度等因子有关。构建的生态系统 C02 通量动力学模型反映了环境因子（气温、土壤温度、光合有效辐射和水汽压饱和差）的影响机理。
