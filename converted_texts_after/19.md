项目名称：肿瘤的糖化学生物学前沿研究

# 一、关键科学问题及研究内容

# 拟解决的关键科学问题：

本项目围绕发现肿瘤功能相关糖链这一目标，拟解决如下关键科学问题：

# 1. 肿瘤相关糖链探针的可获得性问题

随着糖化学生物学、糖组学和糖类药物研究的快速发展，对糖类化合物样品种类和量的需求大幅度增加。如何快速、大量地获取高纯度的糖类化合物便成为糖科学研究中最重要的基础性问题之一。对于糖类化合物而言，由于存在糖型、多功能团修饰、分枝、端基异构等多种因素的影响，使得其结构变得复杂和多样又由于自然界中糖类化合物存在着微观不均一性，寡糖类化合物一般具有高极性和良好的水溶性，而且在肿瘤细胞等组织中含量极低，所以在目前技术水平下，试图采用分离手段从生物材料中直接获取组分均一、结构明确的糖链非常困难。此外，糖类化合物不是基因控制的直接产物，它的合成并不是有模板的复制除受酶基因表达的调控外，还受酶活性的影响，所以糖链合成也不能像核酸和蛋白质那样可采用扩增方法和表达手段进行。因此，如何解决糖类化合物的可获得性问题，是以它为探针探索糖链在肿瘤中的功能首先要面临的一个关键问题。本项目拟发展糖链的高效合成方法，并以此为基础采取体外化学合成和生物合成的途径合成肿瘤功能相关的糖链，为糖链在肿瘤中的生物学功能及干预调控研究提供分子探针。

# 2. 糖链在肿瘤细胞及其微环境中的变化规律及其功能机制

肿瘤是机体细胞失去对其生长的正常调控而形成的新生物，是细胞自身调控和其所处的微环境相互作用的结果。其中，细胞表面和其所处的微环境中大量糖链修饰改变以及凝集素受体功能异常在肿瘤发生发展和转移过程中发挥极其重要的作用。糖链合成的调控以及糖链与凝集素的相互作用参与了肿瘤生物学行为的各个方面，研究和阐明糖链结构及凝集素受体的变化规律和调控机制是揭示肿瘤的生物学行为的一个重要方向。糖基化修饰是蛋白质功能的一个更高层次的调节机制，单从基因、蛋白质的水平揭示肿瘤发生机制是远远不够的，必须从糖链的角度更加全面地认识糖链在肿瘤中的作用，发现肿瘤中糖链的变化规律和调控机制，从而为肿瘤的诊断治疗提供新策略。因此，利用糖化学生物学研究的理念，通过对糖链结构特征、调控糖链合成的糖基转移酶和实现糖链功能的凝

集素受体展开系统研究，阐释肿瘤中糖链的变化规律和调控机制是我们要面临的另一个关键问题。本项目拟以人肺癌、肝癌、乳腺癌和宫颈癌作为研究对象，研究肿瘤相关糖链在肿瘤中的表达、结构、功能特点及其调控规律，筛选和鉴定肿瘤的特征性糖链，为肿瘤诊断和治疗的研究提供新策略和新靶点。

# 围绕关键科学问题，本项目的主要研究内容可分解为如下四个方面：

# 1. 糖链探针的化学合成研究

近年来化学生物学的迅猛发展为利用化学小分子探针了解生命过程提供了新的机遇。化学合成方法是目前获得纯的结构确定的糖类化合物的主要途径。本课题将以肿瘤中特异性  $N-$  糖链和  $O-$  糖链的特征糖链为靶标，设计寡糖探针分子，发展寡糖化学合成的新方法和新策略，以它们的高效合成甚至自动化合成为导向，实现寡糖链的快速而高效的组装，构建肿瘤特异性糖链探针化合物库。另一方面，肿瘤中糖链的异常与糖链加工酶（糖苷酶和糖基转移酶）的活性异常直接相关；因此，本课题拟设计和合成与肿瘤相关的糖链探针，为肿瘤相关的糖链功能研究提供化学工具，并进而发现抗肿瘤药物先导化合物。

# 2. 糖链探针的生物合成研究

由于糖类化合物结构的复杂性，糖链特别是具有复杂结构的细胞表面糖链的合成在某些情况下单纯依靠化学的方法难以实现或合成难度增大。随着生物技术的发展，越来越多的寡糖链可通过微生物或酶法得以合成。为了探索糖链在肿瘤中的生物学功能和机制，需要合成具有结构多样性的寡糖链探针分子。但是，寡糖链及其衍生物的生物合成还面临着许多瓶颈问题：一是自然界不存在的单糖衍生物难以获得，从而使设计出的寡糖衍生物没有相应的单糖原料合成；二是参与糖链合成的天然糖基转移酶专一性强，无法应用非自然界存在的单糖衍生物合成糖链；三是糖苷酶合成寡糖的产率低。本项目拟构建含稀有糖单元的糖库及其糖核苷酸库，并对糖链合成相关糖基转移酶和糖苷酶进行分子改造以拓宽糖基转移酶对糖底物的选择性和提高糖苷酶的区域选择性和产率，建立一套生物法合成寡糖及其衍生物的完整方法，为研究肿瘤相关糖链的功能提供探针样品，进而为进一步研究开发肿瘤治疗疫苗或者药物奠定基础。

# 3. 糖链对肿瘤细胞生物学行为的影响及机制研究

糖基转移酶催化合成糖链，糖链通过修饰连接到蛋白质或脂类分子上，生成糖蛋白或糖脂而发挥功能。功能糖组学的研究已经证明，与正常细胞和组织相比较，肿瘤细胞和组织表达特征的糖蛋白糖链和糖基转移酶。因此，揭示肿瘤特征的糖基转移酶与糖蛋白及糖蛋白糖链之间的特定靶向修饰关系，成为阐明肿瘤细胞特征糖链和糖基转移酶功能的关键。但是，由于研究体系复杂又缺少有效的研究手段，目前鲜有相关报道。本项目拟利用课题1、课题2合成的糖链探针和糖基转移酶，建立糖链探针的体内外肿瘤相关糖蛋白质标记、筛选技术，发现

肿瘤相关糖基转移酶修饰的靶糖蛋白及糖链，研究它们的变化规律，阐释糖链对肿瘤细胞生物学行为的影响及分子调控机制，为肿瘤治疗提供新策略和新手段。

# 4. 糖链受体在肿瘤微环境中的功能机制研究

细胞表面糖链在细胞发挥正常生物学功能中发挥着重要的作用，而糖链与其受体的识别是实现糖链生物学功能的重要步骤。在肿瘤微环境中，凝集素作为糖链的受体，参与了肿瘤的发生发展和转移过程，具有举足轻重的作用。利用课题1和课题2合成的特异性糖链探针以及肿瘤表面糖链，通过二维电泳-质谱分析、芯片技术等分析手段，通过大规模筛选肿瘤组织标本，获得在肿瘤发生发展和转移中发挥重要作用的特征性凝集素，进一步在细胞水平和动物模型中阐明特征性凝集素在肿瘤发生发展和转移中生物学功能和分子机制。另一方面，通过凝集素亲和纯化和质谱鉴定肿瘤微环境中特异表达的糖肽，分析和研究肿瘤中糖链特异改变的糖蛋白及其糖链异常结构，解析糖基化修饰在肿瘤发生发展中的功能机制。在此基础之上，充分利用糖化学生物学手段，筛选和研究特异凝集素和糖链为靶向的糖链探针，为进一步开发以凝集素和糖链为靶向的糖类药物奠定基础。

# 二、预期目标

# 总体目标：

通过糖化学与糖生物学两大学科的交叉和整合，以我国发病率较高的肺癌、肝癌、乳腺癌和宫颈癌为主要研究对象，通过糖化学生物学手段，系统地阐释糖链在肿瘤中的生物学功能及其分子机制；发现肿瘤的糖链、糖基转移酶和凝集素靶标；建立高效合成糖链的化学合成法和生物合成法，为阐明糖链在肿瘤中的结构与功能提供化学工具和探针，并进一步针对肿瘤中糖基化异常进行干预和调控；进而阐释肿瘤特征糖链在其信号转导中的生物学功能及调控机制，从而为肿瘤特性的分子机制诠释和靶向治疗奠定基础。

# 五年预期目标：

1）在糖基化方法和寡糖组装策略方面有所创新，并将发展的方法和策略应用于肿瘤相关寡糖链的合成，建立糖链探针化合物库，为肿瘤糖链生物学功能及抗肿瘤糖类药物研究提供强有力的工具，使糖链的合成向着自动化的最终目标迈进，提升我国在糖研究领域的国际地位。  
2）通过糖化学与糖生物学的整合和交叉，阐释与肿瘤细胞发生、发展和转移相关的糖链结构及功能，及其参与相应糖链合成的糖基转移酶的功能及调控；发现肿瘤特异性糖链、凝集素和糖基转移酶靶标。  
3）发展基于糖链结构的肿瘤诊断和治疗的新方法和新理念，发现肿瘤特异性候选标志分子3-5个，候选药物新靶标1-2个，糖类药物先导化合物5-10个。  
4）以4个课题为项目主要实施载体，力争在五年时间内取得在国际上具有影响力的研究成果。在国际性高学术水平的杂志上发表SCI文章300篇以上，其中影响因子  $>5$  的论文60篇以上，影响因子  $>10$  的论文10篇以上；申请或获得授权专利20件以上；出版专著1-2部。  
5）通过项目的开展，培养若干糖化学生物学的青年优秀人才，使其成为该领域具有承担国家重大项目能力的领军人物；力争推出在本学科具有国际影响力的优秀科学家。培养一批糖化学生物学的专业人才，并带动糖化学和糖生物学等相关学科的建设。培养1-2名杰出青年，培养一批硕士、博士研究生和青年骨干。

# 三、研究方案

# 学术思路：

糖化学生物学为国际科学前沿领域，相关学科交叉广泛，涉及的科学和技术问题众多。因此本项目研究内容的提出，既要考虑需求的合理性、学科领域的前沿性和交叉性，又要考虑我们的基础和特色。利用糖生物学在肿瘤中的功能机制研究结果为糖化学提供研究对象和线索，糖化学研究为糖生物学研究提供物质基础和探针，再进一步促进糖生物学功能研究，使得前沿学科真正交叉，二者相辅相成，相得益彰。本项目选择在我国发病率高的肺癌、肝癌、乳腺癌和宫颈癌为研究对象，采用糖化学生物学研究的技术和手段，从糖链的角度阐明肿瘤在其发生、发展和转移过程中的糖链结构及功能，及其参与相应糖链合成的糖基转移酶以及参与糖链识别的凝集素的功能及调控，发现肿瘤功能相关糖链和新的治疗靶点、靶蛋白，发现新的肿瘤分子标志物，符合国家当前对恶性肿瘤的诊断与治疗方法研究的重大需求；而生物学研究中所需要的各种糖链分子，将作为糖合成方法学研究的目标物。广谱的糖基化方法及寡糖组装新策略的研究是当前糖科学研究中最重要的基础性问题之一，因为如何快速、大量地获取高纯度的糖类化合物已成为制约整个糖科学发展的瓶颈之一。可以说，本项目的研究内容既瞄准了当前国际上糖化学生物学研究的关键科学问题，又考虑到了解决当前国家的重大需求问题。本项目在肿瘤糖链生物学功能研究方面，旨在运用糖化学生物学的研究理念和技术方法，联合国内糖研究的优势队伍并在这些研究队伍前期研究的基础上，以肿瘤的特征糖链为研究对象，重点阐明肿瘤功能相关的糖链结构及功能，及其参与相应糖链合成的糖基转移酶的功能及调控，解决的关键科学问题是肿瘤中糖链的变化规律和功能机制。

在寡糖合成研究方面，选取的目标是为项目的生物学研究提供糖链分子探针，同时发展新的策略和方法使得糖链的合成朝着自动化、标准化的方向行进。还将以微生物来源的糖核苷酸合成酶、糖基转移酶等构建能生产糖链的“超级菌”，发展基于糖基转移酶的多酶组合寡糖生物催化合成体系，同时建立糖苷酶催化的寡糖合成体系，构建相应的寡糖合成技术平台。化学和生物两条寡糖合成途径相互关联，互为补充。糖化学研究力量雄厚。

围绕以上研究内容和目标，本项目设4个课题，计划在5年内完成。首席科学家负责项目的组织实施，课题的具体实施由课题负责人承担。对项目实行动态管理，通过建立服务于项目的网络信息平台、构建肿瘤相关糖链探针化合物库、酶库、结构分析及研究成果数据库等，实现信息、资源共享。并为项目结束后建设国家科技基础条件平台打下基础。

通过糖化学与糖生物学的整合和交叉，以我国发病率较高的肺癌、肝癌、乳腺癌和宫颈癌为主要研究对象，从化学合成和生物合成两条互补的途径构建寡糖探针分子，建立基于高效组装策略的寡糖化学合成平台及基于多酶催化体系的寡糖生物学合成平台，为肿瘤糖生物学研究提供糖链探针；从糖链结构特征、糖链合成调控（糖基转移酶）和糖链功能实现（凝集素）三个方面着手，运用糖链探针研究肿瘤相关特征性糖链和肿瘤微环境中的功能性凝集素受体，揭示糖链在肿瘤发生、发展和转移中的生物学功能和分子机制，发现肿瘤中糖链相关的诊断和治疗靶点；并通过糖化学生物学研究手段，验证和深入研究糖链在肿瘤的功能及分子机制，进而对糖链异常进行干预和调控。

创新点：

本项目首次系统地从分子生物学、细胞生物学、糖合成化学、酶工程及糖生物学等多学科入手，以我国发病率高的肺癌、肝癌、乳腺癌和宫颈癌为研究对象运用糖化学生物学研究手段，阐明糖链在肿瘤中的变化规律、生物学功能及其分子机制，具有鲜明的多学科交叉特色，创新点显著。具体分述如下：

1. 本项目围绕目前制约糖科学发展的瓶颈糖链合成问题，以寡糖的高效合成或自动化合成为导向，既注重糖基化反应的方法和策略的创新，又注重方法和策略在肿瘤功能相关糖链探针的合成中的集成和应用，二者相辅相成、相互促进。  
2. 项目中化学与生物学研究以肿瘤相关糖链为共同的研究对象，相互交叉、互为支撑，相互促进。  
3. 通过比较和研究大量临床组织标本（包括正常组织和癌旁组织）以及患者血清，发现肿瘤微环境中特异性凝集素，使所得到的研究成果更具有临床指导意义。同时通过化学或生物途径合成特征性糖链进一步阐明其生物学功能，为肿瘤的临床诊断和靶向治疗奠定良好的基础。  
4. 采用化学生物学的新方法，从基因到糖链、从细胞内到细胞外等多个层次和水平上，系统研究肿瘤中糖链结构、凝集素受体功能及其合成关键的糖基转移酶基因的定位和表达，揭示肿瘤相关糖链的生物学机制。

要生理活性的复杂糖缀合物的合成，在国际上产生了较大影响，部分成果获得了2010年度国家自然科学二等奖。所提出的多酶催化的“超级菌”概念对寡糖的生物合成具有重要意义；在糖缀合物及糖疫苗方面做出了杰出的工作，他的加盟更增加了糖化学方面的研究力量。已构建了一个小型的  $N$  一寡糖库，内含3一9个糖单元的  $N$  一寡糖核心片断20余个，为大型糖库的构建进行了前期准备。糖化学雄厚的研究实力不仅为糖生物学研究提供了物质上的保障，而且为糖化学与糖生物学的强强联合与相互交叉提供了基础。  
4. 参加本项目的大部分科研人员来自我国糖化学和糖生物学研究的重要研究机构。承担单位仪器设备精良、齐全，有完善的实验动物设施和进行体内外实验的条件，以及分子生物学、细胞生物学、生物化学、药理学、药物化学、分析化学的实验条件，为本项目的顺利进行提供了十分有利的条件。

# 课题1：糖链探针的化学合成研究

# 研究目标：

以肿瘤细胞的特异性糖链为靶标，发展新的寡糖或类糖化合物合成方法和策略，建立糖类化合物合成的技术平台，合成结构多样的糖类化合物探针分子，为其它课题提供化学工具。

# 研究内容：

(1)、糖基化反应基本问题的研究。新的糖基化方法的发展是实现高效寡糖

合成的最根本的条件。拟对糖基化反应的基本问题作深入广泛的研究，包括新离去基、新催化体系、收率、立体选择性、适用性等问题的考察，以发现高效立体专一的糖基化反应。

(2)、寡糖组装策略的研究。组装策略是实现寡糖快速制备的重要基础。主要进行预活化的一釜寡糖合成策略的发展和完善，寡糖合成中的固相、疏水辅助的固-液转换以及离子液体负载策略的研究。  
(3)、肿瘤相关复杂寡糖的合成。将上述发展和完善的寡糖组装新方法、新策略，用于肿瘤细胞相关复杂糖链的合成，为课题3-4的研究提供化学工具。  
(4)、寡糖链修饰物及糖模拟物的设计与合成。基于课题3-4对特征糖链的结构、功能及调控机制的研究结果，设计合成各种寡糖链修饰物及糖模拟物（如亚氨糖等）探针分子，用于糖链和蛋白质相互作用以及糖链加工酶生物功能的研究，进而发现糖类抗肿瘤先导药物。

# 预期成果：

在国际SCI学术杂志上发表论文80篇以上，其中影响因子5以上15篇，影响因子10以上2篇以上；申请发明专利10-15项；发现糖类药物先导化合物3-6个；培养研究生30-40名。

# 课题2：糖链探针的生物合成研究

# 研究目标：

针对肿瘤细胞功能相关糖链，选择肿瘤细胞表面有特异表达的寡糖链  $\mathsf{Le}^{\mathsf{y}}$  和Globo-H等为模式分子，研究该类寡糖及其衍生物的生物合成，进而建立糖类化合物生物合成的技术平台，合成肿瘤相关的糖链，为课题3-4提供探针分子。

# 研究内容：

（1）含稀有单糖单元的糖库及其糖核苷酸库的构建。构建糖库及其糖核苷酸库是糖基衍生化的基础，拟采用新路易斯酸催化剂以获得高立体选择性的烯

糖中间体，再进行功能基（如含氮功能基）的衍生化，进而构建糖库。并利用酶法合成构建相应的糖核苷酸库，为糖基转移酶催化的寡糖合成提供糖基供体。

(2）糖基转移酶和糖苷酶的分子改造。通过理性设计和非理性设计进行糖基转移酶和糖苷酶的定点突变和随机突变研究，拓宽糖基转移酶对单糖底物的选择性以及提高糖苷酶的区域选择性和合成产率。  
(3) 多酶催化反应体系的构建与寡糖探针的生物合成。采用超级粒策略构建多酶催化级联体系，生物合成肿瘤相关糖链及其糖基衍生物，如  $\mathrm{Le}^{\mathrm{y}}$  和 Globo-H 及其衍生物的酶法合成等，制备寡糖链探针。  
（4）寡糖立体构象与功能的相关性。分析肿瘤天然糖链和具有天然糖链相似功能的寡糖衍生物立体构象，确定寡糖立体构象与生物功能的相关性，并对构象进行模拟，指导新型糖链衍生物探针的设计和酶法制备。

# 预期成果：

在国际SCI学术杂志上发表论文80篇以上，其中在影响因子大于5的SCI刊物上发表论文15篇以上，影响因子10以上2篇以上；申请发明专利10项以上；发现糖类药物先导化合物3-5个；出版专著1-2部；培养研究生60名以上。

# 课题3：糖链对肿瘤细胞生物学行为的影响及机制研究

# 研究目标：

利用课题1、课题2合成的糖链探针，建立糖链探针的体内外糖蛋白质标记技术，研究和发现与肿瘤发生发展密切相关的糖基转移酶修饰的靶糖蛋白及其糖链的变化规律，阐释糖链对肿瘤细胞生物学行为的影响及分子调控机制。

# 研究内容：

（1）以课题1、课题2合成的糖探针（如天然糖苷供体衍生物）为底物，通过基于高效色谱的糖基转移酶活性检测技术，研究肿瘤相关糖基转移酶的酶学活性，寻找和验证能够替代糖基转移酶天然糖苷供体底物的糖探针。  
(2) 利用课题 1、课题 2 合成的糖探针（如叠氮化或炔化糖类衍生物），

建立基于蛋白质芯片技术的糖基转移酶修饰的靶蛋白标记检测方法；在此基础上，在体外高通量标记、筛选肿瘤相关糖基转移酶修饰的靶蛋白，发现糖链修饰变化的肿瘤相关靶蛋白和对应的糖探针。

(3) 以上述糖探针标记在肿瘤活体细胞中的糖蛋白, 通过基因转染、RNA干扰技术控制糖基转移酶和糖蛋白表达, 结合蛋白质组学研究技术鉴定和发现, 肿瘤细胞相关糖基转移酶修饰的靶蛋白及其修饰糖链的变化。  
（4）通过分子细胞生物学及动物模型，研究糖化学生物学手段所发现的糖基转移酶及修饰靶蛋白糖链形成的变化，对肿瘤细胞生长、增殖和转移等细胞生物学行为的影响及其规律，阐明其调控的分子机制，为肿瘤治疗提供新策略和新手段。

# 预期成果：

发表SCI论文50篇以上，其中在影响因子大于5的SCI刊物上发表学术论文10篇以上，其中10以上1-2篇；申请专利1-2项；发现肿瘤特异性候选标志分子1-2个；培养1-2名糖化学生物学领域学科带头人，培养研究生60名以上。

# 课题4：糖链受体在肿瘤微环境中的功能机制研究

# 研究目标：

以肿瘤细胞系、动物模型及人类患者的组织标本为研究对象，通过特异性糖链探针、基因芯片和二维电泳-质谱分析等多种研究手段相结合，揭示肿瘤微环境中的特征性凝集素，进一步在细胞水平和动物模型中阐明特征性凝集素在肿瘤发生发展和转移中生物学功能和分子机制，发现肿瘤微环境中与特征性凝集素相关的靶点。

# 研究内容：

（1）通过凝集素芯片和质谱分析等手段研究和鉴定肿瘤细胞表面糖链修饰谱，进一步通过临床肺癌、肝癌、乳腺癌和宫颈癌的组织标本（包括正常组织和癌旁组织）以及患者血清分析肿瘤微环境中肿瘤特征性糖链，发现肿瘤微环境

中2-3种新的特征性寡糖结构，为课题1和课题2糖链探针的设计提供依据。

(2）利用课题1和课题2合成的特异性糖链探针，通过大规模筛选肿瘤组织标本，获得在肿瘤发生发展和转移中发挥重要作用的特征性凝集素。进一步在细胞水平和动物模型中阐明特征性凝集素在肿瘤发生发展和转移中生物学功能和分子机制。  
(3) 通过二维电泳-质谱分析、基因芯片技术等分析手段，比较肿瘤组织标本（包括正常组织和癌旁组织）以及患者血清中凝集素的表达，鉴定肿瘤微环境中特异的凝集素，阐明其识别的糖链结构和在肿瘤发生发展及转移过程中的功能和机制。  
（4）针对肿瘤中特异性凝集素设计糖类化合物，进行大规模筛选获得具有肿瘤抑制活性的先导化合物，进一步在细胞模型和动物模型中阐明其作用机制，为肿瘤治疗提供新策略和新手段。

# 预期成果：

研究成果主要以研究论文形式公布，五年内争取发表SCI论文90篇以上，其中在影响因子大于5的SCI刊物上发表学术论文20篇以上，在影响因子大于10的SCI刊物上发表学术论文5篇以上；申请专利2-4项；发现肿瘤特异性候选标志分子2-4个，候选药物新靶标1-2个；培养1-2名糖化学生物学领域学科带头人，培养研究生50名。
