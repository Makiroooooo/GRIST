项目名称：微生物药物创新与优产的人工合成体系

起止年限：2012.1-2016.8

依托部门：教育部

# 一、关键科学问题及研究内容

# （一）拟解决的关键科学问题

构建微生物药物合成体系的根本目标是实现药物生物合成的定向性和高效性，需要解决微生物药物合成体系的合成与调控、元件及模块组装、底盘细胞与系统优化等科学问题，即（1）微生物药物高效合成与调控的分子基础，（2）微生物药物优质底盘细胞的构建原理；（3）工程化设计生物合成体系的适配性机制；（4）人工合成生物体系对环境的应答机制。

# 1. 微生物药物高效合成与调控的分子基础

微生物次级代谢途径是微生物药物合成的内在基础。在外界特定信号的刺激下，相关药物合成途径得以激活，通过多个催化功能及调控元件的组合进行特定药物分子的合成。药物分子的结构主要取决于催化功能元件的化学选择性及立体选择性，而药物产率则取决于药物合成途径中调控元件以及关键节点酶的催化效率。目前对微生物合成与调控的分子基础已有所了解，应用现有功能元件的优化和替代等代谢工程技术，对药物合成效率有一定促进作用，但是现有知识还无法很好地实现微生物药物的高效制备。加深理解微生物药物高效合成与调控的分子基础，发现并确定合成途径中关键催化元件的作用机制，对其进行理性人工设计及优化，改变催化反应类型或药物主体结构的修饰部位，提高催化反应效率，促进新结构化合物的快速产生及合成速率的提升；构建有序的生物元件组装体，提高反应效率；对调控元件的人工设计和改造，使微生物对人为设计信号进行应答，控制相应药物合成途径激活水平和持续性，是实现大幅度提高药物产量、降低药物生产成本的保证；因此在分子水平上阐明微生物药物高效合成与调控的机理，揭示催化与调控元件的协调作用关系是构建药物微生物合成体系的首要科学问题。

# 2. 微生物药物优质底盘细胞的构建原理

微生物药物底盘细胞的设计是合成人工体系发挥作用的基础。生命的本质是由基因编码的蛋白决定的，一个生物体的基因组序列携带了构成和维持其生命活动所需要的全部信息。从细胞功能必需性的角度可以将基因分为三类：1. 维持细胞基础代谢和执行基本生命活动的必需基因；2. 在特定生境下，克服某些环境压

力，强化生存的条件必需基因；3. 非必需基因。链霉菌作为产生天然药物最大的类群，其线性染色体结构具有核心必需区和柔性可变区，是作为微生物药物底盘细胞的最佳选择。为建立优质的微生物药物底盘，首先应用比较基因组学预测底盘链霉菌的核心必需区和柔性可变区，从后者中进行连续的大片段缺失，使染色体结构更为稳定；从前者中去除可能的次级代谢合成基因簇以及平行转移获得的可移动元件，使简化的底盘细胞进行基本生命活动所需能量更小，生活周期更短。另外一方面，强化或加载一些天然药物合成的共同前体物合成基因，改善菌体生长性状及抵抗发酵过程噬菌体污染的限制-修饰基因，使优质化底盘细胞获得新的生命优势，实现底盘细胞的遗传稳定性和普适性，为微生物药物的高效合成提供平台。

# 3. 工程化设计生物合成体系的适配性机制

在对微生物药物合成及调控机理理解的基础上，应用计算生物学方法构建特定构效关系的生物元件及途径，经热力学评价和动力学模拟，为人工合成体系提供理论指导。以工程学的原理对药物合成途径进行设计，最终在底盘细胞上组装生产线，需要解决途径中各元件间、路径与底盘之间的适配性这个科学问题。为探索适配性机制，可以在体外及体内两个层面上进行。在体外对合成途径进行检验和优化可以绝缘胞内其它因素的干扰，即根据药物的结构特征，设计或构建的新合成途径应该满足反应热力学基本原理，再根据合成途径中的各反应类型，从功能元件库中挑选适宜的元件进行体外组合或装配，模拟计算合成途径的反应进程，检测反应物的流量变化，确定合成途径中的节点元件，对其进行设计和优化，以及环境因子对人工合成途径的影响。在体内需要解决合成途径与底盘细胞其它代谢途径的适配性问题。在封闭的细胞内，其适配性问题主要反映在物质与能量的动态平衡，即结构性物质的流向和电子的传递。基于底盘链霉菌基因组的解析，模拟人工合成途径在代谢网络中的运行，检验代谢流，设计与重新组装调控元件，调节人工合成途径与底盘的适配程度是构建人工生物体系进行药物合成的必然过程。

# 4. 人工合成生物体系对环境的应答机制

在底盘细胞基础上人为构建合成途径，必然会对整个调控网络带来影响，也反映在人工合成生物体系的生理指标方面。因而实现药物的高产量、低成本生产，

还取决于人工生物体系对外界环境的应答机制，主要包括抗逆、与胞外的物质交换和对特定激活信号的响应。由于对复杂生命系统工作原理的了解还存在太多的盲区，因而在基因及基因组的组成与结构水平上研究其抵御生物、物理和化学等因素破坏作用的机制；揭示重要物质的跨膜机理；在调控网络、细胞及细胞群的层面分析人工生物体系对胞外物理化学信号的响应机制。这些都有助于我们深入认识生命规律，所获得的科学认识还为我们在合成途径、调控元件、代谢网络的设计与组装等方面提供理论支撑。

# （二）主要研究内容

# 1. （针对科学问题1）生物元件挖掘与人造功能器件的合成。

针对微生物药物生物合成过程中药物前体合成（初级代谢），药物合成（次级代谢）和药物转运等步骤中涉及到的功能元件和调控元件进行挖掘、分类、定义以及在放线菌底盘生物（链霉菌FR-008）进行标准化。着重于如聚酮类（红霉素、安丝菌素等）、C7N氨基环醇类（伏格列波糖等）、β-内酰胺类（头霉素、克拉维酸等）等合成途径的元件挖掘，从深海极端环境和近海红树林环境中获取更多上述药物生物合成的元件以及应对复杂环境应答的调控元件；对具有新型糖基化修饰功能和调控功能的生物元件进行挖掘，并对其功能和适用范围加以分类；利用蛋白纳米纤维和病毒蛋白骨架自组装的有序特性和特殊的空间结构，构建功能化人工蛋白“脚手架”，实现对功能元件的逐级组装和按设计比例组装，使其发挥最大功能。

# 2. （针对科学问题3）生物元件、路径和网络的模拟与设计。

摸索出一套具有高模块性和通用性的调控元件设计与构建、代谢网络功能控制模块识别和设计方面的计算生物学方法和计算-实验结合技术，构建基于简化氨基酸的随机肽和功能酶库；针对蛋白质改造与设计，提出和发展具有构象和环境依赖特性的新型可极化分子力场以及与之相应的可极化溶剂化模型，最终获得一套崭新的蛋白质改造与设计的理论计算方法和计算程序；设计完成自动获取代谢反应转换模式以及构建基于代谢网络的生物合成途径设计方法相关的技术，构建基于基因组规模代谢网络的生物合成途径设计信息化平台；在放线菌底盘（链霉菌FR008）中插入井冈霉素等微生物药物合成基因簇，研究放线菌底盘通过SOS反应和毒素-抗毒素系统维持外源元件稳定遗传的调控机制。

# 3. （针对科学问题1）催化体系优化与人工组装。

面对人工合成体系的需求，开展酶催化模块的理性设计、定向进化及人工组装工作。从不同结构层次解析蛋白质的模块性，建立重要蛋白质家族中与底物识别、热稳定性、催化效率等性质相关的模块集，并进行功能域/结构模块的设计、替换与优化，获得高度模块化的人工酶；在此基础上应用定向进化技术对关键酶进行性质优化；设计药物前体人工合成途径，从生物元件库中募集不同来源的催化元件，构建体外多酶耦合体系，对催化元件进行优化组合，获得反应效率最大化的体外药物合成多酶耦合体系。

# 4.（针对科学问题2）微生物药物合成的放线菌底盘。

选择微生物中产天然药物最多、遗传操作手段最为成熟的链霉菌为操作平台，利用生物学信息预测与高通量实验相结合，对链霉菌FR008基因组进行设计、改造，建立最简（优）化的放线菌底盘生物；并在此基础上对其染色体进行硫化修饰，赋予其抗酶解等多种生命优势，为发酵过程中的噬菌体污染提供解决之道；建立不同硫化序列和丰度的元件资源库，通过对硫化功能元件和调节元件的定向设计和改造，实现对硫化修饰的人工调控，进一步优化天然药物合成的硫化放线菌底盘，实现抗生素药物的高效合成。

# 5. （针对科学问题1和3）药物合成途径解析与优化。

以聚酮类（红霉素、安丝菌素等）、C7N氨基环醇类（伏格列波糖等）、 $\beta-$ 内酰胺类（头霉素、克拉维酸等）等典型微生物药物为研究对象，充分利用其基因组学和生物合成研究基础，通过计算生物学和计算化学系统揭示其生物合成中的限速步骤及其相关的前体合成、关键生物催化功能大分子（酶）的构效关系，判定活性中心电子结构的分子基础和物理化学特性，建立并利用相关的催化与调控的元件/模块，在原产生菌或底盘链霉菌中实现代谢途径的定向设计和优化，建立上述典型微生物药物分子的合成生物学改造策略。

# 6. （针对科学问题4）新结构代谢物的高效制造。

在生物元/器件、网络设计、酶催化体系、底盘细胞和药物合成途径等不同合成生物学层面研究基础上，以放线菌药物为研究对象，从以下四方面来构建药物结构和调控的改造体系：聚酮合成后修饰模块的替换与导入、生物合成模块的人工改造、生物合成模块的组装及优化和全局转录调控模块作用机制，研究新结

构活性代谢产物途径重构的基本问题。在基因转录、蛋白质表达、中间代谢物三个尺度对具有不同功能模块组合的工程菌进行分析，研究基因和细胞两种不同尺度网络之间的相互作用和耦合关系，实现人工生物合成体系的多尺度优化，最终实现新结构代谢物的高效制造。

# 二、预期目标

# (一) 总体目标：

以建立微生物药物品种创新和性能优化的创新模式为目标，揭示重要产品的高效生物合成与调控机理、优质底盘构建原理、人工合成体系的适配机制及对环境的应答机制，发展微生物药物合成生物学的理论和方法；构建重要微生物药物的功能元件库、底盘生物和人工生物体系，建立重要微生物药物品种创制和性能优化的创新模式，建立微生物药物高效生物制造途径，使我国的合成生物学研究整体水平进入国际前沿，在新型抗生素、抗癌药物等创制和人工设计生物体系等基础研究方面起到国际引领作用，某些方面凸显我国的优势和特色。为提高我国人民健康水平，转变经济增长模式作出积极的贡献。

# （二）五年预期目标：

1. 阐明结构和生物活性特征最具有代表性的抗生素及抗癌药物，如聚酮类（红霉素、安丝菌素等）、C7N氨基环醇类（伏格列波糖等）、β-内酰胺类（头霉素、克拉维酸等）等的合成及调控机理，建立相关结构域/蛋白和合成调控的元件/模块库，完成生物合成途径中关键模块（起始合成单元、生物合成、后修饰、全局转录调控）的构建。  
2. 实现基因组水平上信息导向的药物生物合成途径和代谢网络设计；分析链霉菌功能底盘与引入的生物药物合成基因簇的互作，以设计外源元件遗传稳定的人工系统。  
3. 发展蛋白质模块重组的新理论与新技术，搭建高通量定向进化平台，实现生物元件、模块的设计和优化；提供2-3个可以将多个功能元件进行有序组装以提高整体反应效率的蛋白质纳米人造功能器件；构建体外药物合成多酶耦合体系。  
4. 完成小型基因组的设计与删减，建立最简化链霉菌底盘；硫化链霉菌底盘细胞，获得遗传稳定、抗噬菌体污染的人工体系，为天然产物的高效合成奠定基础；  
5. 建立新结构微生物药物的生物合成途径，建立人工合成体系多尺度优化的基本策略，实现10种以上微生物药物的高效制造。  
6. 在国内外核心刊物发表论文200篇以上, 其中SCI收录论文120篇以上;

申请或获得国内外发明专利40项以上。

# 三、研究方案

# （一）学术思路

本项目面向我国产业结构调整和国民健康水平提升的重大需求，拟选择医药行业中市场价值大、最具代表性的微生物药物为研究对象，开展微生物药物合成相关天然生物元件的挖掘、标准化与最优化，药物人工合成途径设计，体外多酶体系耦合等一系列基础工作；通过基因组最小化和引入特殊抗逆机制，构建微生物药物专用的、具有自主知识产权的链霉菌底盘生物；在前面基础上，将构建的药物分子人工合成体系与底盘细胞内进行整合，建立重要微生物药物的高效合成途径；通过对人工生物系统的多尺度调控和优化，实现微生物药物的高效生物制造，促进我国生物医药及相关产业的升级和发展。

# （二）技术途径

综合应用多学科先进技术方法，注重工程化及理性化的研究手段是合成生物学研究的精髓，因此本项目重视各个研究环节技术途径的先进性和协调性，以提高研究的高效性和准确性。

# 1、生物元件挖掘与人造功能器件的合成

设计系列分子探针，对深海极端环境以及近海红树林环境的微生物基因组和环境宏基因组文库进行优选，通过序列分析和功能注释深度挖掘海洋微生物中潜在的天然活性化合物生物合成基因，丰富生物制药合成生物学研究所需的功能元件和调控元件库；利用生物化学等手段建立体外反应体系，对生物功能元件进行分析、鉴定、分类以及标准化；

通过转基因技术，在拟采用的微生物药物底盘生物链霉菌FR-008中测试上述生物元件的性能，并依据底盘生物基因表达调控特点对元件进行标准化；针对底盘生物链霉菌建立高效遗传学方法，将功能元件在底盘生物中进行高效替换和组装；利用三级串联四级杆液质联用仪和高分辨质谱以及气质联用质谱对结果进行快速定性定量检测。

利用基因工程技术将功能元件加上接口，并与加有相应接口的蛋白纳米纤维和病毒蛋白骨架等人造功能元件进行共表达，利用蛋白纯化技术得到其组装单元，利用原子力显微镜和电子显微镜和对其装配情况进行观察和表征，并在体外建立快速酶活测定方法对新复合体进行检测，最终实现多个功能元件的合理化组装。

# 2、生物元件、路径和网络的模拟与设计

对基因组规模的微生物药物代谢网络的相关先导化合物、代谢反应、代谢酶、代谢途径、以及相关的基因等数据进行整合和数据挖掘，构建基于基因组规模的生物合成途径设计信息化平台和软件，提取生物合成所需的各种生物信息学组件，如基因序列组件、酶序列组件、反应模式组件等，设计合成反应、代谢物、代谢途径以及代谢酶。用代谢流平衡分析等方法模拟路径、网络的流分布状况，发现关键节点和模块，为合成生物学提供候选生物合成路径和代谢网络。

发展新型可极化分子力场等计算方法，建立基于机器学习的ProSAR驱动的蛋白质/酶定向进化技术平台和分子模拟技术，针对生物系统的网络结构进行理性改造与设计，通过改变生物元件（蛋白质）本身以及它们之间相互作用的特异性和强度，合理地控制生物通路的流向、流速，进而定向改造整个生物系统的动力学行为和功能。

以信息导向途径设计策略，构建链霉菌FR008基因组底盘的代谢调控网络，识别各种已知链霉菌应急反应的调控通路。构建携带整合酶基因、att位点、药物合成基因簇和phd-doc家族毒素-抗毒素基因座位的人工岛，特异性插入链霉菌FR008染色体，模拟复杂系统中底盘染色体对导入元件的调控机制。在多次传代和发酵过程，用real-timeqPCR定量测定岛的环出频率，比较整合酶、SOS阻遏因子LexA和毒素-抗毒素等相关蛋白质表达水平，建立底盘对外源元件的调控模型，设计元件遗传稳定的人工系统。

# 3、催化体系优化与人工组装

应用生物信息学方法寻找微生物药物合成中相关先导化合物及次级代谢产物中重要聚酮合酶（PKS）、非核糖体肽合成酶（NRPS）、纤维素酶、糖基转移酶等分子内部的模块属性，进行基于关键基序（Key motif）指导下的模块重组；建立不同家族酶的催化模型，依据底物特征与反应环境的要求，定向设计构建人工嵌合酶，完成功能上“跃迁式”进化；解析或者同源模建蛋白质生物元件的结构，进行分子对接、分子动力学和量子力学等方面的计算生物学研究，分析蛋白

- 蛋白、酶-底物之间的相互作用，建立蛋白质构-效关系，指导生物功能元件的理性设计与改造，建立生物功能元件的理性设计策略。针对非理性设计建立的功能元件突变库，建立基于流式细胞仪荧光激活细胞分选（FACS）、微流控芯片、遗传选择系统的超高通量筛选技术，获得性质优异的生物元件；体外构建多酶耦合体系，分析程序催化中的节点和限速元件，对多酶耦合体系进行优化设计。

# 4、微生物药物合成的放线菌底盘

选择自主优选的特色菌株、多烯大环内酯类抗生素杀念菌素产生菌灰色链霉菌FR-008作为药物底盘，通过种内基因组比较、同源序列比对、功能基因组分析来鉴定维持链霉菌生命活动的必需基因以及条件必需基因，利用连续大片段染色体敲除基本生理代谢中的非必需基因获得基因组最简化链霉菌；利用双亲本结合在FR-008中导入抗生素前体物合成相关基因，在没有冗余代谢途径干扰的情况下实现天然药物高效合成；发展DNA硫化修饰功能和调节元件的催化机理，利用高效基因点突变以及高通量串联四级杆液质联用仪检测，定位硫化系统中决定靶标序列、硫化丰度的功能元件和调控元件，采用理性设计的策略，从基因水平上对元件进行定向改造获得硫化元件资源库，并在底盘FR-008中人工导入优化设计后的硫化元件对链霉菌底盘FR-008的染色体进行可调控硫化修饰增强抗逆性，建立对天然药物合成优质化、遗传稳定的特色微生物底盘。

# 5、药物合成途径解析与优化

针对C7N氨基环醇类、聚酮类、β-内酰胺类药物，根据密码子偏好性等原则构建合成与调控的功能元件/模块；通过各催化蛋白的异源超量表达，并利用大肠杆菌无细胞体系建立体外酶促合成系统，或直接以产生菌建立静息细胞体系，通过前体物的添加和喂养考察主要限制因子；以催化反应为核心，通过小分子底物与生物催化分子（酶）的共价对接、蛋白分子对反应途径的调控模拟等手段，分析合成及后修饰过程的关键步骤，确定酶结构改变策略和影响因素降低/消除策略；利用多位点定点突变技术，优化酶的结构及其催化功能；以工业高产菌为基础，并根据催化反应影响因素降低/消除策略，有效引入与前体物和辅因子供给、抗性提高、产物外运、表达调控等相关的元件/模块，同时将优化基因替换原有拷贝，实现典型微生物药物的高产。

# 6、新结构代谢物的高效制造

应用发酵条件变换、启动子置换和生物信息学等手段发现生物功能分子的单

元合成的基因或基因簇，以及编码蛋白。应用途径杂合、基因敲除、模块置换、定向进化等手段使产生新的结构代谢物，建立细胞水平的活性筛选平台，发现活性先导化合物。将优化的突变体基因拼接到生物合成基因簇中，将整个基因簇整合到宿主染色体中，优化关键功能模块和模块之间的耦合过程，实现新结构代谢物定向、高效的生产。

# （三）创新点

# 1. 微生物药物特有生物元件库的建立

微生物药物合成相关的生物元件种类的匮乏和缺乏标准化是人工合成体系构建的限速步骤。将针对微生物药物生物合成过程最重要和最具有代表性的关键模块（起始合成单元、生物合成、后修饰、全局转录调控）分别进行挖掘、分类、定义，进而建立全面化、性能标准化的微生物药物生物元件库。本项目组在已有我国特有的、丰富的微生物药物产生菌基础上，将深入表征与药物代谢相关的催化及调控元件，使其标准化；还将对特殊生境丰富生物资源（深海放线菌资源及红树林放线菌菌株资源等）进行研究，为微生物药物生物合成提供了大量的具有优良性能的生物元件，为能建成功能全面化、性能标准化的生物元件库提供了重要保障。

# 2.体外人工代谢途径的构建

常规合成生物学研究仅强调在底盘生物体内构建合成代谢途径，但由于基因网络难以预测、元件不兼容、以及底盘生物自身噪音等不利影响，造成人工生物体系组装的成功率低，而且在取得一定进展后很难突破。本项目拟在体外进行多酶耦合体系的构建。由于避开了细胞内的复杂环境，体外多酶耦合体系可极大地

降低体系复杂度，为合理量化的建立体内多酶耦合体系提供量化的数据；同时由于体系相对简单，也可成为微生物产物的一种新型制备方式。

# 3.纳米级人工生物器件合成及生物元件串联绑定

人工代谢通路效率不高的原因之一是各个功能元件在体系中分散度过大，造成中间产物的流失和浓度的稀释而导致的。为突破这一限制，本项目拟应用蛋白纳米纤维和病毒颗粒包装蛋白构建新的通用型、纳米尺度、串联的“蛋白脚手架”，对人工合成途径中的催化元件进行有序“铆定”，以降低酶及底物在反应体系中的分散度，建立高效的催化系统。

# 4. 微生物药物生产用放线菌底盘生物的创建

常规合成生物学研究常使用大肠杆菌、酵母、假单胞菌等通用宿主作为底盘细胞，但由于这些细胞自身合成生物活性药物的种类非常有限，尤其是缺乏一些药物合成的辅助生物组件，导致即使导入药物合成基因，也无法完成药物生物合成途径得到最终产物。针对这种情况，本项目拟选择目前微生物药物生产中应用最广泛的放线菌作为操作平台，以我国自主筛选的特色链霉菌FR-008为研究对象进行人工合成体系的组装。该菌株本身具有生长快、产胞性能好、遗传系统通用简便、背景比较清楚等优良特点。所构建的药物高产菌株可直接与目前的微生物药物生产工艺衔接，有利于研究成果迅速向应用转化。

# 5.DNA硫修饰体系为底盘生物引入新型抗逆机制

DNA硫修饰系统是我国科学家在微生物代谢领域的原创性发现，在微生物药物底盘生物中人工构建“硫基”生命体系，有望赋予其天然生命所不具备的生命优势，为稳定、抗逆生命体的创建奠定遗传基础。本项目拟在对链霉菌底盘进行基因组最小化的基础上整合DNA硫化修饰体系，建立代谢速率高、抗逆性好的通用性微生物药物生产底盘细胞。

# 6. 多层次、多尺度改造微生物药物生产菌株

传统微生物药物菌株改良技术主要通过敲除、过表达等遗传操作技术对细胞代谢途径进行改造，以达到强化目标产品合成途径的目的，不仅效率较低，而且获得的改善程度也很有限。本项目将在充分理解微生物药物生物合成机理的基础上，综合利用元件优化、功能器件集成、途径设计、网络重构等不同层面对微生物药物生产体系进行系统优化，不仅可以提高微生物药物产量、降低生产成本、降低能耗及污染，并且还可以在现有药物基础上高效合成新的药物，为微生物药

物生产菌株的合成生物学改造提供新的技术支撑体系。

# 课题一、生物元件挖掘与人造功能器件的合成

# 1. 主要研究内容：

生物元件是人工合成体系的基本单位，它主要分为功能元件和调控元件两大类型。本课题将针对微生物药物生物合成过程最重要和最具有代表性的关键模块（起始合成单元、生物合成、后修饰、全局转录调控）分别进行挖掘、分类、定义，并在放线菌底盘生物（链霉菌FR-008）中进行标准化，建立药物合成催化与调控元件库和模块库；合成蛋白质骨架系统的人工器件，实现有序的生物元件组装体，为人工合成体系进行微生物药物的高效制造提供工具基础。

针对一些重要萜类药物（如紫杉醇等）的未知生物合成途径，从自然界寻找与其对应的功能元件并确定该功能元件的特定催化功能，为人工合成体系提供具有新型催化活性的功能元件库，以提高目前元件库催化能力的全面性；

针对生物药物生物合成过程中进行修饰并赋予其新活性的功能元件为对象，包括糖基化、甲基化、卤素化等后修饰功能元件进行挖掘，以及对功能进行鉴定，

在微生物药物底盘生物链霉菌FR-008中，对所收集的功能元件和调控元件进行标准化，建立微生物药物专用的催化与调控元件库和模块库。

# （2）人造功能器件的合成

建立基于蛋白纳米纤维和病毒蛋白骨架的功能元件“铆定”体系，研究多酶体系中各个功能元件之间的相互作用关系，并在此基础上优化多酶体系反应效率；通过对蛋白纳米纤维和病毒蛋白骨架等控制的研究，利用其自组装的有序特性和特殊的空间结构，构建功能化人工蛋白“脚手架”，结合上述多酶体系，实现对其中功能元件的逐级组装，比例组装。根据不同酶体系的作用原理对不同多酶体系按照设计好的程序进行组装，使其发挥最大功能。通过人造功能器件发展多酶反应体系逐级有序组装和功能集成的模式，获得功能完备的纳米级生物合成反应链。

# 课题二、生物元件、路径和网络的模拟与设计

# 1. 主要研究内容：

本课题应用计算生物学理论和方法，开展高模块性和通用性的调控元件设计与构建、代谢网络功能控制模块识别和设计工作，发展相关的计算生物学方法，为人工合成体系的理性设计提供指导，为实验科学奠定基础。

# （1）生物元件和功能模块设计与改造

# 课题五、药物合成途径解析与优化

# 1. 主要研究内容：

本课题以C7N氨基环醇类、聚酮类、β-内酰胺类典型微生物药物为研究对象，充分利用其基因组学和生物合成研究基础，展开合成生物学研究，系统分析其生物合成中的限速步骤及其相关的前体合成、关键生物催化功能大分子（酶）的构效关系，判定活性中心电子结构的分子基础和物理化学特性，进行定向设计和改造，并指导若干微生物药物工业产生菌的遗传育种和发酵工艺优化。

# （1）C7N环醇类药物合成途径解析与优化

包括井冈霉素等抗真菌抗生素和糖尿病治疗药物阿卡波糖等数十种微生物药物，其衍生产物伏格列波糖也是重要的糖尿病良药，而且其他多种衍生产物也是重要的糖苷水解酶抑制剂。C7N环醇类药物的合成起始于7-磷酸景天庚酮糖的环化，而且共享多步C7N结构单元的合成步骤，成为研究和应用合成生物学的良好材料。具体研究内容包括建立井冈霉素和阿卡波糖的体外酶促合成系统或静息细胞体系，高效地考察其合成过程的主要限制因子；同时开展以生物合成多酶体系催化反应为核心的量子力学和分子力学的描述以及计算机虚拟突变分析，并结合定点突变实验技术，从酶催化和酶结构的电子结构变化层面进行优化；结合井冈霉素产生菌的全基因组信息，构建以井冈霉素工业高产菌为基础的特定底盘产生菌，实现井冈霉素的高产和工艺优化，并通过阿卡波糖生物合成基因簇在井冈霉素高产菌株中的重构实现阿卡波糖的高产。

# （2）聚酮类药物合成途径解析与优化

抗感染红霉素、抗肿瘤安丝菌素、抗真菌纳他霉素、免疫抑制剂他克莫司等I型聚酮和阿克拉霉素等II型聚酮类药物种类繁多（包括多烯、聚醚、安莎类、聚酮-多肽类、四环素类等），生物活性多样（抗细菌、抗真菌、抗肿瘤、免疫抑制剂等），其聚酮链的生物合成由多功能多肽或多酶复合物催化形成，同时具有多种聚酮合成后修饰，这是生物活性和结构多样性的基础。研究内容包括：

构建聚酮合酶各结构域或催化蛋白的功能元件和催化模块，同时建立聚酮的体外酶促合成系统或静息细胞体系，高效考察各功能元件和催化模块的功能；通过计算生物学和分子生物信息学分析聚酮合成过程中的各个关键步骤，在计算机中重现聚酮合成中与碳链增长相关的生物大分子（酶）几何结构和电子结构变化的合成生物学分子基础；分析聚酮合成各个后修饰反应的特征，寻找影响催化效率的关键因子，结合定点突变实现重要酶的定向人工改造，优化聚酮类药物的产生效率和优化发酵工艺。

# (3)  $\beta$ -内酰胺类药物合成途径解析与优化

由棒状链霉菌产生的头霉素和克拉维酸是重要的抗感染药物，本质上讲它们是结构相对简单的非核糖体肽类药物，其生物合成是由多结构域组成的非核糖体肽合酶催化完成的，同时也具有多种多肽合成后修饰反应。研究内容包括：构建肽合酶各结构域或后修饰蛋白的功能元件和催化模块，同时建立  $\beta$ -内酰胺类药物的体外酶促合成系统或静息细胞体系，高效考察各功能元件和催化模块的功能；通过分析多肽合成过程中的关键步骤，在原子分子水平深入揭示多肽生物合成中前体的生物化学活化的分子机制；通过蛋白结构模拟和分子对接，分析多肽合成各个后修饰反应的特征，寻找控制催化效率和活化位点的作用因子，结合定点突变实验实现酶的定向改造，优化  $\beta$ -内酰胺类药物的生物合成和发酵工艺。

# （4）生物合成途径的调控机制

根据重要微生物药物生产菌的全基因组序列，利用Microarray技术和蛋白组学分析，结合生物信息学分析，发掘和鉴定影响药物生物合成途径的主要调控元件；通过不同阻断突变株遗传互补来确定调控基因间的相互关系，分析各突变株基因表达强度、蛋白与mRNA稳定性，阐明这些调控元件的结构与功能；通过S1核酸酶作图分析或采用EMSA、DNaseI footprinting确定其调控的下游靶基因及顺式作用元件，同时分析调控因子与靶基因启动子结合情况以及在微生物药物合成过程中的体内结合强度变化，揭示这些调控因子对微生物药物次级代谢过程的调控机制。

# （5）合成与调控网络的模拟与重建

次级代谢网络的模拟和重构分为静态和动态两个层次。静态分析采用流平衡分析、代谢调节最小化等多种物质流网络分析方法，而动态分析采用广义代谢流分析法模拟实际发酵过程。具体的生物信息计算将根据每个反应的物质流动和

化学计量系数，列出所有代谢物的质量守恒方程，形成常微分方程组。通过实验测定部分物质转化流速度，定义另一些物质转化流的约束条件（如最大代谢能力等），解出平衡状态下各种物质相互转化流动的状态。在不同的理论假设下，通过线性规划（如流平衡分析）或二次规划（如代谢调节最小化分析）等不同的流网络优化方法，最优化特定次级代谢网络的改造方案。通过比较模型预测与实际观察，反复优化理论代谢流模型，形成基于实验一理论一计算模型关联机制的合成与调控网络逐步重构流程，最终实现微生物药物生物合成途径的工程化设计。

# 课题六、新结构代谢物的高效制造

# 1. 主要研究内容

本课题将组装新的人工合成体系，并对其进行优化，为合成生物学在药物开发领域中的系统化研究建立基础。对获得的新的人工合成体系，通过其对环境应答机理的研究，对表达新型结构代谢物的生物体系进行系统优化并实现理性放大。

# （1）后修饰模块的替换与导入

后修饰基因对于抗生素的生物活性具有重要意义。在抗生素中，多烯类化

合物的后修饰过程独具特色，因此，计划以杀念菌素为多烯类化合物的代表，进行如下研究：

1) 组合不同来源于不同抗生素基因簇的糖基合成酶及糖基转移酶基因导入自身糖基转移酶缺失的杀念菌素产生菌中，获得携带新型糖基的衍生杀念菌素；  
2) 将负责多烯抗生素羟基化/环氧化的P450氧化酶导入杀念菌素产生菌中, 对杀念菌素的内环羟基化/环氧化, 以获得溶解性优化的杀念菌素衍生物;  
3) 将醛基生成P450单氧化酶导入脱羧杀念菌素产生菌中以获得侧链醛化杀念菌素衍生物；  
4) 将来源不同的起始单位合成酶导入杀念菌素自身起始单位基因缺失的突变株中，形成新型起始单位的杀念菌素衍生物。

# （2）生物合成模块的人工改造

从聚酮延伸单元数目的角度看，安莎霉素不像其它大环内酯那样连续，尚未发现6酮、10酮、12酮和13酮等类型的安莎霉素，为人工合成新安莎霉素骨架，拟进行如下研究：

1) 新颖安莎霉素全成基因基因簇的克隆和异源表达, 激活 “隐性” 安莎基因簇的表达, 为创造新骨架安莎霉素提供可用的功能元件。  
(2) 采取下述方法产生新骨架类型的安莎霉素:

改变I型PKS途径：进行启动子和loading domain的替换，同时敲除PKS基因簇原有的TE domain，避免线性安莎霉素以大环内酯形式被释放，获得安莎霉素基因簇嵌合体。通过将不同来源的酰胺合酶（安莎霉素合成途径中的链长决定因子）定向进化，并引入安莎基因簇嵌合体突变株中，获得新骨架安莎霉素。

增加后修饰基因：将不同安莎霉素基因簇来源的后修饰基因如糖基转移酶asml25等在安莎基因簇嵌合体突变株中组成型表达，获得新安莎霉素类化合物。

# （3）生物合成模块的组装及优化

二硫吡咯酮类化合物具有吡咯并二硫环戊烯的双环内酯核心骨架，成员间的差异主要体现在N7位取代基上。已有研究表明，不同羧酸诱导可产生N7位相应酰基单元的多种二硫吡咯酮化合物。本研究拟以二硫吡咯酮类化合物为例，进行如下研究，深入揭示模块组装与优化过程中的生物合成与调控机制：

1) 前期已成功克隆到二硫吡咯酮家族中Holomycin分子的生物合成基因簇，在此基础上确定并表征其中负责吡咯并二硫环戊烯结构单元合成的基因以及编

码蛋白。

2) 通过定向进化筛选具有更宽泛底物选择性的酰基转移酶基因。  
3) 优化前两步所获取的功能元件编码基因的密码子，更换启动子，拼接并整合到宿主的染色体中。  
4) 将现有的各种羧酸单元合成器件经优化后转入上一步的合成体系中，实现新型二硫吡咯酮化合物定向、高效的生产。

# （4）全局转录调控模块作用机制

前期工作中从阿维菌素生产菌株出发，通过突变全局转录调控因子  $\sigma^{\mathrm{HrdB}}$  得到产量提高  $46\%$  的突变菌株，在此基础上拟进行如下研究：

1) 解析全基因组范围内的  $\sigma^{\mathrm{HrdB}}$  结合谱：用 ChIP-seq 技术对转录因子  $\sigma^{\mathrm{HrdB}}$  在 DNA 上的结合位点进行全基因组的筛查，获得  $\sigma^{\mathrm{HrdB}}$  在全基因组范围内的结合谱，将其中基因注释到 GO 中，划归不同的复合功能类，找到可能受  $\sigma^{\mathrm{HrdB}}$  调控的生物功能模块。  
2)  $\sigma^{\mathrm{HrdB}}$  结合谱中基因转录水平比较与分析：找出在高产菌株及出发菌株中表达差异显著的模块，即与阿维菌素高产相关的关键生物功能模块。  
3) 人工生物合成体系的构建与多尺度优化: 根据 (2) 的结果, 在异源表达阿维菌素生物合成基因簇的重组菌株中相应调整关键生物功能模块的转录水平,结合后修饰模块与起始合成单元的替换, 获得新结构阿维菌素类化合物。对具有不同功能模块组合的工程菌进行在基因转录、蛋白质表达、中间代谢物三个尺度的分析, 将功能模块与宿主细胞原有代谢整合, 实现人工生物合成体系的多尺度优化。
