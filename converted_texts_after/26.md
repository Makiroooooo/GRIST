# 1、项目的立项依据（附主要的参考文献目录）。

研究意义

随着宽带综合数字网络的发展，传统的电信网、数据通信网和计算机网正在逐步融为一体，一方面为网络的广泛应用带来巨大的前景，另一方面也为网络带宽等资源分配和拥塞控制等提出了许多新的研究课题。在高速宽带网络中，定义了具有不同传输优先度的CBR、rt-VBR、nrt-VBR、ABR和UBR等业务，在基于统计复用和包交换的环境下，高效率地利用网络的资源和保证实时业务的传输服务质量（QoS）往往是一对矛盾，在大型高速网络的环境下，如何对穿越多个网络系统的网络资源进行有效的调配，还有许多尚待解决的问题，

计算机互联网本身是由多个网络或多个自治系统构成一个复杂的大系统，网络各节点或各自治系统间存在复杂的相互影响和制约关系，尤其是引入QoS要求之后，要求高效率地利用网络资源使得系统的性能分析和控制十分复杂，以至基于整个系统的严格分析事实上不可能实现。目前，在互联网中把划分成一个个自治系统作分布式的网络管理，在ATM论坛的PNNI协议中把一个大型的网络分解成若干层、若干个组的选路方法，实际已经无形中引入了大系统的处理方法。本研究将从大系统分析的理论高度对高速网络业务管理和流量控制进行研究，寻求对网络资源调度和分配的最优化。目前本研究所提出有关分析和研究方法在国内外还鲜有报道，本课题的研究可望对大型高速网络的分布式动态管理提出有一定指导意义的理论结果，对高速网络技术的完善、发展和实用化具有十分现实的意义。

# 2. 目前的研究现状

（1）对多种业务在网络中同时传输时的带宽的分配、所需的缓冲的大小等有关的研究往往局限于个别结点，在端到端传输过程中各网络结点或自治系统间应如何协调等有关研究还非常有限；  
(2) 每一种业务的传输特性往往孤立地进行研究，例如研究对ABR业务对反馈控制时没有或很少考虑信道可用带宽随其它更高级别业务所需带宽变化的影响；  
（3）目前主要的研究方法和结果大都来自排队论，一般只能得到稳态解或数值解难以反映网络瞬态特性；  
（4）实时业务，尤其是交互式的实时业务的传输要求在保证传输带宽的同时确定传输的延时上限，目前可对延时上限作定量分析的所谓“确定型（deterministic）”法难以分析信道的带宽利用率。另外，降低包实时调度运算的复杂性还有待研究。  
(5) 目前, 互联网的应用范围不断扩大, 其标准化组织 IETF 相应地制定了不少支

持实时业务传输的协议或协议草案，如IntServ和DiffServ等有关标准。这些协议往往只是一些类似信令系统结构的规范，具体如何高效率地实现还有待人们深入研究。对通信网络的流量控制的研究国内外有两大课题：一是集中于单个端到端的传输控制问题，另一个则是着眼全局的性能分析。前一问题的研究已有许多成果发表，在复杂的大型计算机网络中已经很难获得进一步大的突破。而进行后一方面研究的已发表的成果还很少，而专门针对大型网络的研究就更少。通过对近五年来的国内外主要刊物的检索，专门针对大型通信网络的流量传输问题的研究只有两篇文献，一是希腊的Logothetis等人的研究，二是上海交大韩兵等人所做的工作。前者在将分散控制的思想用于VP带宽的优化分配，而后者则用分散控制方法对ATM网VP资源调度问题作了研究。与严格的最优解相比，它们的最大误差均在  $20\%$  以上。这体现了简单的分散优化方法的局限性及寻求更精确方法的必要性。

# 2、项目的研究内容、研究目标，以及拟解决的关键问题。

# 研究内容：

(1) 根据高速网络各交换节点或自治系统间对流量相互制约和分散控制的特点，采用动态大系统中有关分散控制的方法，确定各节点间带宽等资源分配和流量控制的有关参数和调节机制，解决包括带反馈控制的多节点网络中系统的稳定性及传输性能综合最优等问题，确定优化后系统端到端传输的有关延时上限、信元丢失率等性能参数。由于对大型计算机网络不可能实现集中控制，因此关键是如何将现有大系统理论中分散控制可靠性高，简单和局部控制效果好的优点数据流量等参数的动态控制，将大系统理论中的多目标优化方法应用到高速网络的分析研究之

中，在保证用户服务的 QoS 的基础上寻求获得最佳网络性能的方法。

(2) 根据网络业务统计上具有自相似性、VBR 业务带宽变化具有一定相关性等特性，在现有的线性和非线性预测理论研究的基础上，分析研究业务流变化的特点及有关的预测方法，解决在保证实时业务特点的延时上限、包出错及丢失概率等前提下的网络资源合理分配和最佳利用等问题，其中关键是如何根据不同的信元模型及特性合理选择预测方法及分析预测误差，寻找能保证业务传输 QoS 的调度策略。  
(3) 系统地将大系统理论应用于大型计算机网络带宽资源协调分配的研究，以获取精确的最优解；提出一种快速的步进式子网带宽分配算法。  
(4) 网络性能参数的高精度快速估算。这是保证流量控制算法是否具有实时性涉及的一个重要问题  
(5) 对骨干网络节点上的业务流的合成规律进行研究

# 研究目标：

(1) 采用大系统理论分析大型高速网络性能，并得到一种基于分散控制和多目标决策及优化的方法；  
(2) 研究高速网络中多种业务混合传输时的有关特性, 采用线性和非线性预测理论的方法对 VBR 业务带宽的变化特性进行预测, 寻求一种较小实现代价的满足各类业务预定 QoS 要求的分布式网络资源动态管理和分配的方法。

# 拟解决的关键问题：

(1) 利用大系统理论的分析方法，对大规模网络进行层次分割，在分布管理的环境下，采用分解协调的方法，对包括虚通路的建立，网络带宽动态分配和流量控制等进行计算，实现对大型网络资源的综合优化分配。  
(2) 将多目标优化算法应用于高速网络的业务管理和流量控制, 以保证在网络中多种业务并存的环境下, 在保证实时业务传输的同时, 使网络的资源获得高效率的利用。  
(3) 研究基于 VBR 业务带宽预测的带宽分配算法及高效率的保证服务质量的包调度算法。

# 3、拟采取的研究方案及可行性分析。

# 研究方法：

将大系统分析的基本原理和现代信号处理理论用于高速网络多种业务的管理、业务量变换预测、流量控制和性能分析，并对有关网络流量和带宽调节控制的算法、包调度策略等进行计算机模拟。

# 技术路线：

(1) 将动态大系统理论中的分散控制方法及多目标优化法应用于高速网络的分析, 遵循集结法和分级分解法的思路, 对大型高速网络进行大系统建模, 实现对复杂计算机网络系统的分解协调控制。在此基础上作进一步的分析与计算机仿真研究, 寻求全局意义上的最佳网络资源配置与传输控制策略, 将其结果用于高速网络设计、规划与优化。  
(2) 通过资料的搜集与计算机仿真研究，分析比较当前诸多的 VBR 业务模型，计划尝试采用包括传统的线形预测算法、非线形的如人工神经网络、模糊逻辑等预测算法等多种不同的角度，建立 VBR 业务带宽预测模型，决定最佳的多种业务下带宽等资源动态分配和包调度策略的算法。

可行性分析：

目前大系统控制理论已被成功地应用于大型电力、交通和能源等传输控制系统，并取得显著经济效益。计算机网络与此类系统有相似之处，从原理上来说，将同样的研究分析方法用于高速网络业务流的管理与控制是完全可行的。另外，现代信号处理理论中各种线性与非线性的预测分析方法，已经广泛用于各种智能控制和动态调节的场合，将大系统控制理论与现代信号处理的方法有机地结合于高速网络的性能分析与资源的动态分配，可望获得新的有实际意义的成果。

# 4、本项目的特色与创新之处。

(1) 将大系统控制理论引入复杂多节点和多自治系统计算机网络系统的控制和性能分析, 采用多目标决策的方法动态地对网络进行管理和控制, 以求获得最佳的网络综合性能。  
(2) 综合分析和研究在高速网络中多种业务并存时的行为，在对各业务流变化预测的基础上，确定在各节点和自治系统上根据对网络性能要求的定量调节控制的实现方法。

# 5、年度研究计划及预期研究结果。

2004年1月～2004年12月根据大系统理论的基本原理研究适用高速网络分布式管理和控制的分解协调模型，研究和寻找有效的在给定目标下的分解协调和分布式算法。研究各种网络业务模型的统计和动态变化特性，应用各种线性和非线性的预测理论，研究网络流量动态预测的方法；

2005年1月～2005年12月在有关基于大系统的分解协调模型和网络流量预测分析的基础上，研究带宽网络资源动态分配的单目标和多目标优化控制的方

法，对理论分析结果进行模拟仿真，结合实际的模型和环境，检验分析的正确性、有效性和实用性。并在此基础上改进有关的算法。

2006年1月～2006年12月总结、论文撰写和成果验收。

预期研究成果：

提出一种根据大系统理论对高速计算机网络分解协调的分布式控制的方法，以及单目标和多目标决策的算法对网络资源进行动态分配，以期获得综合最佳的网络综合性能。

根据切合实际的流量模型，获得对高速计算机网络在综合业务传输的条件下业务流特性动态变化预测的方法和高效率的包调度策略。

上述研究成果将主要以论文形式发表。预期在国内外重要学术刊物发表论文20篇以上，其中三大索引不少于5篇。此外还将开发出一套数值计算程序包。
