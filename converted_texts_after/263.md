项目名称： 机械装备再制造的基础科学问题

起止年限：2011.11-2016.8

依托部门：教育部

# 一、关键科学问题及研究内容

# 1. 拟解决的关键科学问题包括：

针对目前严重制约我国机械装备再制造产业化进程的深层次理论与技术基础难题，本项目以再制造对象可再制造性评价、再制造毛坯键离/键合“形、性”调控、再制造产品服役安全等关键共性技术为突破口，围绕下列三个关键科学问题展开研究。

# 科学问题一、再制造对象跨尺度损伤演变规律及可再制造性评价理论

根据再制造对象的损伤程度及其剩余寿命等多方面因素，对可再制造性问题做出科学合理的评价是产品再制造基础研究中的首要问题。再制造对象在长期服役过程中形成磨损、腐蚀、疲劳等各类跨尺度损伤，其损伤程度的高低决定了再制造对象剩余寿命的长短以及是否可以再制造，并进一步对再制造成形过程的效率、成本和质量产生直接影响，同时也会对再制造后产品的寿命和服役安全性产生重要影响。但再制造对象在服役过程中受多场耦合（力场、热场、材料组织场等）作用，损伤行为具有复杂性、多样性及不确定性特点，疲劳、磨损与腐蚀损伤行为与损伤结果具有多尺度性、强关联性和高随机性，这给可再制造性评价带来了极大难度。如何准确把握再制造对象服役环境和多场作用对跨尺度损伤的影响规律？如何对损伤状态进行定量化物理表征？如何基于损伤程度确定再制造对象剩余寿命，进而确定可再制造的临界损伤阈值？这些问题必须给出准确、系统性的回答。

# 科学问题二、再制造毛坯的键离/键合“形、性”调控基础

对再制造对象及毛坯的键离/键合过程进行准确形、性调控是再制造基础科学研究中的核心问题。再制造成形加工需要解决两大关键问题：其一，再制造毛坯在复杂服役环境、外场载荷的作用下其表面和亚表面往往形成纳米级、微米级甚至更深的污染层、变性层等，为满足再制造键合过程要求，首先需结合再制造毛坯表层和亚表层各种缺陷类型及其特点实施污染层解离、键离，变性层去除及表面功能恢复。但目前对污染层、变性层的形成和结合机制、解离和键离/键合机制仍缺乏深入研究。其二，再制造毛坯键合过程中，二元多相异质材料结合形成新的具有复杂表/界面的亚稳材料体系结构，热、力等复杂外场作用易于造成表/界面各种宏微观缺陷，直接影响再制造产品的服役安全性。

因此，阐明多场条件下污染层、变性层的形成、演化机理，研究污染层的解离、键离和变性层的去除、改善机制，揭示再制造键合/嵌合过程中材料与能量的相互作用机理，建立再制造键离/键合过程的控形、控性基础理论及实现方法，是再制造基础理论研究必须解决的核心问题。

# 科学问题三：再制造产品的服役安全与再制造过程的综合决策

再制造基础研究中的关键问题是再制造产品的寿命预测与服役安全保证。但相比新品，二元多相亚稳材料体系下的再制造产品宏微观特性更加复杂，表面高成分复杂性、高结构紊乱性、高自由能及多界面性导致再制造产品寿命演变呈现高度复杂性与非线性，其寿命评估及服役安全性预测更为困难。

其中，需要对再制造零件表/界面结构在服役条件下的行为进行研究，从而揭示二元多相材料结构的性能演化规律；同时，必须掌握亚稳材料体系高自由能与多界面性关联竞争机制，才能揭示再制造产品的寿命演变规律，保证其服役安全性。在突破以上理论和技术瓶颈基础上，进一步发展再制造决策支持理论和评价方法。

# 2. 主要研究内容包括：

# 1）再制造对象的多强场、跨尺度损伤行为与机理，可再制造的临界阈值

针对再制造对象损伤问题的物理学本质，研究多场作用下跨尺度损伤的萌生及演化规律；揭示外场作用、跨尺度损伤、多物理参量之间的映射关系；建立再制造对象损伤定量化表征体系；探索再制造对象损伤行为和状态与其剩余寿命的关联机制，确定再制造对象损伤临界阈值。

主要研究内容包括：

□□  $\bullet$  跨尺度疲劳损伤的微观物理机制  
□□●多强场作用下的跨尺度磨损、腐蚀损伤行为与机理  
□□● 再制造对象多物理场多尺度损伤演化模型  
□□● 多类型损伤间的损伤映射规律  
□□  $\bullet$  损伤容限及可再制造性评判机理

# 2）再制造毛坯的键离/解离原理与性能调控

研究基于物理、化学复合作用的再制造毛坯污染层的绿色解离、键离机理；提出多场多介质耦合作用下的复合约束解除新原理新方法；阐明再制造毛坯变性层对键合作用的排斥机制及其在外场诱导作用下的性能恢复机理；研究变性层去除过程对再制造毛坯表面/亚表面宏微观特性影响规律；探索再制造毛坯微裂纹在外场作用下的逆止方法。

主要研究内容包括：

□□● 再制造毛坯的键离/解离机制及实现  
□□● 再制造毛坯表面变性层的形成机制与改善、消除机理

□□● 再制造毛坯微裂纹的逆止机理

# 3）再制造毛坯的键合/嵌合机理与实现

针对再制造毛坯键合/嵌合加工过程中的形、性准确调控需求，研究再制造毛坯成形加工中的异质、异相材料键合/嵌合机理及实现方法；揭示键合、嵌合过程中异质材料的微观行为及其成分和性能的变迁机制；针对键合/嵌合成形层非光滑表面特点，研究精密低应力加工新原理、新方法。

主要研究内容包括：

异质材料间的原子键合成形及控性理论与方法  
异质材料间的机械嵌合成形及控性理论与方法  
□□●键合/嵌合成形层的精密低应力加工理论与方法

# 4）再制造零件的表面/界面行为与机理

针对再制造零件二元多相亚稳材料体系下，表/界面物态、组织结构在外场作用下的物理化学性质变迁、性能演化机制的复杂性问题，需要研究表/界面的服役匹配行为，揭示表面性能对摩擦匹配适用性及化学相容性的作用机理，阐明表/界面结构和能量的交互作用机制及表/界面性能演化机制，掌握极端工况下表/界面抵抗外加载荷侵蚀破坏的作用规律，提出适于评价再制造零件表/界面行为的新原理、新方法。

主要研究内容包括：

□□● 再制造零件表面/界面的亚稳态结构特征  
□ ● 再制造零件表面的服役匹配行为、机理及设计  
再制造零件表面的润滑与自修复作用行为与机理  
再制造零件表面的极端工况服役行为与机理  
$\bullet$  面向再制造全过程的材料与结构演变仿真建模

# 5）再制造零件的寿命预测与再制造产品的服役安全验证

针对具有二元多相材料体系特性的再制造零件在高自由能与多界面性的关联竞争作用下，其寿命演变呈现高度复杂性与非线性，以及再制造产品受多强场作用耦合损伤，其服役安全缺乏可靠性保障的重大难题，需要研究高自由能与多界面性“关联竞争机制”对再制造零件寿命演变的影响规律，并进一步揭示再制造产品在复杂服役环境下个体零件随机损伤间的耦合作用机制。同时基于实验力学和数学分析方法，实现再制造产品寿命演变及服役安全预测与验证也是再制造领域无法回避的理论与技术基础问题。

主要研究内容包括：

□●高自由能与多界面性竞争作用下的再制造零件寿命

□● 再制造零件的服役动态健康监测  
□  $\bullet$  再制造零件的寿命演变机理及寿命预测数学模型  
再制造产品的服役安全预测与验证

# 6）再制造过程的决策支持与综合评价理论

以空气压缩机转子等核心部件为研究对象，针对再制造过程数据缺失，建立基于预估的再制造全生命周期评价体系；剖析再制造对象更换、回收及配送体系优化机理及制约因素并建立决策与评价模型；研究不确定环境下再制造单件少批量订制式生产系统运行理论与方法；构建产品非均衡寿命分配模型，揭示主动再制造时机临界点选择机制，提出主动再制造设计方法，发展再制造决策支持理论和方法体系。

主要研究内容包括：

□  $\bullet$  面向再制造的全生命周期评价体系  
□●再制造对象更换、回收及配送体系决策支持与综合评价  
□  $\bullet$  不确定环境下再制造单件少批量订制式生产系统运行理论  
$\bullet$  面向再制造的设计基础

# 二、预期目标

# 本项目的总体目标：

本项目以大型机械装备的核心部件为研究对象，深入研究再制造领域中涉及机械、材料、力学、物理、化学及数学等多学科交叉的重大关键基础科学问题，创建可再制造性评价理论，创立异质材料原子间的键合新原理，阐明再制造零件的竞争性寿命机制，构建面向再制造的决策支持基础理论与综合评价模型，形成再制造的完整理论体系。通过本项目研究，使我国机械装备再制造基础理论研究达到世界先进水平，提供亟需的核心再制造技术，形成国际一流的机械装备再制造学术带头人和中青年研究队伍；同时为国家再制造领域法规、政策和标准的制定提供科学依据。

# 五年预期目标：

# (1) 在理论方面

解决机械装备再制造中的关键科学问题，为再制造技术与装备的快速发展提供理论基础，跻身国际再制造科学研究领域的前沿。

□  $\bullet$  揭示多强场耦合作用下再制造对象跨尺度损伤机理和演变规律，阐明再制造对象损伤与多物理参量间的映射关系，创建可再制造性评价理论。  
□  $\bullet$  揭示再制造毛坯变性层的性能恢复机制和异质材料间的绿色键离机理，阐明异质材料原子间的键合、嵌合及扩散行为与作用机制，建立再制造键离/键合高效控形控性理论。  
□  $\bullet$  揭示再制造零件表面/界面结构与性能演化规律，构建特定属性表面/界面行为评价与设计的新理论与新方法；创建面向再制造全过程的材料与结构演变数学模型；揭示高自由能与多界面性关联竞争机制，建立再制造产品的寿命预测模型及服役安全性评价理论。  
□ ●创建再制造决策支持基础理论与综合评价模型，提出主动再制造设计方法，建立再制造对象更换、回收和配送优化模型，揭示面向再制造的批量化生产协同机制，构建面向再制造的全生命周期评价体系。

# (2) 在技术方面

通过本项目研究，在再制造的若干关键技术上取得源头创新成果，提供亟需的核心再制造技术，并在相关企业进行验证。

# 主要关键技术指标如下：

实现再制造对象表面与内部微缺陷、微损伤的多参量集成无损检测。缺陷检测的最小当量尺寸小于100微米，应力集中区域的检出率达到  $90\%$  。  
□ ● 研发键离过程与键离物质绿色化与无害化的复合清洗技术及其相应装备，在满足清洗表面质量要求的前提下，相比传统清洗技术，提高资源利用率，减少综合污染排放：实现节水  $50\%$  以上，节能  $10\%$  以上，污染物及有害物质回收率达  $95\%$  以上。  
实现再制造毛坯高效率、高质量地键合/嵌合。针对易氧化、易变形、高碳含量等难修复零件，研究开发变极性等离子熔覆、光纤激光熔覆等技术与工艺装备，实现键合区孔隙率低于  $5\%$  。  
□ ● 突破再制造零件多类型损伤的在线监测技术与服役能力的综合评估技术，创建再制造零件的服役动态健康监测系统和再制造产品（如重载车辆的再制造发动机）服役安全验证平台，通过对声、光、电、磁等弱物理信号的准确提取与处理，实现对再制造产品服役寿命的高置信度预测，置信度不低于  $90\%$  。  
- 再制造后的大型空气压缩机转子符合API标准，电跳值达到  $6.3 \mu \mathrm{m}$  以下，机械跳动值小于  $20 \mu \mathrm{m}$ ；转速  $20000 \mathrm{rpm}$  时，高速平衡偏移量小于  $0.3 \mathrm{~mm}$ ；典型转子（装配  $100 \mathrm{~kg}$  叶轮5部，主轴重量  $800 \mathrm{~kg}$ ，工作转速  $20000 \mathrm{rpm}$ ）低速平衡偏移量小于  $0.41 \mathrm{~mm}$ 。  
□ ● 开发面向再制造全过程的材料与结构演变仿真系统，构建及开发面向再制造的全生命周期评估工具及其支撑数据库，实现对再制造产品的全生命周期清单分析；再制造系统全工艺过程的耗能量化计算及碳轨迹量化描述。

# 三、研究方案

# 1. 总体研究思路和技术路线及可行性

# 1）学术思路

结合我国装备制造业发展的重大需求，围绕机械装备再制造面临的重大挑战，确定关键科学问题，开展六个方面的课题研究。

本项目形成理论方面的源头创新成果包括：再制造对象的可再制造性临界损伤阈值的确定；再制造的绿色键离机理与键合过程控形控性理论；再制造表/界面效应与产品寿命演变规律；再制造零件的竞争性寿命预测、再制造产品的服役安全预测理论；包括更换、回收与配送体系，主动设计，批量化协同生产等。

# 2）技术途径

本项目技术途径。结合再制造技术的研究前沿和再制造行业的发展需要与趋势，从机械装备再制造面临的科学挑战出发，凝练关键科学问题，采取理论探索、科学实验与数值计算相结合的方式，集中高等学校与再制造重点企业各自的优势，进行再制造基础理论的研究，实现理论技术成果与工程应用示范的紧密融合，指导再制造的工程生产实践。

# 3）可行性分析

近年来，我国机械制造业面临着两个日益突出的矛盾：原材料与一次能源的大量需求与供给紧张之间的矛盾；废旧车辆、机械装备等的大量更新淘汰与处理技术的缓慢发展之间的矛盾。机械装备的再制造为上述矛盾提供了高效率、低成本的绿色解决方案，既减少对自然资源的高度依赖又能够利用我国丰富的再制造资源。我国已在中长期科技发展规划中提出建立循环经济的技术发展模式，近期国务院及各部委更是将再制造提升为战略性新兴产业，明确要求加快再制造技术与产业发展水平。因此，本项目的研究高度符合了国家、行业的战略需求与发展方向，具有显著的需求牵引性。

本项目参与单位在再制造领域开展了大量前瞻性的探索，并在某些方面取得了原创性成果，对重型车辆发动机、曲轴等关键零部件的再制造，如喷涂技术、涂层材料、专用设备等，进行了深入研究并部分实现了工程应用。相关企业是我国最大的压缩机生产基地之一，在大型压缩机核心部件的修复与再制造方面进行了深入的研究和实践。相关企业是我国规模生产的再制造重点企业，已成为国内再制造科研实验基地。

本项目集中了国内致力于机械装备再制造研究的七所重点高等学校，以及相关企业。参加单位长期从事表面工程、再制造工程、材料加工以及绿色设计与制造等方向的基础及应用研究，科研起点高。参与单位依托装备再制造技术国防科技重点实验室、工业装备与结构分析国家重点实验室、摩擦学国家重点实验室、精密测试技术与仪器国家重点实验室等，具有良好的研究平台与支撑条件。研究队伍由中国工程院院士、千人计划特聘教授、国家杰出青年基金获得者及优秀中青年骨干组成，为项目的深入开展提供了必要的人才保障。

# 2. 创新点

# 1）学术思想和立项构思特色

当前再制造模式主要有两种：国外再制造模式和中国自主创新的再制造模式。本项目立足国际前沿，探索研究中国自主创新的再制造模式，将先进的无损检测理论与技术、表面工程理论与技术和熔覆成形理论与技术引入再制造，与国外再制造模式相比，中国自主创新的再制造模式不仅能对表面较轻度损伤的零件进行再制造，还能对表面重度损伤及三维体积损伤的零件进行再制造；不仅能修复零件损伤部位的尺寸超差，而且能全面恢复零件整体结构、性能；不仅可提高旧件再制造率，而且能充分挖掘再制造的重大节能减排潜能。

本项目围绕中国自主创新的再制造模式拟开展的一系列研究工作，在学术思想和立项构思上体现了显著的创新性。

针对具有二元、多相材料体系特性的再制造零件在高自由能与多界面性的“关联竞争作用”下，其寿命演变呈现高度复杂性与非线性，以及再制造产品受多强场作用耦合损伤，其服役安全缺乏可靠性保障的重大难题，研究在高自由能与多界面性“关联竞争作用”下再制造零件的竞争性寿命演变规律；构建基于多信息融合的智能传感单元对再制造零件服役全程监控的动态健康监测体系；阐述局部添料对再制造零件动力学特性等固有服役特性的影响机理，揭示再制造产品在复杂服役环境下个体零件随机损伤间的耦合作用机制；建立基于实验力学和数学分析的再制造零件服役寿命预测模型，实现再制造产品安全服役。

研究内容：

# 1）高自由能与多界面性竞争作用下的再制造零件寿命

研究高自由能对再制造零件磨损、腐蚀寿命的增强行为，揭示多相材料体系中增强相在抵抗材料去除及无机离子溶解扩散腐蚀的作用机理及增强相与基质相间的协同效应；研究多界面性对再制造零件疲劳寿命的削弱行为，揭示疲劳微裂纹在界面处的形核萌生、扩展连接的作用规律，以及疲劳剥落的失效机理；研究高自由能与多界面性在服役过程中的结构演化与寿命竞争行为，揭示其“关联竞争”的作用条件及作用机制。

# 2）再制造零件的服役动态健康监测

研究新型智能传感单元的设计原理与实现方法，智能传感单元感知末梢的使役原理；研究智能传感单元在再制造零件二元多相复杂材料体系中的嵌入与匹配机制，揭示智能传感单元对再制造零件动态微损伤特征参量的定域捕捉，实现复杂服役工况下再制造零件中微弱损伤信息的实时、准确传输与提取；研究基于多信息融合与多智能传感单元集成的智能传感器件对再制造零件多物理信息损伤参量的实时连续的辨识与采信，揭示再制造零件动态微损伤信息与多物理参量间的映射规律，实现再制造零件的服役动态健康监测。

# 3）再制造零件的寿命演变机理及寿命预测数学模型

研究再制造零件的服役寿命演变行为，揭示导致再制造零件寿命衰退的犁削、粘着、微动、无机离子渗透、交变应力等的作用规律及破坏机制；研究基于实验力学的累积损伤统计与回归，以及基于时间序列与多元分析的数学计算，揭示再制造零件寿命演变的数学内涵，建立再制造零件服役寿命预测的数学模型；研究加速破坏条件下再制造零件的寿命演变与衰退行为，揭示加速因子促进再制造零件寿命衰退的作用规律，建立基于加速试验条件的再制造零件服役寿命预测数学模型。研发可模拟再制造零件典型工况的寿命试验机，为再制造零件的寿命预测

及加速寿命预测提供实验平台。

# 4）再制造产品的服役安全预测与验证

研究再制造产品的非线性动力学行为，阐述局部添料对再制造零件动力学特性等固有服役特性的影响机理，揭示再制造零件之间失效的相关性规律，建立基于再制造零件个体寿命模型的再制造产品服役寿命预测模型；构建基于电、磁、声、热多物理场信号提取的，具有综合判据集成分析性能的，可进行在线监测、离线检测功能的大型智能考核台架，以完成典型再制造产品服役全过程损伤动态提取与寿命演变实时评判，实现再制造产品的服役安全验证；基于大型智能考核台架，对比验证基于数学预测的再制造产品服役寿命与基于台架考核的再制造产品服役寿命。

# 课题6：再制造过程的决策支持与综合评价理论

# 研究目标：

针对再制造中的回收过程、生产模式及材料结构演变等不确定性问题，建立面向再制造的全生命周期评价体系；剖析再制造对象更换、回收与配送体系中利益相关者合作机理以及制约因素并建立基于地理信息系统的决策与评价模型；研究不确定环境下实时工况驱动的计划和调度控制方法，物料流控制方法和质量控制方法，建立再制造单件少批量订制式生产系统运行理论体系，并进一步研发实时工况驱动的再制造执行系统；针对再制造毛坯状况的不确定性，建立产品设计信息与服役特性之间的映射模型和产品非均衡寿命分配模型，揭示主动再制造时机临界点选择机制。为实现不确定环境下再制造产品的能源消耗评估、材料与工艺选择、回收模式制定、再制造生产以及产品主动再制造等难题提供解决方案。研究内容：

# 1）面向再制造的全生命周期评价体系

建立再制造成形及加工典型工艺能耗模型，提出再制造产品全生命周期碳排放量化计算方法及其低碳运行优化方法；基于再制造工艺过程的物质流与能量流分析，建立全生命周期评价系统边界模型；提出基于清单分析的再制造产品数据多生命周期分配方法，研究再制造过程缺失数据的预估及其敏感性分析方法，建立再制造与新品制造过程生命周期评价及比较模型；以再制造成形技术、材料和工艺的绿色筛选为目标，提出再制造产品全生命周期多目标多因素优化理论与方法；研究建立适用于我国国情的再制造产品全生命周期评估工具及其支撑数据库。

# 2）再制造对象更换、回收与配送体系决策与评价

建立基于地理信息系统的再制造对象更换、回收及配送系统，提出基于再制造对象产生与分布不完全信息的决策评价模型与方法；针对再制造对象更换与回收的不确定性，运用DEMATEL方法识别制约再制造对象更换、回收及配送的主要因素和关键因素；分析再制造对象产生、回收与配送过程中利益主体的多样性和复杂性，建立多阶段博弈模型和演化博弈模型，进一步剖析各利益相关者的影响与合作机理并建立激励机制；面向资源节约、环境友好和经济可行的多目标，研究再制造对象回收与配送的网点布局优化。

# 3）不确定环境下再制造单件少批量订制式生产系统运行理论

研究再制造单件少批量订制式生产系统运行机制与实时工况间时滞效应产生机理，建立以降低时滞效应为目标的基于实时工况的变权式动态调度与决策方法，构建再制造生产系统模型，实现再制造单件少批量订制式生产系统的仿真；研究再制造系统车间物料流瓶颈漂移机理及预测方法，建立基于瓶颈的生产组织、

计划、调度和物料流控制机制；研究再制造系统产品质量误差流形成规律，建立不确定环境下的再制造系统产品质量控制方法；在上述理论研究基础上，研发实时工况驱动的再制造执行系统。

# 4）面向再制造的设计基础

基于再制造零件表/界面组织结构与性能演化规律，建立产品设计信息与服役特性之间的映射模型；研究零件的结构功能梯度、强度冗余与服役寿命规律，提出面向再制造的结构梯度功能设计和强度冗余设计方法；研究机械装备不同零部件之间的非线性耦合影响行为，揭示产品不同零部件退化的相互竞争机制；建立产品性能升级预测模型和非均衡寿命分配模型，提出基于再制造的非均衡寿命匹配设计方法；基于产品服役行为、失效机理及其寿命演变规律，研究产品主动再制造时机临界点决策机制，应用多目标多因素优化理论方法，建立产品主动再制造时机临界点决策模型，构建主动再制造设计理论与方法

# 4，课题间关系

围绕机械装备再制造的三个关键科学问题，共设路六个研究课题。课题设路思路参见图3。针对科学问题一，设置课题1，研究多强场耦合作用下再制造对象跨尺度损伤行为及演变规律，揭示再制造对象损伤与多物理参量间的映射关系，确定可再制造的临界阈值。针对科学问题二，设置课题2和课题3，研究服役条件下再制造对象表层与污染层的结合机理以及变性层形成、去除和改善机制，提出基于复合物理-化学效应的绿色键离/解离方法，建立再制造毛坯键合/嵌合高效控形控性基础理论。针对科学问题三，设置课题4、课题5和课题6，研究再制造零件服役过程中的表/界面行为及其整体稳定性，揭示再制造零件寿命演变机理，为实现再制造产品的安全服役提供基础理论和技术依据。在以上几个课题研究和应用验证基础上，提出再制造的基础决策支持理论和综合设计、评价方法。通过上述六个课题研究，解决三个科学问题，建立机械装备再制造的基础理论体系，在关键技术方面取得源头创新成果，为机械装备再制造提供理论基础和技术支持。

课题之间的关系是：课题1是为可再制造的实施提供评判依据，因此为后续五个课题提供实践研究的基础。题2和课题3则是在课题1可行的前提下完成再制造的实际实施过程，从而为后续课题4和课题5提供理论研究的保障。课题4和课题5是对前面三个研究课题所完成的再制造零件和再制造产品的综合寿命进行量化研究和科学验证，课题6则是为了上述五个课题提供基于全生命周期的理论决策支持和最终能够完善机械装备再制造的理论体系。
