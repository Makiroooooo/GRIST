# 耐热芽孢杆菌HS08高温中性蛋白酶及其特性研究

# 1、项目研究意义

蛋白酶是一种重要的工业化应用的酶制剂，其生产几乎占酶制剂市场的  $65\%$  以上[1]，已大量应用于食品、洗涤、纺织、皮革、医药、化工等行业，而细菌来源的蛋白酶又是蛋白酶的重要来源之一，如在洗涤待业中细菌蛋白酶占到总微生物蛋白酶生产量的  $35\%$  [2]。蛋白酶按其作用的pH分为酸性蛋白酶、碱性蛋白酶和中性蛋白酶。近年来对酸性蛋白酶、碱性蛋白酶，特别是一些高产菌株产碱性蛋白酶的研究已十分广泛[2-5]，也最早被工业化应用，尤其是在洗涤行业中。由于通常条件下微生物产生的蛋白酶不耐热，因而限制了其应用于高温行业，但在许多应用中均需要高温条件如高温洗涤、高温纺织加工、食品高温蒸煮、高温发酵以防杂菌污染等，因此研究高温蛋白酶很有必要。近年来发现来自于耐高温的微生物所产生的蛋白酶具有较高的耐热性与热稳定性而引起了众多研究者的关注，其中研究最多的是耐高温碱性蛋白酶。中性蛋白酶由于其作用条件温和，虽也被应用于食品、洗涤、毛纺、丝纺、制药、化妆品、皮革加工等行业，但是由于大部分中性蛋白酶的热稳定性相对较差，生产与使用条件复杂，因而限制了其应用范围，同时国内虽有部分报道[6-8]对高温中性蛋白酶的研究，但也仅限于对某些来源的产高温中性蛋白酶微生物的分离与酶的初步特性进行了少量研究，而对其结构特点、作用机理等没有深入，因此限制了其应用。本研究拟从嗜高温微生物（嗜热芽孢杆菌，Thermophilic Bacillus）中分离得到产耐高温中性蛋白酶菌株，并主要对其生化特性与水解机理进行系统研究，期望为耐高温中性蛋白酶的应用提供理论基础。

# 2、项目研究目标及与申请者研究工作长期目标的关系；

通过该项目的研究，期望分离纯化得到高温中性蛋白酶，研究得出该酶的最适作用条件，并对其分子结构与水解机理进行探讨，长期目标是在对该酶理论研究基础上，生产应用该酶于食品、洗涤、皮革、制药等待行业，提高这些行业的蛋白酶应用效率。

# 3、项目研究内容，研究方案和进度安排；

# 研究内容：

首先从优化培养的高温菌株中分离得到耐高温蛋白酶，然后研究高温蛋白酶的分离纯化方法，得到经过高效液相色谱、电泳等鉴定的纯高温蛋白酶，再对该蛋白酶包括分子量、最适作用条件、氨基酸组成与一级结构、不同底物米氏方程等在内的生化特性进行研究以确定该酶特性，最后探讨该酶的水解蛋白质方式与作用机理。

# 研究方案：

(1) 产高温中性蛋白酶菌株的培养: 利用项目组实验室已分离得到的产高温中性蛋白酶的耐热芽孢杆菌, 通过斜面培养 (培养基组成主要为牛肉膏、蛋白胨、氯化钠、琼脂等)、发酵培养 (培养基组成主要为玉米粉、豆粕粉、葡萄糖、磷酸氢二钠、碳酸钠、硫酸镁、氯化钙等)于  $60^{\circ} \mathrm{C}$  下  $12 \sim 36 \mathrm{~h}$  后得含高温中性蛋白酶的发酵液。  
(2) 高温蛋白酶的分离纯化: 将经高温培养的菌液冷冻离心去菌体, 硫酸铵分级沉淀后透析脱盐得粗酶液, 粗酶液经 DEAE 离子交换 NaCl 梯度洗脱得有酶活性组分, 此组分再经 Sepharose、Sephadex、Sephacryl S 等不同型号的凝胶过滤进一步纯化, 得到电泳单一色带的蛋白酶。  
(3) 高温蛋白酶生化特性研究: 将该酶在不同温度、 $\mathrm{pH}$  等条件下反应测定酶促反应速率以确定最适温度与  $\mathrm{pH}$ , 同时试验不同类型抑制剂条件下酶促反应速率以确定该蛋白酶的类别; 经 SDS-PAGE 电泳与凝胶过滤测定蛋白酶的分子量, 经盐酸水解后高效液相色谱测定其氨基酸组成; 经等电聚焦测定该酶的等电点; 采用蛋白序列分析方法确定该酶的氨基末端序列并同现有的蛋白酶氨基末端序列进行同源性比较; 通过不同肽或蛋白为底物的酶促反应测定该酶的米氏方程常数。

（4）高温蛋白酶水解机理研究：已知序列的人工合成肽（如pro-BPN’I、Amylins等）经该蛋白酶水解  $0.5\sim 4\mathrm{h}$  后，产物用HPLC/MS联用仪测定小肽混合物的分子量分布，从分子量分布情况推断该酶水解原肽的氨基酸位点，经多个不同肽反复试验来了解该酶水解蛋白质的作用位点与作用机理。

# 进度安排：

2006.01~2006.10 微生物培养、酶的分离纯化工艺优化

2006.11~2007.05 酶的生化特性研究

2007.06~2007.11 酶的作用机理研究

2007.12 材料总结、项目结题

# 4、项目创新之处；

蛋白酶因应用广泛而研究者较多，近年来因从耐高温菌中分离得到耐高温蛋白酶能应用于高温条件而在国际上倍受关注，国内也开始研究。但由于洗涤行业应用蛋白酶占酶制剂应用一大部分，故绝大部分研究都集中于耐高温的碱性蛋白酶的分离纯化与特性研究上，而对中性条件下有高活性的高温蛋白酶却研究不多，国内仅有的几篇报道也仅限于对其生产菌的培养、未纯化酶的最适条件研究上，而对于进一步的生化特性、作用机理却未涉及。而蛋白酶的生化特性、作用机理又是蛋白酶应用的基础，因此该研究在高温中性蛋白酶的分离纯化、生化特性与作用机理方面的研究具有创新性，也有很大的应用前景。

# 5、工作基础与工作条件；

研究小组成员中有长期从事蛋白质分离纯化方面的研究人员，也有从事微生物分离培养方面的研究人员，因此项目组有能力承担该项目研究。同时，项目主要成员已开始对高温微生物产高温蛋白酶进行初步研究，筛选得到具有产高温中性蛋白酶活性的菌株，现正在进一步研究。项目组所在的实验室拥有AKATA Purifier100蛋白层析仪、Waters高效液相色谱仪、氨基酸分析仪、冷冻离心机、冷冻干燥机、离心浓缩仪、无菌操

作室、喷雾干燥仪、凝胶成像仪等仪器设备，基本能满足该项目研究所需。

# 6、预期研究结果及其利用研究结果的计划和今后发展的思路；

通过本项目研究预期能分离纯化得到一种耐高温中性蛋白酶并获知该酶的基本生化特性与最适作用条件及其水解蛋白质的作用方式，所有研究结果将以论文和最后的项目总结报告形式表达。

在本项目研究结束后，计划通过两种方式继续发展与开发本项目研究。其一是继续在理论上研究该耐高温中性蛋白酶的酶活性中心氨基酸组成、结构域与高级结构，以更完整了解该酶；其二是对该耐高温中性蛋白酶在食品、洗涤、皮革行业中的应用进行研究开发，争取开发多种适用于不同行业应用的产品。
