# （一）立项依据与研究内容（4000-8000字）：

# 1、项目的立项依据

近年来磁性纳米线引起了人们的注意。纳米线直径一般为  $10\sim 300$  纳米，处于磁性材料的单畴尺寸范围内，因此具有单畴颗粒才具有的一些特性，例如高矫顽力等。由于纳米线具有天然的形状各向异性，在纳米线有序阵列中，其易磁化方向一般沿着纳米线垂直于膜面，具有良好的矩形比。在我们制备的纳米线阵列中典型的纳米线间距为60纳米，如果每一个纳米线可以存储一个比特的信息，它的理想存储密度可以达到200Gbit/in²，是目前所用软盘的20万倍。因此，磁性纳米线阵列在超高密度垂直磁记录和传感器中将具有重要的应用价值。另一方面，磁性纳米线的研究具有重要的科学意义，例如它能够被考虑为研究低维磁结构的相互作用，磁化和反磁化机制，以及单畴颗粒系统磁性的模型系统。

氧化物磁性材料由于具有优良的磁性，电性和化学稳定性而得到广泛的应用。例如， $\mathrm{CoFe_2O_4}$  和  $\mathrm{BaFe_{12}O_{19}}$  是很好的磁存储介质。最近发现的氧化物特大磁电阻效应和氧化物稀磁半导体的磁电效应，有望在磁电子和光电子装置中得到重要应用。人们对过渡金属及其合金纳米线已经进行了大量的研究，但是对磁性氧化物纳米线的研究很少，目前国际上对氧化物纳米线磁性的研究处于起步阶段，有很多问题还没有解决或有待深入研究。为此我们提出对氧化物纳米线的制备和磁性的研究。我们已经成功制备了 $\mathrm{ZnO,CoFe_2O_4}$  和  $\mathrm{CoO / Co}$  纳米线阵列，并对其结构和磁性进行了初步的测量，观察到一些新的现象。

# 2、项目的研究内容、研究目标，以及拟解决的关键问题。

# 研究内容

(1) 高质量氧化物纳米线样品的制备及纳米线的参数（直径，间距及长度）对磁性的影响。  
(2) 磁存储介质  $\mathrm{CoFe}_{2} \mathrm{O}_{4}$  和  $\mathrm{BaFe}_{12} \mathrm{O}_{19}$  纳米线的磁性及其机制。  
(3) 铁磁元素掺杂的氧化物半导体  $\mathrm{ZnO}$  纳米线的磁性及其机制  
(4)  $\mathrm{Co} / \mathrm{CoO}$  纳米线的交换各向异性。

# 研究目标

本课题将围绕磁性氧化物纳米线的制备和磁性机制，进行系统的实验研究和理论探讨，并力求获得具有应用前景的新型磁性纳米线材料。

# 拟解决的关键问题

(1) 高质量样品的制备。为此必须对现有的设备进行改进, 同时进行系统的实验, 以获得最佳制备工艺条件。  
(2) 氧化物纳米线磁性的准确测量。为此将采用多种分析手段如 VSM, SQUID, FMR, NMR 和 Mössbauer 谱等, 结合 XRD, HTEM, Ramman 光谱等测量手段进行详尽细致的研究。

# 3.拟采用的研究方案及可行性分析。

# (1) 样品制备

利用电化学阳极氧化方法制备不同孔径, 不同孔心间距的纳米孔洞氧化铝模板。利用电镀结合高温氧化法, 溶胶凝胶法和 CVD 方法制备氧化物磁性纳米线。通过改变制备条件, 寻求得到高质量氧化物纳米线的最佳工艺条件。我们已经成功地制备了  $\mathrm{ZnO}, \mathrm{CoFe}_2\mathrm{O}_4, \mathrm{Co} / \mathrm{CoO}$  纳米线阵列, 并初步分析了制备条件对氧化物纳米线的结构及磁性的影响。

# (2) 结构分析

利用X射线衍射分析样品的晶体结构和晶格常数，利用高分辨透射电子显微镜分析样品的微结构，研究氧化物纳米线的生长规律及其对磁

性的影响。对氧化物半导体材料，利用晶格常数随掺杂量的变化及微结构分析结果，结合拉曼光谱研究掺杂离子在样品中的位置及其状态。

(3) 磁性测量

a. 测量铁磁金属 / 反铁磁氧化物纳米线的磁滞回线, 得出交换偏置场及矩形比, 研究交换各向异性的规律及其机制。  
b. 测量氧化物纳米线的磁化强度与温度及磁场的关系，可以得到自发磁化强度 Ms 和居里温度 Tc。从 Ms 的数据可算得分子磁矩及磁性离子在四面体座和八面体座中的分布。结合铁磁共振谱及 Mössbauer 谱分析结果，研究氧化物纳米线中磁性离子的磁矩及其相互作用规律，从而了解氧化物纳米线的磁性机制。  
c. 测量不同方向磁场下矫顽力随磁场及温度的变化关系, 研究氧化物纳米线的磁的各向异性的规律, 及磁化和反磁化过程的机制。

# 3、本项目的特色与创新之处。

目前对纳米线阵列的研究多限于金属及合金，对氧化物纳米线磁性的研究极少，氧化物磁性材料的磁性来源及其规律不同于金属及合金，氧化物纳米线的磁性规律知之甚少，因此本课题是一个创新性的工作。

# 4、年度研究计划及预期研究结果。

研究计划

2005年，对现有的纳米线制备方法进行改进，研究制备条件对纳

米线生长的影响，探索出最佳工艺条件，制备出高质量的氧化物纳米线。分析样品的结构及其随制备工艺变化的规律。

2006年，全面测试样品的磁性，测量样品的磁化强度与温度的关系曲线，在不同的温度下测量样品的磁化曲线，磁滞回线，铁磁共振谱和Mössbauer谱，得到样品的磁性随温度，磁场及纳米线阵列参数（孔径，孔心间距及纳米线长度）的变化规律。

2007年，总结分析实验结果，得出氧化物纳米线磁性的规律，提出理论模型进行计算，对其磁性机理进行解释，对有突出性能的样品进行重点研究，探索其物理机制和应用前景。

# 预期成果

1) 制得高质量的氧化物纳米线, 给出最佳制备工艺以及晶体生长的规律。  
2) 得到过渡金属氧化物/过渡金属纳米线的交换耦合规律。  
3) 得到氧化物纳米线的磁性与温度，磁场及制备工艺条件的变化规律，对样品的磁性，磁化及反磁化机理的规律做出解释。  
4) 得到氧化物稀磁半导体磁性的规律及其与铁磁元素掺杂量和制备工艺的关系，对铁磁性的来源做出解释。

所有成果将以论文形式提交，在国内外核心期刊发表论文8到10篇。

# （二）研究基础与工作条件

# 1、研究基础

(1) 多年来, 我们已经对具有尖晶石结构, 石榴石结构, 磁铅石结构

和钙钛矿结构的单晶，多晶，薄膜及非晶态铁氧体材料进行了比较系统的研究。利用X射线衍射，电子显微镜观察，磁性测量，NMR及FMR技术以及红外光谱，拉曼光谱，磁光谱，Mössbauer谱研究了氧化物磁性材料的结构，磁性，磁光性能及巨磁电阻效应，取得了很多新的具有科学意义和应用前景的结果。在国内外著名刊物上发表了论文多篇，论文在大型国际学术会议上做口头报告，多篇论文被国内外杂志多次引用。

(2) 最近几年，本研究小组开展了金属及合金纳米线阵列的研究。在国内外比较早地利用有序模板制备了矩形比达 0.92 的过渡金属及其合金的纳米线阵列。纳米线长度达 100 微米，长度与孔径比大于 1000，纳米线的密度达到 200G 每平方英寸。研究了它们的结构和磁性，在国内外核心刊物和国际会议上发表论文多篇，申请国家发明专利多项，合作者具有丰富的制备纳米线的经验。

# 2、工作条件

(1) 现有自制的氧化铝模板以及纳米线生长的设备, 为了制备氧化物纳米线需要一些改进。  
(2) 具有美国进口的振动样品磁强计 (VSM), 温度从  $4.2 \mathrm{~K}$  到  $1100 \mathrm{~K}$ ,可以进行变温磁性测量。具有美国进口的 SQUID 可以进行微弱磁信号测量。  
(3) 本校具有 X 射线衍射仪，高分辨电子显微镜，微区拉曼谱仪，FMR，NMR，化学分析等多种先进设备可供本课题使用。
