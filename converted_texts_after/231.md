项目名称：我国富铁矿形成机制与预测研究

起止年限：2012.1-2016.8

# 一、关键科学问题及研究内容

# 1. 关键科学问题

为实现富矿成矿理论的创新和找矿预测的突破，本项目拟解决的关键科学问题为：铁的巨量聚集、超常富集与富铁矿形成机制。具体包括：

（1）大规模岩浆作用、地幔柱活动、岩石圈改造与铁质巨量供应；  
(2) 早前寒武纪缺氧环境、地球大氧化事件、变质热液叠加改造与沉积变质型富铁矿的形成;  
(3) 富铁熔体/流体的形成、演化与铁质超常富集成矿

# 2. 主要研究内容

瞄准国家急需的大型富铁矿，立足4种重要铁矿床类型（沉积变质型、矽卡岩型、火山岩型和岩浆型），针对关键科学问题，选择大型矿集区及其典型矿床为重点解剖对象，开展如下主要研究。

(1) 巨量铁聚集成矿与构造-热事件特别是重大地质事件的耦合关系

围绕不同地质历史时期一些重要构造-热事件特别是重大地质事件对巨量铁聚集成矿的控制作用，开展以下研究：太古宙-古元古代大规模幔源岩浆活动、大氧化事件以及区域变质作用与沉积变质型富铁矿形成和分布；二叠纪地幔柱基性-超基性岩浆作用与岩浆型钒钛磁铁矿成矿；古生代洋陆转换过程中海底火山作用与海相火山岩型铁矿的形成；中生代岩石圈改造背景下大规模岩浆活动与矽卡岩型和陆相火山岩型铁矿的形成。

(2) 富铁成矿流体/熔体的形成和演化机制

早前寒武纪缺氧环境下海洋中巨量铁的溶解、富集与搬运机理；富铁岩浆及铁矿浆的形成条件与形成机制；膏盐层、流体相分离与高盐度富铁岩浆热液的形成。

(3) 铁质快速卸载与巨量堆积成矿的机理

流体中铁沉淀的物理化学条件与铁的快速卸载机制；海底喷流作用与铁富集

成矿；钠钙质蚀变、退化蚀变与富铁矿的形成；岩浆熔离、分离结晶与铁质堆积成矿。

（4）后期地质作用的叠加改造与富铁矿的形成

区域变形变质、混合岩化、热液活动及构造叠加对沉积变质型磁铁富矿的贡献以及关键控制因素；沉积变质型赤铁富矿的形成环境和形成机制；高温热液蚀变对海相火山岩型富铁矿的贡献。

（5）成矿模式、找矿模型及找矿预测

通过典型矿床和矿集区的解剖研究，总结主要类型富铁矿成矿规律，建立成矿模式；总结关键找矿标志，建立低缓重磁异常评价和矿致弱信息提取指标体系，建立典型矿集区尺度和大比例尺地质-地球物理找矿模型，选择若干典型矿床，开展三维立体找矿评价示范研究；圈定富铁矿找矿靶区。

# 二、预期目标

# 1．总体目标

通过对沉积变质型、矽卡岩型、火山岩型和岩浆型铁矿典型富铁矿床与矿集区的系统研究，探讨不同地质历史时期和不同地质背景中铁矿富集机理，特别是大型富铁矿的形成条件和形成过程，创新和完善富铁矿成矿理论；建立低缓重磁异常提取和弱信息评价的理论模型以及大比例尺定位预测模型，开展靶区圈定和深部矿体定位预测研究，推动快速发现一批富铁矿床。

# （1）发展和完善富铁矿成矿理论

以大型富铁矿为主要对象，研究主要构造-热事件，特别是重大地质事件对铁大规模聚集成矿的控制作用，阐明特定地质环境和地质单元中铁巨量堆积的关键因素，揭示岩浆-热液系统、变质变形过程、深部流体交代过程中铁元素的运移和超常富集机制，建立矿集区尺度和典型矿床尺度的成矿模式，发展和完善大型富铁矿成矿理论，特别是具有我国特色的沉积变质型磁铁富铁矿成矿理论，为实现大型富铁矿的找矿突破提供理论指导。

# （2）完善铁矿预测方法体系

在成矿理论研究成果的基础上，应用综合信息成矿预测理论与方法，发展低缓重磁异常提取和弱信息评价技术，甄别矿致异常，建立不同类型铁矿、不同尺度的地质-地球物理找矿模型，完善铁矿找矿评价体系；开展大比例尺定位预测和三维立体找矿示范研究，圈定靶区。提供找矿靶区5～10处，并与地勘部门、矿山企业合作进行勘查验证，为实现大型富铁矿的找矿突破提供技术支撑。

# （3）培养一支高水平铁矿研究队伍

通过本项目研究，将形成一支高水平的铁矿研究队伍，培养活跃于国际铁矿研究领域的矿种专家，造就一批优秀的中青年学术带头人。培养研究生和博士后40～50名。

(4) 发表  $100 \sim 120$  篇高质量学术论文（其中SCI收录论文不少于60篇）。  
（5）通过项目实施，逐步将项目第一承担单位打造成在国内外具有重要影响的铁矿资源研究中心。通过对典型大型铁矿床的解剖研究，形成一批铁矿科学

研究基地。

预期研究时间5年（2012年1月～2016年12月）。项目设6个课题（见后），其研究目标各有侧重，但执行期间将围绕项目总目标协同进行。五年工作安排如下。

工作内容：全面收集、分析、处理已有资料；全面开展野外地质调查；开展岩石、矿石样品鉴定、分析、测试。

年度目标：选取有望取得突破的重点工作区，根据已有资料，初步建立主要铁矿产地数据库、航空重磁数据库；在富铁矿成矿机制研究的某些方面取得初步认识；发表论文  $5\sim 10$  篇。

# (2）2013年

工作内容：继续开展野外地质调查研究，并在已知矿区开展深部矿体定位的地球物理找矿试验；开展成岩和成矿时代精测、元素和同位素地球化学分析测试以及综合研究；通过国际合作与交流，系统开展国内外铁矿主要类型的对比研究。

年度目标：完成部分岩石和矿石的高精度年龄精测；初步确定沉积变质型铁矿床、矽卡岩型铁矿床、火山岩型铁矿床和岩浆型铁矿床的成矿时空结构；初步确定成矿事件与主要地质事件的关系；初步形成航空重磁低缓异常找矿评价的理论基础；发表论文  $10\sim 20$  篇。

# （3）2014年

工作内容：深入开展重点工作区野外研究、地质样品分析测试；针对重大科学问题开展国际合作研究。

年度目标：研究沉积变质型富铁矿床的主要特点、时空结构和分布规律，与国外类似矿床开展对比，初步提出富矿形成的基本要素和控制条件；初步提出深部找矿定位的地质-地球物理找矿模型和技术方法体系；发表论文25～30篇。培养研究生和博士后5～7人。

# （4）2015年

工作内容：补充重点工作区的野外研究工作；系统分析、处理获得的资料和数据；对新提出的问题进行系统分析和深入研究；开展国际合作与交流。

年度目标：初步厘定四类铁矿的成矿背景，总结成矿规律，建立部分典型矿床尺度和大型矿集区尺度的成矿模式；初步提出铁超常富集形成富矿的成因机制；完善隐伏大矿综合信息找矿模型和深部铁矿体定位的理论和技术方法，进行靶区圈定；推动形成2～3个铁矿研究基地；发表论文30～40篇；培养研究生和博士后20～25人。

# （5）2016年

工作内容：补充、完善各项研究工作；综合分析研究，撰写课题和项目结题报告；完成结题验收。

年度目标：加强综合研究，进一步完善富铁矿成矿理论和两个尺度的成矿模式以及大型矿集区和深部矿体定位两类找矿模型和技术方法的集成；有机结合成矿理论和找矿技术方法研制，选择典型矿床开展立体找矿示范研究，圈定  $5\sim 10$  个找矿靶区；推动形成  $2\sim 3$  个铁矿研究基地；发表论文  $30\sim 40$  篇；培养研究生和博士后  $15\sim 20$  人。

# 三、研究方案

# 1.学术思路

选择我国重要类型铁矿典型矿集区和典型富铁矿床，开展成矿地质背景、铁矿形成机制和成矿规律研究，建立典型矿床成矿模式和区域成矿模式，创新铁矿成矿理论；总结典型矿床找矿标志和区域找矿标志，并分别与大比例尺地球物理地球化学信息和中小比例尺重磁信息有机结合，分别建立大比例尺定位预测模型和中小比例尺区域找矿模型，丰富完善找矿预测理论；圈定找矿靶区，以产业部门和生产单位为依托，推动靶区验证。

# 2. 技术途径

为了高质量完成项目设计的研究内容，在研究中将充分应用相关学科的最新研究成果，采用一系列新技术、新方法，通过学科交叉、多种手段的运用和实施，实现项目设计的研究目标。

# (1) 消化资料、理清思路

对国内外有关铁矿的资料进行全面收集，充分利用已经建立的全国铁矿产地数据库和全国铁矿储量库中的信息，充分收集航磁数据库。通过资料的二次开发、整理、分析，并结合国内外对比研究，进一步梳理思路，提出一系列工作模型。

# (2) 选择重点、解剖典型

选择重点成矿区带和大型矿集区开展研究，解剖典型矿床。

# (3) 野外调查、夯实基础

全面扎实地开展野外地质调查，充分收集扎实的野外第一手资料。在野外工作过程中坚持五个结合：地表与井下坑道观察结合，矿体与相关地质体观察结合，自身观察与前人工作结合，面上展开与重点解剖结合，宏观地质现象与微观组构特征观察相结合。针对工作内容，系统采集样品，确保样品新鲜和具有代表性。

# (4) 先进方法、精确测试

在传统测试方法基础上，运用SHRIMP锆石U-Pb、Re-Os和Ar-Ar等最新的高精度测年技术和Sr-Nd-Hf同位素示踪技术，全面厘定不同类型铁矿床的时空分布规律，探讨成矿地质事件和地质背景。结合铁、硅、铜等最新非传统稳定同位素测定技术，探讨成岩成矿物质的来源和成矿机理。开展熔体包裹体、流体包裹体研究，探讨成矿的物理化学条件和成矿过程。

# (5) 综合研究、建立模式

在上述野外调研和精确测试的基础上，开展综合研究，建立典型矿床成矿模式，归纳总结区域成矿模式。

# (6) 综合信息、集成创新

全面开发工作区已有1：20万、1：5万、1：2.5万航磁资料和重力资料，结合成矿理论研究，开展找矿靶区圈定。根据矿床模式，有效集成地球物理示矿信息和矿田构造研究成果，在重点成矿区带及典型矿床开展立体找矿示范研究。提出和发展两个尺度的找矿模型和集成技术方法。运用GIS技术在MAPGIS平台上开展综合信息集成研究，提高工作效率和精度。

# （7）优选目标、推动验证

对圈定的找矿靶区进行优选；与产业部门密切配合，进行靶区验证。

# 3．创新性

# （1）针对我国特色的磁铁富矿，建立和发展富铁矿成矿理论

针对我国多期次构造-岩浆-流体叠加改造成矿的复杂性以及沉积变质型磁铁富矿的特殊性，深入研究不同类型富铁矿的成矿机制，发展具有我国特色的富铁矿成矿理论。

# （2）建立不同类型铁矿三维立体地质-地球物理找矿模型

基于航空、地面和井中不同尺度地球物理测量，开展低缓重磁异常评价和弱信息矿致异常识别，建立不同类型铁矿三维立体地质-地球物理找矿模型。

# 4. 与国内外同类研究相比的特色

(1) 本项目针对国家大宗矿产——铁，专攻最紧缺的富铁矿，有别于以往以特定成矿区带为研究对象的973项目。

（2）国外大型-超大型富铁矿主要形成于稳定的克拉通环境，且以赤铁矿石为主。本项目针对我国陆壳演化的多期性和复杂性背景下形成的沉积变质型磁铁富矿，开展成矿机制深入研究，推动富铁矿成矿理论创新和找矿重大突破。  
（3）以富铁矿成矿模式为基础，集成航空、地面和井中三维地球物理信息和地质信息，针对隐伏矿床开展低缓异常识别和弱信息提取，建立不同类型铁矿的地质-地球物理立体找矿模型。

# 5.取得重大突破的可行性分析

# （1）关键科学问题明确、研究思路清晰

如前所述，这项研究是面向国家需求，针对富铁矿找矿勘查中出现的关键科学技术问题开展工作，而且这些问题在全球层面具有特殊性，是一项立项起点较高的前沿性工作。这是本项目能够取得重大突破的立论基础。理论研究与找矿预测技术方法相结合，建立找矿模型，研究思路清晰。

# (2) 技术路线先进可行

为实现项目的研究目标，根据目前科技发展水平，项目设计了科学合理的技术路线。其中，锆石U-Pb原位离子探针等高精度同位素测年、非传统（铁、硅、铜、锌）稳定同位素示踪均是当前国际上矿床研究最先进的技术手段。基于我国现有的海量航空重磁数据的再开发利用以及多种技术的综合信息研究，结合成矿理论的创新，可能达到推动快速发现铁矿的目标。项目所选择的其它主要技术途径，都有成功的研究实例可供借鉴。因此，整体技术路线先进可行。

# （3）具备良好的工作基础和平台

项目申请单位在上世纪70年代铁矿会战期间，曾经有三支专门队伍承担铁矿床研究和勘查任务，取得了大量重要成果，积累了丰富的资料，一批富有经验的老专家依然活跃在矿床学研究舞台，有丰富的铁矿知识储备可供项目组利用。来自该所的项目组主要成员正在执行“全国矿产资源潜力评价”和“全国矿产资源利用现状调查”项目，且均以铁为主要矿种之一。通过这两个项目实施，掌握了全国铁矿床的基本特点、资源潜力和储量利用现状，储备了大量的铁矿基础资料，并且已与各省地勘单位和矿山建立了密切合作关系，为本项目的顺利实施提供了保障。

相关单位有一批专家长期从事铁矿或与铁矿有关的研究工作，取得了大量的成果，积累了丰富的资料。相关单位长期开展铁矿找矿勘查工作，将电、磁、重力等勘探技术方法广泛用于铁矿勘查实践，积累了一大批地质物化探及生产勘探资料。相关单位自成立以来，一直承担国家航空物探测量任务，积累了大量的地球物理资料，并掌握了先进的数据处理技术和方法。

# （4）研究集体是一支高水平的应用基础研究队伍

本项目推荐的首席科学家是国家杰出青年科学基金获得者，一直从事基础及应用基础研究，在面向地质找矿开展理论创新研究方面积累了一定的经验，近年来对岩浆型铁矿以及火山岩型铁矿有关理论问题开展了大量研究，研究成果发表在国内外著名专业刊物上，取得了很多新认识，为本项目的顺利完成奠定了良好的基础。项目组的主要科研骨干曾参加多项国家“973”计划和国家科技支撑计划项目研究，而且课题负责人近年来均承担有与铁矿相关的研究项目，是一个具有合理知识结构、多年共事、密切合作的研究群体，对本项目涉及的主要学科—矿床学、岩石学、矿物学、构造地质学、地球物理学、流体包裹体地球化学、元素-同位素地球化学、同位素年代学的研究内容和研究方法，具有较好的专业基础和工作经验，可以胜任本项目的研究任务。

# （5）研究的技术条件有保证

课题1：沉积变质型铁矿成矿环境与富矿形成机制

# 研究目标：

阐明早前寒武纪沉积变质型铁矿的时空分布规律和成矿机制，建立沉积变质型磁铁富矿的成矿模式，总结找矿标志。

# 研究内容：

选择典型矿集区，以富铁矿为主要对象进行重点解剖和对比研究，阐明地球早期大气、海洋和地质环境对铁大规模聚集的控制；查明不同类型硅铁建造的时空分布规律和形成过程；研究后期变质变形和热液活动对硅铁建造的叠加改造富化机制，揭示沉积变质型富铁矿的形成机理及关键控制因素，建立我国沉积变质型磁铁富矿成矿模式。

# 具体研究内容如下：

# （1）沉积变质型铁矿的形成环境与分布规律

通过精确定年，进一步厘定我国硅铁建造的形成时代，阐明早前寒武纪缺氧环境和大氧化事件对硅铁建造的控制作用；研究原始沉积盆地和沉积环境对成矿的控制；总结其时空分布规律。

# （2）沉积变质型铁矿的形成机理

研究海底火山-热液活动与成矿的关系；研究铁质的沉淀形式和控制因素；揭示条带状构造的形成机制；对比我国阿尔戈马型和苏比利尔型铁矿形成条件的异同。

# (3) 沉积变质型富铁矿的成矿条件与关键控制因素

针对具有我国特色的沉积变质型富铁矿，即以磁铁矿为主的富铁矿的形成机理开展研究，探讨区域变质热液和混合岩化热液加富、构造加富以及深部流体交代加富的机制和富铁矿形成的关键条件。

# 经费比例：  $20\%$

课题2：矽卡岩型富铁矿成矿机制及控制因素

研究目标：

揭示砂卡岩型富铁矿的成矿机制、岩浆岩成矿专属性的控制因素及大型矿集区与岩石圈改造和壳幔相互作用的关系；建立和完善砂卡岩型富铁矿的成矿模式，发展砂卡岩型铁矿的成矿理论；总结找矿标志。

研究内容：

以典型矿集区的典型矿床为重点，并与其他矿集区对比，探讨大型矿集区形成的地球动力学背景，研究含矿岩体成因，阐明岩浆成矿专属性的控制因素；揭示高浓度富铁流体的形成机理；建立矿集区尺度和矿床尺度的成矿模式。

# 具体研究内容如下：

# （1）岩石圈改造与大型矿集区的形成

精确厘定砂卡岩型铁矿及有关侵入岩体的形成时代和期次，综合区域地质-地球物理研究成果并对比国外砂卡岩型铁矿的主要成矿构造环境，揭示大型铁矿集区形成的地球动力学背景及与中国东部中生代岩石圈改造的时空耦合关系。

# (2) 含矿岩体的成矿专属性

根据成矿岩体、非成矿岩体及岩浆包体的矿物学和地球化学研究，探讨岩浆源区的性质和部分熔融过程及壳-幔相互作用对岩浆组成、性质和演化的控制，查明岩浆成矿专属性的关键控制因素，总结含矿岩体和非含矿岩体的判别标志。

# (3) 高浓度富铁流体和铁矿浆形成机理及铁质快速卸载机理

研究岩浆热液的出溶、演化及矽卡岩的形成过程，查明成矿流体的组成、性状及铁的运移形式和浓缩机理，剖析铁矿浆的形成条件和控制因素，探讨退化蚀变和钠质蚀变与铁质卸载形成富铁矿的关系。

# 经费比例：  $15\%$

课题3：火山岩型富铁矿成矿规律和成矿机理

研究目标：

揭示我国海相和陆相火山岩型富铁矿的形成机理以及大型矿集区与壳幔相互作用的耦合关系；建立和完善火山岩型铁矿的成矿模式，发展火山岩型铁矿成矿理论；总结找矿标志。

研究内容：

选择典型矿集区典型矿床为重点进行解剖研究，揭示火山作用的深部地质过程与大型矿集区形成的耦合关系；阐明铁矿浆和富铁流体的形成条件和关键控制因素以及高温热液蚀变叠加作用与富铁矿形成的关系。

# 具体研究内容如下：

(1) 重大地质事件与大型铁矿集区的形成

精确厘定铁矿以及海相、陆相火山岩的形成时代，揭示海相火山岩型大型铁矿集区的形成与晚古生代洋陆转换以及陆相火山岩型铁矿集区与中国东部中生代岩石圈减薄/改造等重大地质事件的耦合关系。

(2) 多旋回火山活动与岩浆成矿专属性

研究多旋回火山活动和多阶段岩浆侵位过程，对比国外典型矿床，阐明闪长质岩浆的形成机制及其与岩浆成矿专属性的内在成因联系，探讨铁矿浆形成的可能性及其关键控制因素。

（3）陆相火山岩型富铁矿的成矿机制

研究岩浆-热液体系演化与矿浆和富铁流体的形成机制；揭示钠钙质蚀变作用机理及其与铁质卸载成矿形成富铁矿的关系；研究膏盐层对铁矿成矿的贡献与作用机理。

（4）海相火山岩型富铁矿的形成机制

研究海底火山作用与铁矿形成的内在成因联系；探讨铁矿浆形成的可能性以及关键控制因素；厘定高温热液蚀变叠加改造的时代及其与火山旋回的可能关系，

揭示叠加改造作用对富铁矿成矿的贡献；对比海相和陆相火山岩型富铁矿的形成机制，建立相应的成矿模式。

经费比例：  $20\%$

课题4：岩浆型大型铁矿成矿过程和控矿因素

研究目标：

总结成矿规律；揭示大型钒钛磁铁矿特别是富矿的成矿机制，建立成矿模式；总结找矿标志。

研究内容：

选择典型矿集区典型钒钛磁铁矿床进行解剖，并与国外典型含矿岩体中的钒钛磁铁矿床进行对比，总结成矿规律；在已有工作基础上，综合运用元素和同位素示踪方法，探讨大型矿集区形成的深部地质过程，揭示基性-超基性岩浆起源和演化对钒钛磁铁矿床特别是富铁矿形成的约束机制。

# 具体研究内容如下：

(1) 大型钒钛磁铁矿矿集区与爆发性幔源岩浆事件

重点剖析含矿基性-超基性岩体地幔源区地球化学性质、部分熔融过程及其原始岩浆特点，特别是富铁钛原始岩浆形成的可能性和关键控制因素；揭示地幔柱对大型岩浆型钒钛磁铁矿矿集区的控制。

（2）岩浆演化与铁质的大规模聚集

研究含矿基性-超基性岩体的母岩浆性质，系统阐明原始岩浆的演化过程（如分离结晶、地壳同化混染、液态不混溶等）、岩浆多期补充以及物理化学条件变化对铁钛氧化物大规模聚集-成矿的控制作用。

（3）钒钛磁铁富矿的形成机制

借鉴实验岩石学的研究成果，运用微量元素和同位素示踪方法，结合岩浆熔体热力学模型，探讨镁铁质岩浆液态不混溶形成铁质矿浆的可能性，系统分析层状岩体底部块状富矿层的形成机制。

经费比例：  $15\%$

课题5：铁矿综合信息找矿预测体系及三维立体找矿示范研究

# 研究目标：

完善铁矿找矿评价方法体系，建立主要类型铁矿区域地质-地球物理综合信息找矿模型和大比例尺定位预测模型；开展三维立体找矿示范，圈定找矿靶区5～10处，并与地勘部门、矿山企业结合进行勘查验证；为发现新的大型铁矿、富铁矿提供技术支撑。

# 研究内容：

以不同类型铁矿成矿模式为基础，选择典型铁矿床，开展多元含矿信息评价，建立和完善区域和典型矿床找矿模型，创新铁矿找矿预测理论和方法体系，开展大比例尺定位预测和三维立体找矿示范研究。

# 具体研究内容如下：

# （1）铁矿找矿技术方法体系研究

确定不同地质条件下矿致磁异常的衰减模型和量化指标，研究航空和地面重磁数据联合反演新技术，结合瞬变电磁法（TEM）和可控源音频大地电磁法（CSAMT），建立我国重要类型铁矿的综合信息找矿方法体系。

# （2）区域地质-地球物理找矿模型

开展重要矿集区地质-地球物理（重、磁、电）背景与铁矿区域分布规律的关联性研究，分别建立沉积变质型、矽卡岩型、火山岩型和岩浆型铁矿床的区域地质-地球物理找矿模型。

# （3）典型矿床三维立体找矿模型与示范

综合利用地质信息和航空、地面及井中地球物理信息，选择典型铁矿床，建立大比例尺定位预测模型，开展三维立体找矿示范研究。

# （4）找矿靶区圈定与推动验证

开展中小比例尺找矿预测研究，圈定矿集区尺度找矿远景区；与矿山企业和地勘部门紧密结合，在选定的矿集区开展大比例尺预测研究，提交找矿靶区，推动工程验证。

经费比例：  $15\%$

课题6：富铁矿成矿背景与成矿作用综合研究

# 研究目标：

集成关键科研成果，阐明重大地质事件与大型矿集区的关系；总结各类富铁矿的分布规律；揭示巨量铁的来源、运移和超常富集过程和关键控制因素，发展与完善铁矿成矿理论；提出不同类型富铁矿床的成矿潜力与找矿前景。

# 研究内容：

从地质历史演化角度，揭示富铁矿形成的特殊地质背景和时空分布规律；通过不同类型富铁矿形成机制研究成果的凝练总结，结合对赤铁矿富矿的研究，并通过国际对比，重点阐明富铁矿形成的主要控制因素以及铁的超常富集过程和机制，发展和完善富铁矿成矿理论。

# 具体研究内容如下：

(1) 富铁矿的成矿地质背景、构造热事件及其时空分布规律

研究我国主要成矿带和矿集区富铁矿的成矿地质背景；从地质历史演化的角度，阐明构造热事件特别是重大地质事件和环境突变对铁大规模聚集成矿的控制以及特定地质环境和地质单元中铁巨量堆积的关键因素，总结富铁矿的时空分布规律。

（2）大型赤铁矿富矿床的形成机制

对铁矿的形成时代、地质背景、矿床成因等进行深入研究，揭示赤铁矿富矿的形成过程和关键控制因素，厘定矿床成因类型。

(3) 主要类型铁矿床铁同位素示踪理论模型构筑

以我国主要类型铁矿为对象，查明不同类型铁矿的铁同位素特征，结合同位素分馏理论，初步构筑运用铁同位素进行成矿作用示踪的理论模型；探讨铁同位素对识别成矿岩体与非成矿岩体的作用。

# （4）大型富铁矿的关键控制因素与富铁矿成矿理论集成

研究富铁熔浆/流体的形成条件和关键控制因素；探讨膏盐层在矽卡岩型与陆相火山岩型铁矿成矿过程中的作用；对比研究不同类型富铁矿成矿过程中物理化学条件的控制作用（特别是氧逸度的控制）；揭示变质变形过程以及深部流体交代过程中铁元素的运移和超常富集机制。集成主要类型富铁矿的矿床模型，发展和完善富铁矿成矿理论。
