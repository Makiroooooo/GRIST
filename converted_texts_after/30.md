# 报告正文

# （一）立项依据与研究内容

# 1、项目的立项依据

国际上钢铁材料的发展方向是高强钢，利用控轧控冷工艺实现组织细化与均匀化是提高强韧性的主要手段，而钢铁材料的组织与形变奥氏体在冷却过程中发生的相变有着直接的联系。实际上，变形过程中在多晶体组织中的应力应变分布是极其不均匀的，并且这一不均匀性对相变过程有着显著的影响。因此，如何合理控制形变一相变，得到理想的组织还有许多工作要做，特别是变形条件下组织分布的不均匀性和相变机理都有待于深入研究。本研究通过模拟与实验，分析宏观变形与多晶组织中微观应变分布的不均匀性的对应关系及其对相变的影响意义重大。

目前，国内外在控轧控冷制备细晶粒钢方面做了大量的实验和模拟工作[1]。Hurley等人[2-4]的研究结果发现：除奥氏体晶界外，热变形使奥氏体晶粒内部产生大量的缺陷（如位错、变形带等），这些由变形导致的缺陷提供了额外的铁素体形核位置，故铁素体形核密度增加，铁素体晶粒细化。齐俊杰等人[5]认为：在应变量较小时，铁素体在奥氏体晶界和奥氏体晶内的变形带处形核。随着应变和铁素体量增加，由于变形的非均匀性一形变主要集中在较软的铁素体内，铁素体核心开始在铁素体相前沿反复形核，且铁素体形核区逐渐向奥氏体晶粒内部推进。当应变量较大时，铁素体则可能发生动态再结晶，这是形成超细铁素体的原因。由于受到实验手段限制，以上观点来自于室温观察结果的反推，目前尚未形成统一的、令人信服的相变机理。

最近，荷兰金属研究所的Offerman等人[6]在Science上发表了关于碳钢连续冷却过程中形核和长大的文章。他们利用欧盟的同步加速器第一次实时测出了一定体积普碳钢中奥氏体一铁素体相变的形核率以及单个铁素体晶粒长大速度，其实验结果与经典形核理论相比，形核激活能至少要小两个数量级。在同期Science上，加拿大学伦比亚大学的冶金教授Militzer[7]认为：Offerman等人的实验为人们在晶粒尺度上深入研究钢中铁素体形核和长大机理提供了新的表征方法，同时也为建立更准确的数学模型，系统研究相变以及控制相变制备性能优异的钢铁材料提供了基础。但他们没有进行变形条件下奥氏体一铁素体相变的实验研究。

在相变模拟方面，Zhang[8]和Lan等人[9]采用元胞自动机方法模拟了连续冷却过程

中非变形奥氏体一铁素体相变过程；Tong等人[10]采用蒙特卡罗方法模拟了奥氏体一铁素体等温和连续冷却相变过程，但这些研究中都假设相变前的奥氏体为非变形态组织。

此外，国内外已有许多学者使用经验的物理冶金模型和宏观有限元来研究热变形奥氏体分解的相变动力学过程（铁素体相分数和晶粒尺寸随时间的演化等）[11-13]，以及用热力学方法来研究相变在平衡态的一些性质（比如平衡态的碳浓度分布和相分数）[14,15]。这些方法只能给出平均相分数和平均晶粒尺寸，而事实上由于多晶体塑性变形的不均匀性[16,17]（比如变形在晶界和某些晶内的局部集中）使得新相形核也不均匀（如在原始奥氏体的晶界上和变形带上优先形核）。与宏观有限元相比，晶体塑性有限元在介观尺度上将一个晶粒离散为多个单元，研究应变和应力在晶间和每个晶粒内的局域分布，利用它可以很好地模拟塑性变形在多晶体内分布的非均匀性，而且模拟结果可利用英国Sheffield大学的ChristophePinna博士的微网格刻蚀法实验进行验证[18]。我们课题组已采用晶体塑性有限元模拟了铝热压缩过程。通过考虑多晶体滑移系的运动，研究了剪切应力在晶粒间和晶粒内部的非均匀分布。Rabbe[16]和Radhakrishnan[17]等人分别采用晶体塑性有限元一元胞自动机和晶体塑性有限元一蒙特卡罗耦合模型模拟了静态再结晶的组织和冷轧织构演变过程。通过模拟，Raabe[16]发现再结晶核心主要集中在大的累积剪切应变区域，即剪切带处，具有很强的非均匀性；再结晶核心也并非如Avrami-Johnson-Mehl理论所描述那样向其周围自由长大，而是在再结晶的初期就相互接触，继而停止长大。而对于热变形奥氏体一铁素体相变的晶体塑性有限元一元胞自动机(或蒙特卡罗)耦合模拟工作，目前国内外未见报道。


综上所述，无论是目前已有的实验研究还是基于经典形核长大模型的理论计算，都仅仅能获得宏观变形条件和相变行为之间的关系，实际上这些方法都是唯象的。这主要是因为传统的实验方法、经典的形核长大理论和宏观有限元方法都不具备处理晶粒变形引起的微观应力应变的不均匀性对相变形核长大的影响。而事实上，变形条件下的多晶体组织中微观应力应变的不均匀性和由此导致的存储能分布的不均匀性对于相变过程以及相变组织的不均匀性有着重要的影响。

本研究提出的将模拟多晶材料塑性变形的晶体塑性有限元与模拟奥氏体一铁素体微观组织转变的元胞自动机方法耦合起来，跟踪变形态奥氏体分解过程中的铁素体形核和长大，并采用同步加速器实测铁素体形核率和长大速度的方法，能够弥补常规实验手段不能实时观测铁素体在变形奥氏体基体中形核长大过程和采用经验物理冶金模型只能模拟变形态奥氏体分解过程中的铁素体相变平均结果的不足，加深对变形态奥氏体一铁素体相变机理的理解。本项目对于相变机理研究和相变介观模拟技术的发展以及指导细晶粒钢的制备均具有重要意义。

# 2、项目的研究内容、研究目标以及拟解决的关键问题

# 研究目标：

实验与模拟计算相结合，分析存储能分布的不均匀性对铁素体形核长大的影响，揭示热变形条件下多晶奥氏体分解的机理，为优化热变形工艺细化组织提供理论依据。

# 研究内容：

本研究工作由实验研究和数值计算两部分组成，二者互为补充，具体内容如下：

1）利用同步加速器测量Q235变形奥氏体分解过程中铁素体的形核率与长大速度；  
2）利用晶体塑性有限元方法，计算奥氏体热变形时在介观尺度上造成的存储能分布的不均匀性，并用微网格刻蚀法进行实验验证；  
3）晶体塑性有限元一元胞自动机的耦合介观相变模拟；  
4）Gleeble3500热力模拟实验机上的热变形实验，验证模拟结果。

# 拟解决的关键问题：

1）用同步加速器实时测量变形奥氏体分解过程中铁素体的形核率和长大速度。  
2）晶体塑性有限元一元胞自动机耦合模拟变形奥氏体分解过程的奥氏体一铁素体相变。

# 3、拟采用的研究方案及可行性分析

# 1）利用同步加速器测量铁素体形核率和长大速度

本实验主要在中国科学院同步辐射实验室的4W1A束线和形貌实验站，利用由单极扭摆器4W1引出的白光束线进行X光衍射研究。制备的单轴压缩试样厚度范围是 $0.5\mathrm{mm} - 2.0\mathrm{mm}$  。将试样在  $950^{\circ}\mathrm{C}$  保温5分钟奥氏体化，然后冷却到变形温度  $900^{\circ}\mathrm{C}$  进行热变形（应变  $10\% -50\%$  、应变速率  $0.5 - 5.0 / \mathrm{s})$  ，变形结束后以  $0.1^{\circ}\mathrm{C / s}$  的速度冷却到500 $^{\circ}\mathrm{C}$  。在冷却过程中，利用从同步加速器中引出的单色X射线（最大能量：  $14\mathrm{KeV}$  ，光斑大小：  $0.5\times 0.8\mathrm{mm}^2$  ）照射实验样品（  $2\times 15\times 15\mathrm{mm}^3$  )，相变过程中析出的铁素体晶

粒会在二维探测器上产生不同的衍射斑点。从标准衍射理论可知：每个铁素体晶粒衍射点强度与其体积成正比，每个衍射点的衍射角反映了铁素体晶粒取向。通过北京同步加速器实验室的IP板记录实验过程中不同时刻探测器上的衍射图，分析铁素体衍射点数量和衍射点强度可获得奥氏体分解过程中的铁素体形核率和各铁素体晶粒的等效半径，即长大速度。利用系列实验数据，可以建立铁素体形核长大的动力学模型。在相同冷却条件下测量非变形奥氏体连续冷却过程中铁素体形核长大动力学数据。通过对比上述两类实验数据，研究热变形对奥氏体分解过程中铁素体的形核和长大动力学的影响。由于北京同步加速器的X射线与欧洲同步加速器相比能量较小（欧洲同步加速器X射线能量： $\sim 100\mathrm{KeV}$ ），铁素体晶粒产生的衍射角较大，我们拟缩短探测器与样品间的距离来获得完整的衍射图。另外，我们将采用自行设计的单轴压缩装置作为同步分析的附加装置来研究变形态奥氏体分解过程中铁素体的形核长大过程。

# 2）晶体塑性有限元模拟及相关实验研究

利用大型有限元分析软件ABAQUS的UMAT用户子程序，采用晶体塑性有限元方法，利用由单晶体的塑性本构关系经过统计平均而得到的多晶体的塑性本构关系计算在 $900^{\circ}\mathrm{C}$  温度下奥氏体的变形态组织，获取变形态奥氏体组织的形貌/取向/微观应力应变及存储能的分布。其中弹性模量、剪切模量等常规参数可以通过金属拉伸实验测得或从文献中获取。而晶体塑性模型中的初始屈服分剪切应力、临界屈服剪切应力和硬化影响因子等参数可以通过比较计算结果和实验数据，采用逆算法获得。利用微网格刻蚀法制备的含有微观应变信息的试样，验证晶体塑性有限元模拟结果的准确性。

# 3)晶体塑性有限元与元胞自动机耦合模拟

将晶体塑性有限元模拟结果中的单元积分点上的储存能通过线性插值映射到元胞自动机模型中的单元中，通过判断元胞自动机模型中各单元在有限元网格中的对应位置

来获取各单元的取向。这样，就将晶体塑性有限元的计算结果设定为相变模拟的初始条件。利用元胞自动机方法，模拟变形态奥氏体分解过程中的铁素体非均匀形核及长大过程。获得铁素体形核位置、铁素体晶核数量、铁素体组织形貌、取向分布以及体积分数。

# 4）热模拟实验

在Gleeble3500上进行热模拟实验。将试样在  $950^{\circ}\mathrm{C}$  保温5分钟奥氏体化，然后冷却到  $900^{\circ}\mathrm{C}$  进行热变形，变形结束后以  $0.1^{\circ}\mathrm{C / s}$  冷速冷却到  $500^{\circ}\mathrm{C}$  。在冷却过程中的一系列温度下将试样淬火，通过观察金相组织验证介观模拟的结果。

# 可行性分析：

中国科学院高能物理所拥有同步加速器，该加速器的4W1A束线和形貌实验站适用于进行晶体相变的原位观察研究。该实验站由各种相机、样品环境室和X光CCD成像系统组成。具体参数如下：能量范围  $3.5\sim 22\mathrm{KeV}$  ，最大光斑尺寸  $40\times 12\mathrm{mm}^2$  ，样品室温度  $300\sim 1400\mathrm{K}$  。目前，申请实验机时已获批准，2004年将进行初步试验。欧洲同步辐射实验室曾经使用同步加速器研究多晶体材料塑性变形过程中的晶粒旋转现象和未变形奥氏体冷却过程中的铁素体形核长大，所以本项目计划进行的实验方案是可行的。欧盟的同步加速器ESRF是面向全世界开放的，我们也提出了实验申请。本课题组与英国Sheffield大学ChristophePinna博士有着长期而密切的合作关系，已经利用它们的微网格刻蚀设备进行过一些变形试样微观应变分布的实验工作，积累了大量数据，可以用于对晶体塑性有限元计算结果的验证。本课题组在晶体塑性有限元和元胞自动机模拟方面已经开展了相关的预研工作，取得了一些令人鼓舞的成果。金属研究所拥有Gleeble3500热力模拟实验机，最大冷速可达  $10000^{\circ}\mathrm{C / s},$  可满足热模拟实验的要求。

# 4、本项目的特色与创新之处

采用同步加速器首次实时测量体材料中变形奥氏体分解过程中铁素体的形核率和晶粒长大速度，在与经典形核长大模型对比的基础上，建立更可靠的铁素体形核长大动力学模型。晶体塑性有限元与元胞自动机耦合进行相变模拟将首次揭示微观应变不均匀性对铁素体形核长大的影响，以期解决宏观有限元与常规实验无法完成的微观变形不均匀性与相变之间的关系等关键问题。

# 5、年度研究计划及预期研究结果

# 年度计划

2005年，用晶体塑性有限元方法模拟热变形奥氏体组织，获得变形态奥氏体中的应力、应变及变形储能的非均匀分布状况。与微网格刻蚀法实际测量变形奥氏体中的微观应变分布比较，修正模型。  
2006年，进行同步加速器实验，实时测量变形态奥氏体一铁素体相变过程中铁素体的晶粒个数和晶粒尺寸随时间的变化规律，建立相变动力学模型。  
2007年，进行晶体塑性有限元与元胞自动机耦合计算，系统分析微观应变不均匀性对铁素体形核长大影响，揭示热变形条件下低碳钢中多晶奥氏体相变机理。

# 预期研究结果：

获得低碳钢Q235变形奥氏体一铁素体相变的形核率和晶粒长大速度实时测量数据，建立多晶奥氏体在热变形条件下相变的形核与长大动力学模型；建立变形奥氏体一铁素体相变介观组织演化的晶体塑性有限元一元胞自动机耦合模型；通过分析微观应变与铁素体形核不均匀性，揭示多晶奥氏体在热变形条件下的相变机理。

在SCI期刊上发表论文5-6篇。

# （二）研究基础与工作条件

# 1、工作基础

申请人从1991年开始从事材料制备过程微观组织的模拟，对有限元、有限差分及元胞自动机等方法有深入的了解。同时，课题组拥有中高级研究人员、博士后及10多名研究生，在热变形与凝固过程组织演化的模拟与实验研究方面具有良好的工作基础。我们已经用原胞自动机以及蒙特卡罗方法建立起了相应的模拟模型，模拟了等温和连续冷却过程中的奥氏体一铁素体相变，并发表了4篇学术论文（Acta Materialia2篇，Scripta Materialia1篇，Metallurgical &Materialias Transactions A1篇）。对于这部分的研究工作，本课题组已经有了较扎实的基础。但是这些模拟工作中关于存储能分布和形核动力学方面做了很多简化和假设，有待于在基金资助下开展深入研究。课题组参加的一期973计划中的“新一代钢铁材料的重大基础研究”项目已经顺利结束。本课题组开发了热轧模拟软件，采集了大量的实验数据，进行了材料热物性的测量，为本申请项目的研究打下了扎实的基础。其他课题组的研究结果也将为本基金项目的深入开展提供参考。本研究组目前聘请了长江学者—西北工业大学工程力学与土木建筑工程学院岳珠峰教授为访问学者，以便更好地指导晶体塑性有限元方面的研究。目前，已利用晶体塑性有限元模拟了铝热压缩过程中的变形行为，获得了初步结果，模拟结果显示了等效应力在不同晶粒间和各晶粒内的局部非均匀分布。另外，在晶体塑性变形的实验验证方面，本课题组与英国Sheffield大学的ChristophePinna博士合作密切，可以用微网格刻蚀法(见参考文献18)对晶体塑性有限元模拟结果进行验证。

# 2、工作条件

硬件方面，光学电子显微组织分析可在我所分析测试部完成。我所拥有Gleeble3500实验机可以进行热模拟实验。而利用同步加速器测定变形奥氏体一铁素体的形核率和晶粒长大速度的实验可到中国科学院北京正负电子对撞机国家实验室进行。课题组有多台性能良好的工作站和微机，可以完成计算工作。软件方面，课题组拥有ABAQUS和ANSYS等大型有限元分析软件和熟练运用这些软件的科研人员，可以确保课题的顺利完成。
